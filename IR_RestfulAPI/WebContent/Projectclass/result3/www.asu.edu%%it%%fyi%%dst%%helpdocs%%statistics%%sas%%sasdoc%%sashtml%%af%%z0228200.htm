Critical Success Factor: _getValue


Returns the current value of the critical success factor

Inherited from Widget

Syntax
Details

Syntax

CALL NOTIFY (csf-name, '_getValue', csf-value);

Argument Type Description
csf-value
 N returns the value of the critical success factor

Details

The value returned by the _getValue method is visually represented as the location of the arrow on the CSF picture.
