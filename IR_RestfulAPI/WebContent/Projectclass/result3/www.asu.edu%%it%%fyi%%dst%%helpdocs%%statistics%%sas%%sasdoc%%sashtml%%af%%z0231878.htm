Extended Table: _getVisrow

Extended Table: _getVisrow


Returns the number of rows that are currently visible

Syntax
Details

Syntax

CALL NOTIFY (extended-table-name, '_getVisrow', out-visrow);

Argument Type Description
out-visrow
 N the number of table rows that are visible

Details

The visible rows are the number of logical rows that can be displayed in the region.
