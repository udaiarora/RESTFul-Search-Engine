Random and Exogenous Variation in the Model

Controlling Time Units

Each of the random variable control panels also shows a Units combo box. See Figure 6.2 for an example. With this you can identify a value for the time units. Then, with the Clock Options window, (see Figure 6.5) accessed by selecting Options  Clock Options... from the Simulation window (Figure 6.3), you can assign time units to be used for the simulation clock.


Figure 6.5: The Clock Control Panel

If you select time units for the simulation of seconds and an inter-arrival time distribution for some random variable on a Sampler has time units of minutes, then the simulation will automatically factor in the conversion in the sampling.
