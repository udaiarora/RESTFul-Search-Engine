FTITLE

FTITLE


Selects the default font for the first TITLE line.

Used in: GOPTIONS statement
Default: (1) font specified by FTEXT=, if used; (2) SWISS font
See also: FTEXT

Syntax

Syntax

FTITLE=title-font

title-font
specifies the font for the TITLE1 statement. See SAS/GRAPH Fonts for information about specifying fonts.

Featured in: Example 1. Ordering Axis Tick Marks with SAS Datetime Values and Example 6. Enhancing Titles
