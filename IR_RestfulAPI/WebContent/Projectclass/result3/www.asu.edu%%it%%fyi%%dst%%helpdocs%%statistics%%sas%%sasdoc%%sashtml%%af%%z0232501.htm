GET_DATA Data Vector: _getColumn

GET_DATA Data Vector: _getColumn


Returns the column coordinates

Syntax
Details

Syntax

CALL SEND (data-vector-id, '_getColumn', column);

Argument Type Description
column
 N specifies the identifier of an SCL list to contain the column coordinates.

Details

The _getColumn method is valid if the viewer is a form editor or a table editor.
