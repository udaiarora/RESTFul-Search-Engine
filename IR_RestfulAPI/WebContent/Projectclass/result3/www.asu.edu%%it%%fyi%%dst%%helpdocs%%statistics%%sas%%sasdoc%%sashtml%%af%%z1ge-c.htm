Using the Range Model

The Range model lets you set minimum, maximum, and increment values that another object (typically a viewer) can use.

To set the range, define values for the minimum and maximum attributes.

CAUTION:
The value of the minimum attribute must be less than the value of the maximum attribute. Since the range model works as a span of lesser to greater values, you must set these attributes correctly.  

To set the increment values, use the smallIncrement and largeIncrement attributes.
