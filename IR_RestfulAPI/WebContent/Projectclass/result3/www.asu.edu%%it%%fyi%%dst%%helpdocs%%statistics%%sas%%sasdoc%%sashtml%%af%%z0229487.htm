Icon: _setLabelPosition

Icon: _setLabelPosition


Assigns the position of the icon's label

Syntax

Syntax

CALL NOTIFY (icon-name, '_setLabelPosition', position-char);

Argument Type Description
position-char
 C specifies the position of the icon's label:


'R' to the right of the icon


'B' below the icon
