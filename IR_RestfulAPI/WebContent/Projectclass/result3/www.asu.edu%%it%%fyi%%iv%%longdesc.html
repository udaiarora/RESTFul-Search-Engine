Descriptions of Screen Captures

Mount Instructor Volume Dialog

Screen capture of Windows dialog box with a blue text box displaying the text Enter name of Instructor Volume and press Enter, leave blank to cancel.

Chooser Window

Screen capture of Macintosh Chooser window divided into two panes (left and right). The left pane is further divided into the top and bottom sections. The top-left section consists of a list box containing four icons, labeled AppleShare, HP Accupage, KeyAccess, and LaserWriter 8. The AppleShare icon is highlighted. The bottom-left section contains a list box (labeled AppleTalk Zones), with the following list of zones:

  • GLG PSH
  • GWC Math
  • HAYDEN LIBRARY
  • Honors
  • HRC Main
  • ICA
  • Information Technology (selected)
  • ISA

The right pane contains a list box (labeled Select a file server:) which lists the following entries:

  • IT Distribution
  • IT Instructional
  • IT Keyserver

Below the list box are two buttons, labeled Server IP Address . . . and OK. Below these are a pair of radio buttons, the pair being labeled AppleTalk; the buttons are labeled Active and Inactive.

Connect to File Server Dialog

Screen capture of Macintosh dialog box with the words Connect to the file server "IT Instructional" as: followed by two radio buttons, labeled Guest and Registered User. Below the radio buttons are a text box (labeled Name: and containing the text gizmo17) and a password box (labeled Password: and containing four masking characters). Below the password box (which is selected) are the words 2-way Encrypted Password. At the bottom is a button labeled Change Password . . . as well as standard Cancel and OK buttons, the latter being labeled Connect.

Server Volume Selection Dialog

Screen capture of Macintosh dialog box with the words IT Instructional at the top, followed by a list box containing:

  • Applications (grayed)
  • Applications 2 (grayed)
  • Instructor Volumes (selected)
  • Instructor Volumes 2

Each item in the list box has a checkbox to its rights, none of which is checked. Below the list box is the text Checked items ([image of checked checkbox]) will be opened at system startup time. At the bottom of the window are standard Cancel and OK buttons.
