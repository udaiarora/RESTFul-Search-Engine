PrintKey2000

PrintKey2000 is a freeware print screen utility for Microsoft Windows. Unlike the character mode, the graphical mode of User Interface 1.0 does not include a print screen feature. Many users have requested print screen capability without having to copy to the clipboard and paste into another document. PrintKey2000 handles this function well, and does not impact the performance and reliability of the Windows environment. We have tested it on Win95, Win98, NT4, and Win2k. You may also find this utility useful for applications other than Benefactor and Colleague.

 

Installation Procedure:

After retrieving the distribution file execute PrtKey. The installation file will begin the setup process.

There are no suprises in this installation. You may accept the defaults or customize the locations. After completing the installation, you may want to copy the PrintKey2000 icon to the startup folder, to load this utility automatically.
