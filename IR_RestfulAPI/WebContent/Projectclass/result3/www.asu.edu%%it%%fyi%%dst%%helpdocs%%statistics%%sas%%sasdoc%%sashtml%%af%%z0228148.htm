Command Push Button: _setLength

Command Push Button: _setLength


Sets the length of the command push button

Syntax
Details

Syntax

CALL NOTIFY (field-name, '_setLength', length);

Argument Type Description
length
 N assigns the length of the field, in characters

Details

If the length specified is too long for the window, this method returns without doing anything.
