Frame Class: _getTable

Frame Class: _getTable


Returns the object identifier for the extended table containing a specified widget

Syntax

Syntax

objectName._getTable( objectName, extendedTableID );

Argument Type Use Description
objectName Character Input specifies the name of a widget in the frame
extendedTableID Numeric Output returns the object identifer for the extended table containing the widget or a value of zero if the widget is not in an extended table
