Custom Search Forms

ASU has signed up with Google (Google University Site Search) to be our official search engine for the asu.edu domain and www.asu.edu. This search engine offers site visitors a single interface to a website that spans several servers.  This is simpler and more convenient for visitors than requiring them to visit a different search engine and/or search page for each Web Server at ASU.  However, there are times when it's desirable to offer the visitor a page that searches only a subsection of the ASU Web.

Directory-Specific Searches on www.asu.edu

While Google's University Site Search is not 100% optimized for searching at the directory level (for example, www.asu.edu/mydirectory), due to many requests from universities to search at the directory level, Google has created a solution that allows this additional functionality.  To offer your visitors the ability to search your specific directory on www.asu.edu add the following line of html to the ASU Google search and change yourdirectory to the name of your directory.

<input type=hidden name=hq value="inurl:www.asu.edu/yourdirectory">

Below is the Google Search source code taken from http://www.asu.edu/search/, with the directory-specific change added but with no other aesthetic or functional modifications. Farther down on this page you will find the code to make your search box look like ours on the Jukebox and also instructions for those ASU departments that have their own webservers or a new webserver.

(includes "inurl:www.asu.edu/it/" to return results from the "it" directory on http://www.asu.edu/)

Code Sample for the Search Box on the Jukebox

We've modified the Google search box for the Jukebox search (see it in action directly below or within the grey column to the left).  You can modify your department's search box in a similar way.  Here is sample source code if you want your search box to look & function similar to those found in the Jukebox:

Example:

Search the Jukebox:

Source Code

<!-- Search Google -->
<form method="GET" action="http://www.google.com/u/arizonastate">
<b>Search the Jukebox:</b><br>
<input type=text name=q size=10 maxlength=255 value="">
<input type=submit name=sa value="Go!"><input type=hidden name=domains value="asu.edu">
<input type=hidden name=sitesearch value="asu.edu">
<input type=hidden name=hq value="inurl:www.asu.edu/jukebox/">
</form>

Do you have your own server?

Do you want a Google Search box to search your pages on your department server? As an example, to have a Google Search box on http://herbergercollege.asu.edu/, you would modify the two hidden input types for the domains and sitesearch values in the Google Source Code to equal "herberbercollege.asu.edu" instead of "asu.edu".  In addition, if you wanted to search only a specific directory on this server, you would change the two hidden input types and also add a new hidden Input type to contain a value : "inurl:is.asu.edu/adirectory"

Example:

Do you have a NEW server?

If you have a new web server, you will need to let Google know about it so it can be made a part of their regularly scheduled index.  Follow the instructions on Google's site to add your server to their search engine.

How to keep Google from indexing or caching your webpages

Google adheres to the Robots Exclusion Standard.  You can keep Google from indexing your pages by either using a robots.txt file at the root of your web server (if you are a server administrator) or by including a "robots" NOINDEX meta tag (e.g., <META NAME="ROBOTS" CONTENT="NOINDEX">) in the header of individual html files.  To keep Google from storing a cached version of your webpage on a Google server you can include a "robots" NOARCHIVE meta tag  (e.g., <META NAME="ROBOTS" CONTENT="NOARCHIVE">) in the header of your html files.  More information on Google's interaction with robots.txt files and specific meta tags can be found at http://www.google.com/webmasters/.

How can I remove my webpages from Google search results?

If you no longer want your webpages to appear in search results from Google, please visit http://www.google.com/remove.html and follow the instructions found on that page.  Google has specific requirements for removing content from their search indexes.
