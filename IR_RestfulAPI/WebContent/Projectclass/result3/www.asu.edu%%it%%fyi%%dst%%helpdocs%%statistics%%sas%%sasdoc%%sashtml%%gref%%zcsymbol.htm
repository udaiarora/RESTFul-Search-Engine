CSYMBOL

CSYMBOL


Specifies the default color for SYMBOL definitions when a color has not been specified.

Used in: GOPTIONS statement
Default: first color in current colors list

Syntax
Details

Syntax

CSYMBOL=symbol-color

symbol-color
specifies any SAS/GRAPH color name. See SAS/GRAPH Colors for information about specifying colors.

Details

CSYMBOL= is overridden by any color specification in a SYMBOL statement. See SYMBOL Statement.
