List Box Control Events

Events specified for the List Box Control class are described here.

click
Occurs on a click

doubleClick
Occurs on a double-click

font changed
Occurs when the font attribute is changed

items changed
Occurs when the items attribute is changed

popup
Occurs on a popup request

rows changed
Occurs when the rows attribute is changed

selectedCount changed
Occurs when the selectedCount attribute is changed

selectedIndex changed
Occurs when the selectedIndex attribute is changed

selectedIndexes changed
Occurs when the selectedIndexes attribute is changed

selectedItem changed
Occurs when the selectedItem attribute is changed

selectedItems changed
Occurs when the selectedItems attribute is changed

selectionMode changed
Occurs when the selectionMode attribute is changed

textColor changed
Occurs when the textColor attribute is changed

textOrientation changed
Occurs when the textOrientation attribute is changed

title changed
Occurs when the title attribute is changed

titleHeight changed
Occurs when the titleHeight attribute is changed

topRow changed
Occurs when the topRow attribute is changed
