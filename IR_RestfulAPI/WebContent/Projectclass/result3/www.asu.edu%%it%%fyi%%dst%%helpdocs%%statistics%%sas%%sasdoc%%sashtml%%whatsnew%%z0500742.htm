ODS and Web Enablement

The DTREE, GANTT, LP, NETDRAW, NETFLOW, and NLP procedures are all ODS-capable in Version 7. In addition, the GANTT and NETDRAW procedures are now web-enabled. The HTML= option enables you to specify a variable in the activity data set that defines a URL for each activity. This reference will be associated with all the schedule bars or network nodes for the activity.

If a Gantt chart or network diagram is routed to an HTML file by using ODS, then you can click on a schedule bar or network node and browse text or other descriptive information about the associated activity. The IMAGEMAP= option in the GANTT and NETDRAW procedures produces an output data set that contains a description of the area of a graph and a link for each area. This information can be used to create custom HTML files with drill-down graphs.

In Version 8, the DTREE procedure is also web-enabled. The clickable areas in the decision tree diagram correspond to the stages in the decision model. The HTML= option specifies a variable in the STAGEIN= data set that defines a URL for each stage.
