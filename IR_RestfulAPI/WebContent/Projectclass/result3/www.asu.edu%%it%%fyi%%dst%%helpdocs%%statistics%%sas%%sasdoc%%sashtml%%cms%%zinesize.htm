LINESIZE=

LINESIZE=


Specifies the line size of SAS procedure output

Default: terminal width; 132 (batch)
Valid in: configuration file, SAS invocation, OPTIONS statement, OPTIONS window
CMS specifics: default

Syntax
See Also

Syntax

LINESIZE=MIN | MAX | n | hexX

MIN
specifies the minimum line size (64).

MAX
specifies the maximum line size (256).

n
specifies an integer line size.

hexX
specifies a line size in hexadecimal notation.
