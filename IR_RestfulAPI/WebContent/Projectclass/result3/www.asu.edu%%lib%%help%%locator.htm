Finding Materials Once You Have a Call Number


The lists below will help you find materials once you have found the call number by using the ASU Libraries' Online Catalog. The Catalog will give you a location code that indicates the library that holds the item. In addition, you will find information that tells you in which section of that library the item is located. For materials located in the Hayden and Noble Libraries, additional tables are included to indicate the floor in the library where you can find that call number.
