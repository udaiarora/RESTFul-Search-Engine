Getting Started with SAS/EIS Software, First Edition

Get started building your own enterprise information system (EIS) by following the easy, step-by-step instructions provided in this book. You will learn how to use SAS/EIS software to create a Multidimensional Database, register data in the Metabase, build applications, and to quickly build and print sophisticated business graphs and reports. This book is intended for prospective SAS/EIS software users who are familiar with SAS software. This book is an excellent companion to the online tutorial that is available with Release 7.00 of SAS/EIS.
