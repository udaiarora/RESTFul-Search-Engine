Ada compiler & Motif stuff on the General Cluster

The General Cluster does have the Ada compiler, however the man pages are not installed. If you wish to read the man(ual) pages on the Ada compiler, simply enter the following at the system prompt.

export MANPATH=/opt/SUNWspro/Ada2.1.1/man

This will setup the right man path variable and enable you to access the manual pages. The compiler is located in: /opt/SUNWspro/Ada2.1.1/bin and by default it will look for ada.lib in your working directory.

The builder should be in it too (X-windows version) gxv_ada gxv_ada (1) - generate Ada and Ada XView binding source code for user interface elements described in a GIL file.

Below are the X-libraries available on general:

ls /usr/lib/*X*
/usr/lib/libX11.so.4
/usr/lib/libXol.so.3
/usr/lib/libXt.so.4
