UTILITY

The utility program is executed as a batch job. The utility program consists for JCL statements and RACF commands. This utility program is the only method available for communicating with RACF unless you are TSO, in which case you can enter the RACF commands on the TSO command line or you can use ISPF option 9.R.

Sample JCL for submitting the utility program is shown below.

   // JOB
   //TSO EXEC PGM=IKJEFT01,DYNAMNBR=20
   //SYSTSPRT DD SYSOUT=A
   //SYSTSIN  DD *
   *** You would replace this line with one or more RACF commands ***
   /*
   //

You can submit the utility program with as many RACF commands as you need. Each command will be executed in the same order that they appear in the batch job. If a RACF command fails to execute properly, it will be skipped and the execution will continue with the remaining RACF commands.

In practice, most RACF commands are only one line long; that is, they will fit on one 80-column line. You can start the command in any column of the line. If the command requires more than one line, you can continue the command from one line to the next by ending the continued line at any point where you would normally type a space; after the space you would type a plus sign +. You would then continue typing the remaining portion of the command on one or more subsequent lines. The following illustrates one command continued on a second line.

     ADDSD 'WYL.XX.YYY.DATA' +
       UACC(READ)

The following is a list of RACF commands which you can use in a batch job to perform various RACF functions.

  • ADDSD - Add a data set profile so you can change it's protection
  • ALTDSD - Alter a data set profile; change data set protection
  • ALTUSER - Alter information in user profile; change name or password
  • DELDSD - Delete a data set profile; data set becomes fully protected
  • LISTDSD - List existing data set profiles and protection settings
  • LISTUSER - List attributes associated with your userid
  • PASSWORD - Change password or password interval
  • PERMIT - Add or remove data set access for particular userids
  • RALTER - Add or remove tape volume access for particular userids
  • RLIST - List existing tape volume profiles and protection settings
  • SEARCH - List data set names which have been added to RACF
