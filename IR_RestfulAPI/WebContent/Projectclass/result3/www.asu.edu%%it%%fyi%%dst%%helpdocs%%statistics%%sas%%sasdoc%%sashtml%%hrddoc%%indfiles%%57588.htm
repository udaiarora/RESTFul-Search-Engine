What's New in SAS Software for Version 8

This document includes brief overviews of most changes and enhancements that have been made to SAS software products since Release 6.12--or, for the mainframe platforms (OS/390, CMS, and OpenVMS VAX), since Release 6.09E. Detailed information about new or enhanced features is available in the Help system, on the World Wide Web, and/or on the SAS OnlineDoc, Version 8, CD-ROM. This What's New document is useful both to sites that upgrade from Version 6 to Version 8 and to sites that upgrade from Version 7 to Version 8.
