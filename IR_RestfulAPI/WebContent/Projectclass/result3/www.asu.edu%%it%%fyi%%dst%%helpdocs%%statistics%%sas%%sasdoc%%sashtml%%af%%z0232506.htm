GET_DATA Data Vector: _getFormatLength

GET_DATA Data Vector: _getFormatLength


Returns the format length

Syntax
Details

Syntax

CALL SEND (data-vector-id, '_getFormatLength', length);

Argument Type Description
length
 N returns the format length

Details

The _getFormatLength method is valid if the viewer is a form editor or a table editor. The value returned by the _getFormatLength method is the maximum number of characters that the viewer can display.
