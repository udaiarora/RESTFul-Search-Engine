HOW TO START USING THE ASU DATA WAREHOUSE

To start using the ASU Data Warehouse, follow these five steps to get up and running. If you want to know more about the Warehouse and why you might want to use it, click here. If you have questions about anything to do with the Warehouse, send an e-mail note to WARE-Q@asu.edu. (WARE-Q on Outlook.)

1. Make Sure You Have The Necessary Computer Hardware.
IBM PC - 233 MHz, 32 MB Memory (more is better) 600x800 resolution screen minimum (higher is better), 300 MB on hard drive, Ethernet connection.

2. Make Sure You Have The Necessary Computer Software.
Operating System: Windows 98, NT, 2000, XP or ME. Windows 95 can work with the older Sybase 11 software, but Sybase 12 is preferred.
Connection software: Sybase 12 native drivers are fastest. They are available from Information Technology at no charge and can be downloaded from http://www.asu.edu/it/licensing/sybase.htm. You can also use Microsoft's ODBC, however it is slower. If you use ODBC, you still need the download from IT because it provides the Sybase ODBC software you need to connect ODBC to Sybase.
Query Software: BrioQuery Explorer 6.6 is the software supported by Data Administration. Click here for more information regarding the software. You can use other query software of your choice (i.e. Microsoft Access), however the Warehouse classes use BrioQuery. Brio is preferred for its ease of use and lower learning curve for non-technical users as compared to Access.

3. Make Sure You Have Enough Experience With Computer Stuff.
Familiarity with the Windows environment is a MUST. Know how to move, minimize, maximize, and resize windows and how to save files and open files and find them on the hard drive.

Suggested skills include experience with a spreadsheet and experience with the data you want to use - Student Information System (SIS) experience if you want to use the STUDENT database, Advantage experience if you want to use the FINANCIAL database, HRMS experience if you want to use the HUMAN_RESOURCE database, and Coeus experience if you want to use the SPONSORED database.

4. Send A Completed Warehouse Access Request Form to Data Administration. (Fax 5-1559)

Data Warehouse Access Request Forms are available on the Computer Accounts web site at http://www.asu.edu/it/tempe/cac/accounts/forms.htm. There is a web form, or a printable form you can fill out and fax. Select the database(s) you need access to and make sure to indicate WHY you need access. Your supervisor must sign the printed form . If using the printed form, fax the form to Data Administration at 1-1559. New STUDENT warehouse users will not be notified of their user ID and password until they attend training. Users of other databases will receive an email notice when access is set up.

NOTE: Data Trustees grant access to the Warehouse separately from the production systems (SIS, Advantage, HRMS, Coeus). Your Warehouse user ID and password will not work for the production systems. If you are requesting several computer accounts on different computers, it is best to do a separate form for the Data Warehouse and send it to Data Administration (ADMIN B 370, Mail Code 1203) rather than Computer Accounts. DA will route it to the data trustees for approval and then to Computer Accounts.

5. Sign Up for Training

The following courses are available for Data Warehouse customers:

Intro to the Student Data Warehouse (4hrs) ($50 fee)
Human Resources Data Warehouse Training (3.5 hrs)
Financial Data Warehouse Training (3.5 hrs)
Facility_Reporting Data Warehouse Training (3.5 hrs)

Before signing up for training, please make sure your request for access has been submitted. To find out dates for upcoming classes and register you may call (480) 965-4751 or visit the Employee Training website at http://www.asu.edu/hr/training/. Click on the Register/Catalog link, select the All Categories/All courses option for the dropdown box, and search for Data Warehouse.
