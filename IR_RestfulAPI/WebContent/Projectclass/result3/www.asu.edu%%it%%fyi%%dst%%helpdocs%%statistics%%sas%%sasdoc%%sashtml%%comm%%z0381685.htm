Communications Access Methods for SAS/CONNECT and SAS/SHARE Software

Using the SAS/CONNECT and SAS/SHARE Documentation Set

The following three books are used with SAS/SHARE and SAS/CONNECT software:

  • SAS/CONNECT User's Guide

  • SAS/SHARE User's Guide

  • Communications Access Methods for SAS/CONNECT and SAS/SHARE Software.

Communications Access Methods provides information to establish a connection between two hosts across a network. For using SAS/CONNECT or SAS/SHARE software, start with either the SAS/CONNECT User's Guide or the SAS/SHARE User's Guide and then refer to Communications Access Methods for SAS/CONNECT and SAS/SHARE Software when making a network connection.
