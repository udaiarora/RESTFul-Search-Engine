Conclusion

This lesson has given you the basics about evaluating sources. While each source is different, these skills should guide you in making an informed decision about using the source in your research. As you become more experienced in doing research, evaluation will become an important aspect of the process.

Don't be afraid to ask the reference librarians for help when you need it. To find contact information for the librarians who can help you evaluate sources, see "Who can help you?"
