Check Box: _setColor


Assigns a color to a check box

Restrictions: Some systems do not support using methods to change the color of a check box.

Syntax

Syntax

CALL NOTIFY (check-box-name, '_setColor', color-name);

Argument Type Description
color-name
 C specifies the color for the check box.
