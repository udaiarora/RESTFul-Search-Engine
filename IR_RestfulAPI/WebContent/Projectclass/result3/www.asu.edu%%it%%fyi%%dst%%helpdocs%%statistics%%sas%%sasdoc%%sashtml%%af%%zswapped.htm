Widget Class: _isSwapped

Widget Class: _isSwapped


Reports whether the object is swapped out

Syntax

Syntax

objectName._isSwapped( isSwapped );

Argument Type Use Description
isSwapped Numeric Output returns a value indicating whether the object is swapped: 0 if swapped, 1 if not swapped
