SAS/AF Software: Class Dictionary

Overview

Use the Graphic Text class to display text using a variety of fonts, font sizes, and colors. The Graphic Text class provides more flexibility in fonts and font sizes than does the Text Label class.

Note:   You can use the Graphic Text class only if SAS/GRAPH software is licensed at your site.  

Parent:
sashelp.fsp.widget.class

Class:
sashelp.fsp.GText.class
