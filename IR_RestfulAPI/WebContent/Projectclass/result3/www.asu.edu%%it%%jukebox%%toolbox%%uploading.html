Uploading

Once you've created your webpages, graphics, images, multimedia files, and others you next need to "upload" these files to your web space which is a www directory located in your AFS space (If you need to brush up on these terms or how web space is organized, check the Beginners section). Below are links to instructions and software for both MacOS and Windows users. In a few minutes, you'll see your pages online!

SSH software for SFTP access will be required for access to all Information Technology servers as of December 19, 2003. The SFTP client is included in the distribution. The SSH Secure Shell (secure Telnet) Client and SSH Secure File Transfer (SFTP) Client are the industry standard for secure remote access to host servers (Telnet and FTP). SSH solves the most important security problem on the Internet, securing userIDs and passwords. We have clients for Windows 9x, Windows NT, Windows ME, Windows 2000/XP, Macintosh, and several UNIX flavors. The distributions also include source code for servers.

Download SSH software at http://www.asu.edu/security/software/html/ssh/.
