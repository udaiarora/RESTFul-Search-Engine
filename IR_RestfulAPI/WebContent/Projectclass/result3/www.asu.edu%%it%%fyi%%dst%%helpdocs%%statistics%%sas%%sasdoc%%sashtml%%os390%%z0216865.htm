FILEREUSE

FILEREUSE


Specifies whether to reuse an existing allocation for a file that is being allocated to a temporary DDname

Default: NOFILEREUSE
Valid in: configuration file, SAS invocation, OPTIONS statement, OPTIONS window
Category: File Control: EXTFILES
OS/390 specifics: all

Syntax
Details

Syntax

FILEREUSE | NOFILEREUSE

Details

If FILEREUSE is in effect and there is a request to allocate a file that is already allocated, the existing allocation is used whenever the new allocation would cause a temporary DDname (of the form @SASnnnn) to be generated.
