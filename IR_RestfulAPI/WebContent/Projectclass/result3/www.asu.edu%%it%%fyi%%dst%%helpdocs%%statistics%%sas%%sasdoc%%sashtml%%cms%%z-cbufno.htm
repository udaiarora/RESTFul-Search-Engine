CBUFNO=

CBUFNO=


Specifies the number of extra page buffers to use for each SAS catalog

Default: 0
Valid in: configuration file, SAS invocation, OPTIONS command, OPTIONS window
CMS specifics: default, range of values

Syntax

Syntax

CBUFNO=MIN | MAX | n | hexX

MIN
specifies the minimum number of extra page buffers (0).

MAX
specifies the maximum number of extra page buffers (20).

n
specifies an integer number of extra page buffers.

hexX
specifies the number of extra page buffers in hexadecimal notation.
