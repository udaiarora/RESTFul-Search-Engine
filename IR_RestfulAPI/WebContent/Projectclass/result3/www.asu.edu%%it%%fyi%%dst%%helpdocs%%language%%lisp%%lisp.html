Franz Allegro Common Lisp on the AI Server

To execute Lisp on the AI Server, type:

composer

There is also an alias named "cl" which can be typed to access Lisp. The "composer" executable resides in /usr/local/acl4.3.1/bin .

For additional information on Lisp, you can access the online help while within Lisp by typing:

:help

Note that a colon must be included for this command to work. To exit Lisp type:

:exit

Several subdirectories of sample programs can be found in: /usr/local/acl4.3.1/contrib . They are:

bm Benchmarks
clx CLX under Franz Lisp
lps CPS NeWS client interface
mkant-cmu Utilities for Common Lisp
ops5 Common Lisp OPS-5 source
rpc Sun RPC via Lisp
defctype

If you'd like to use an emacs interface you'll need a .emacs file in your home directory which contains: 

   (setq load-path
      (cons "/usr/local/acl4.3.1/home/emacs/fi"
            load-path))
   (load "fi-site-init")

There is additional information on Franz Inc.'s implementation of Lisp on their WWW homepage, located at http://www.franz.com/ .
