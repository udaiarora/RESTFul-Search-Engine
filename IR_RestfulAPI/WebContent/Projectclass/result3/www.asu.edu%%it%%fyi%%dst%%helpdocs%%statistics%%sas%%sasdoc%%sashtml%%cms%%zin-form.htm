Host-Specific Frames of the FORM Subsystem

The FSFORM command can be issued from any window to alter or create a print form to be used for printing full-screen output.

The FORM window includes frames that are used to tailor a FORM entry in the SASUSER.PROFILE catalog for the CMS environment and printers. FORM entries are used to send output from various procedures and commands to the printer you specify. (For example, the PRINT command sends window output to the printer specified by a FORM entry.) Information such as printer margins, printer control language, and font control information can be specified. See the Help for base SAS for complete information.

The FORM subsystem includes a Print File Parameters window that contains system-dependent information. This window enables you to specify printer, text format, and destination for your output.

To get help on any field in the frame, put your cursor on a field and press the function key that you use to issue the HELP command. The fields in the CMS Print File Parameters frame correspond to the parameters you would use in a CP TAG or SPOOL command, so you can also receive help from the CMS system. Refer to the TAG and SPOOL entries in VM CP General User Command Reference for more information.
