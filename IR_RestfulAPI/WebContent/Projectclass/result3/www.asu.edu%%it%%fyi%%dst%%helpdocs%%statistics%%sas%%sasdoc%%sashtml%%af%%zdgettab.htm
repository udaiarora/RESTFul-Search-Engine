Widget Class: _tab

Widget Class: _tab


Indicates whether a widget was tabbed to

Syntax

Syntax

objectName._tab( direction, inOut );

Argument Type Use Description
direction Numeric Input specifies a value indicating the direction of tabbing: 1 tabs forward, -1 tabs backward
inOut Numeric Output returns a value indicating where the cursor has been moved: 1 tabbed to the widget, 0 tabbed out of the widget
