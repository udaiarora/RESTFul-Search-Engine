Annotate Variables

When an Annotate data set is processed, the Annotate facility looks at the values of specific variables in order to draw graphics. This section describes all of the Annotate variables in alphabetical order. Not all variables are used with all functions. Refer to the description of the individual functions in Annotate Functions for more information about how each variable is used with each function. For a summary of Annotate variables and their uses, see Summary of Annotate Variables.
