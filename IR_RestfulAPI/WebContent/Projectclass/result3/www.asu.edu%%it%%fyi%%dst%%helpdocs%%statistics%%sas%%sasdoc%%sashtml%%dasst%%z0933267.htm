Editing a Graph Interactively

If you create a graph, you can edit the graph interactively. From the Report Output window, select Edit Report from the Edit menu to invoke the SAS/GRAPH software interactive graphics editor, as shown in the following display. You must have SAS/GRAPH software installed to edit a graph interactively. Refer to the online help for this window for more information.

Editing a Graph Interactively

To return to the Report Output window, select End from the File menu.
