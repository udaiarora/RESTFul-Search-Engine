USER=

USER=


Specifies the location of the default SAS data library

Default: none
Valid in: configuration file, SAS invocation, OPTIONS statement, OPTIONS window
Category: Environment Conrol: ENVFILES
OS/390 specifics: library-specification

Syntax
See Also

Syntax

USER=library-specification

library-specification
can be a DDname that was previously associated with a SAS data library, the name of a physical file that comprises a SAS data library, or a UNIX System Services directory.
