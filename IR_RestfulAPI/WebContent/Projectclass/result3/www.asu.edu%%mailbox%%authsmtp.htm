Authenticated SMTP

 
ASU's Authenticated SMTP service has been created in an effort to give ASU e-mail users the ability to send using their ASU IMAP Client while traveling. Your ASURITE ID and password will be the information required to authenticate with the ASU AUTH SMTP service.

The following links will provide you with instructions on setting up Authenticated SMTP on the respective clients.

Eudora (Mac)

  1. Start Eudora
  2. Click the Eudora menu in the upper-left corner of the screen and select Preferences.
  3. Click Getting Started on the left side of the page.
  4. In the SMTP Server field, enter: smtp.asu.edu.
  5. Click Sending Mail on the left side of the page.
  6. Ensure that Use submission port (587) and Allow authorization are both checked.
  7. Scroll down and select SSL on the left side of the page.
  8. For SSL for SMTP, select Required (TLS)
  9. Click OK.
     

Entourage

  1. Start Microsoft Entourage
  2. Click the Entourage menu and select Account Settings.
  3. Select your account and click the Edit button.
  4. Click Click here for advanced sending options.
  5. Ensure that SMTP service requires secure connection (SSL) is checked.
  6. Ensure that Override default SMTP port is checked, and then enter 587 in the blank spaces.
  7. Ensure that SMTP server requires authentication is checked.
  8. Select Log on using and then enter your ASURITE User ID and password in the appropriate fields.
  9. Close the dialog box, and then click OK to save your changes.

Apple Mail

  1. Start Mail
  2. Click on the Mail menu and select Preferences.
  3. Click on the Accounts tab on the top of the page.
  4. Click the Server Settings button.
  5. In the Outgoing Mail Server field, enter: smtp.asu.edu.
  6. In the Server port field, enter: 587.
  7. Ensure that Use Secure Sockets Layer (SSL) is checked.
  8. Under Authentication, select Password.
  9. Enter your ASURITE User ID in the User Name field.
  10. Enter your ASURITE User Password in the Password field.

Thunderbird

  1. Start Thunderbird.
  2. Click on the  Tools menu and select Account Settings.
  3. Click Outgoing Server (SMTP) on the left side of the page.
  4. Click the Edit button.
  5. The Description field can be left blank.
  6. In the Server name field, enter: SMTP.ASU.EDU
  7. In the Port field, enter: 587.
  8. Ensure that Use name and password is checked.
  9. In the User Name  field, enter your ASURITE User ID.
  10. In the Password field, enter your ASURITE password.
  11. Under Use secure connection, choose TLS.
  12. Click OK to save your changes, and then click OK once more to exit the Account Settings dialog box.

 

Outlook 2003

  1. Start Outlook
  2. Click on the Tools menu and select E-mail Accounts.
  3. With the radio button next to View or change an existing e-mail account, click next.
  4. Highlight the E-mail account you would like to edit and click Change.
  5. Ensure that the Outgoing mail server is SMTP.ASU.EDU and click More Settings...
  6. Click on the Outgoing Server tab and put a checkmark in the checkbox next to My outgoing server (SMTP) requires authentication
  7. Click on the radio button next to Log on using.
  8. Enter your ASURITE ID and Password in the fields provided.
  9. Click on the Advanced tab.
  10. Change the default Outgoing Server port to 587
  11. Click on the box next to This server requires an encrypted connection (SSL) putting a checkmark in that box.
  12. Click OK to exit this window.
  13. Click Next, and then Finish to exit. 

Outlook Express

  1. Open Outlook Express
  2. Click on the Tools menu and choose Accounts.
  3. Under the Mail tab, highlight the account that you want to change and click Properties.
  4. In the servers tab of the next window, place a check in the box next to My server requires authentication and click Settings.
  5. Click Log on using, and enter your ASURITE ID and Password in the fields provided.
  6. Click OK to get back to the Properties Window.
  7. Click on the Advanced tab, and place a checkmark next to This server requires a secure connection (SSL).
  8. Change the default Outgoing Mail (SMTP) port to 465.
  9. Click OK to exit the properties window, and then Close to exit the Accounts window.
