Setting the Angle of Tilt or Rotation in a Histogram Control

For a Histogram control, the rotation angle is the current angle relative to the Y axis. Similarly, the tilt angle is the angle relative to the X axis.

To set a rotation angle or tilt angle, use the rotationAngle or tiltAngle attributes. For example, for a control named histogram1, the following code sets the rotationAngle attribute to 35 degrees: 

histogram1.rotationAngle = 35;

You can also use the actionMode attribute to adjust these angles interactively. Set the actionMode to Tilt to adjust the tiltAngle, or Rotate to adjust the rotationAngle (see Setting the Action Mode in a Histogram Control). To set both angles at once, set the actionMode to Spin.
