Overview

The External File Model class allows a text viewer to display an external file. This model is automatically attached to an external file viewer (see the External File Viewer class) and may also be attached to a text viewer (see the Text Viewer class).

In addition the external file model can be subclassed to create a custom model that provides more specific functionality or displays text from a source not already covered with the external file model.

The External File Model class is a subclass of the Text Model Template class. See the Text Model Template class.

The External File Model and related classes use the model/view paradigm are described in SAS/AF online help.
