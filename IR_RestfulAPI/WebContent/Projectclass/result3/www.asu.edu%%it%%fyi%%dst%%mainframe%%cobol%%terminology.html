Terminology

The following is typical IBM terminology that will be used throughout this help topic.

Cataloged procedure
A previously stored set of JCL statements.
Symbolic parameter
A variable, coded into a cataloged procedure, whose value can be changed when the procedure is executed.
Source module
COBOL language statements, input to the COBOL compiler.
Object module
Machine instructions output by the COBOL compiler.
Load module
An executable program produced by the linkage editor using object modules and/or other load modules.
Loader
A system program which performs the linkage editor functions and immediately executes the program (a load module cannot be saved).
