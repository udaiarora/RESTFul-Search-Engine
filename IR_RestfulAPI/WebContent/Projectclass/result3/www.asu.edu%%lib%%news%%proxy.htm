Researching from home just got easier!


The library has modified the authentication and authorization procedures to allow off campus ASU users to access almost all electronic journals, indexes, abstracts and databases from the library's web site. To take advantage of this, go to the E-journals page or Find Articles & More to locate the resource you want to access. (Please note: links from the online catalog may not work at this time. If you find a link in the catalog which is not working from off campus, try the title in the e-journal page.)

If you are off campus, click on the "Off Campus Users Sign In" button at the top of the page or to the left of the link to connect to the index on the Information page for the resource you are accessing (see below) .

Resource Information Page


You will be prompted to enter your name, university ID and library PIN. This is the same information that is already used to gain access to your library account ("My Account") or course reserves. (See image below)




Affiliated ASU students, faculty and staff are automatically added to the library system. It is up to each user to create a library PIN to use the authentication/authorization system. If you do not have a PIN yet, please set one using the "My Account" feature in the library online catalog. (Note: Library PINs are independent from those used for the university's In-Touch system. Even if you already have an In-Touch PIN, you will still need to set a library PIN.)

When you have successfully authenticated, you will get a message letting you know you now have access to the library resources and can continue.

If you are having problems authenticating your ASU status, please contact Circulation at 480-965-3605.

If you need any other assistance, or have any questions, please contact Ask a Librarian.
