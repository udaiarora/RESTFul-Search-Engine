SAS Technical Report P-246, SAS/TOOLKIT Software: Graphics Capabilities, Release 6.08

Use this book to learn how to write a graphics procedure using SAS/TOOLKIT software. The library of graphics routines described in this report is based on the graphics kernal system (GKS) standard.

140 pages.
