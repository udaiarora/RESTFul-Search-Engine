$KANJIXw.


Removes shift code data from DBCS data

Category: DBCS
Alignment: left

Syntax
Syntax Description
Details

Syntax

$KANJIXw.

Syntax Description

w
specifies the width of the output field.
Restriction: The width must be an even number. If it is an odd number, it is truncated.
Range: The minimum width of the format is 2.

Details

The input data length must be 2 + (SO/SI length)*2. The data must start with SO and end with SI, unless single-byte data are returned. This format always returns a blank for DBCSTYPE data that do not use a shift-code mechanism.
