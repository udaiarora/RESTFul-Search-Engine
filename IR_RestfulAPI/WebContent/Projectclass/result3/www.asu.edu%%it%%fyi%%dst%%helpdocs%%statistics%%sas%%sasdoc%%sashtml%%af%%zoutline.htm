Widget Class: _setOutline

Widget Class: _setOutline


Sets the border style of an object

Syntax

Syntax

objectName._setOutline( style );
objectName._setOutline( style, width );
objectName._setOutline( style,width, lightSource );

Argument Type Use Description
style Character Input specifies the border style: SIMPLE, SHADOW, BUTTON, EMBOSS, NONE
width Numeric Input specifies the width of the border
lightSource Character Input
