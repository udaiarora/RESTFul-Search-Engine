Frame Class: _addDragRep

Frame Class: _addDragRep


Adds or registers this representation as a valid drag representation

Syntax
Details

Syntax

return = objectName._addDragRep( );

Details

A drag site is defined the first time a representation is defined for the object. During the drag, the list of representations is matched with the drop sites. The drop is allowed when there is at least one representation match.
