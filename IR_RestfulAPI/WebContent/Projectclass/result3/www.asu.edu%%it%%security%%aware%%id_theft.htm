Identity Theft

Security Issues and Identity Theft

Computer security and identity theft is an issue for everyone. It is important to keep in mind the safety and security of your personal information at all times, and if you feel that your identity has been compromised contact DPS immediately.

To see ASU's policy on privacy, please visit www.asu.edu/privacy.

For more information on the risks of identity theft and ways to protect yourself, see the U.S. Government's central website for information about identity theft.

Federal Trade Commission ID Theft Web site

Good Practices

Information Technology has some basic guidelines to follow and provides information on ways you can secure your workstation. It is a good idea to follow these basic rules in order to reduce your risk of identity theft.

If You Log In, Log Out

Whenever you log in to a computer or a Web site it is important to log out and end your session in order to prevent other users from accessing your personal information.

Change Your Password Often

When selecting new passwords, don’t pick familiar words that might be associated with you, and use combinations of letters and numbers. Finally, don’t share your password with anyone.

Use Secure Web Sites

When entering any personal information in a Web page, make sure the Web address starts with https://.

Securing Workstations

With all the services and access that IT provides for you, it is important to protect your computer from viruses and outside intruders who could modify or delete your files. By keeping your machine secure, you also keep intruders from accessing ASU’s computer network while you’re connected to the servers. There are basic things you can do to protect your machine. ASU provides detailed instructions on doing this on the Computer Security Web page.

The Basics

  • Install latest OS
  • Install latest browser software
  • Install antivirus software
  • Download latest antivirus updates
  • Install firewall software or hardware
  • Use on-line service to test security
