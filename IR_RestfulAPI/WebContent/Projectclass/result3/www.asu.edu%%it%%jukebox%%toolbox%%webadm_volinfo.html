Webadmin - Volume and Space Usage Information

Example Screen:

www.YOURDIRECTORY                537333873 RW     332652 K  On-line
    filesrv6.asu.edu /vicepgj
    RWrite  537333873 ROnly          0 Backup  537333875
    MaxQuota     400000 K
    Creation    Fri Feb  7 14:42:36 1997
    Last Update Fri Apr 28 10:57:11 2000
    95435 accesses in the past day (i.e., vnode references)

    RWrite: 537333873     Backup: 537333875
    number of sites -> 1
       server filesrv6.asu.edu partition /vicepgj RW Site
Volume Name                   Quota      Used %Used   Partition
www.YOURDIRECTORY            400000    332652   83%         73%

Space Usage Report (may take a while ...)

What It Means:

  • filesrv1.ASU.EDU /vicepgj: File server information. The /vicepgj section will change depending on your directory.
  • RWrite: Read/Write Information.
  • ROnly: Read Only Information.
  • Backup: Backup Information.
  • MaxQuota: The quota set on this volume.
  • Creation: When the volume was created.
  • Last Update: When the volume was last updated.
  • Accesses in the Past Day: Total accesses in the last day.
  • Number of Sites: Other severs with this volume.
  • Quota Details: How much space is available in your volume.
  • Space Usage Report: Detailed information regarding the usage of your volume.
