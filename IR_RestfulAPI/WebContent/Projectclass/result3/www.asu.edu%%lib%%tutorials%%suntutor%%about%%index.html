About SunTutor

SunTutor is a web-based research tutorial developed by the Arizona State University Libraries. The tutorial is designed for use by individuals and as part of course-integrated instruction. Its focus is on general research skills that will be applicable in any library, but the tutorial also explains how to access specific tools through the ASU Libraries' web site.

About the Modules

There are currently four modules included in SunTutor. Each module contains a basic lesson on a particular aspect of research. Quizzes for each module are available through my.asu.
