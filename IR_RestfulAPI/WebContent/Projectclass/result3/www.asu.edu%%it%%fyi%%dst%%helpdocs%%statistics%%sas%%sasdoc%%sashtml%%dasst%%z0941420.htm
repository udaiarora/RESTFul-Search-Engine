Introduction

This appendix describes how to migrate IBM QMF queries to Query and Reporting. With the query migration tool, you can export your QMF queries and move them to Query and Reporting through a three-step process. You then can use the SQL queries in Query and Reporting just as you would use your other queries. Individual users or a system administrator can perform this migration process, depending on each site's setup. The migration process consists of three stages:

  1. Generate an export procedure in SAS that can be run in QMF.

  2. Run the export procedure in QMF.

  3. Import the queries into SAS/ASSIST Query and Reporting.

Each of these steps can be executed independently.
