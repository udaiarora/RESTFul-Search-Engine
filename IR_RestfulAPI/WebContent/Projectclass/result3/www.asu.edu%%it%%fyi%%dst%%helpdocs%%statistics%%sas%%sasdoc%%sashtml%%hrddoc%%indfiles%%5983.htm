SAS Technical Report R-104, A Simplified Algorithm for the W-Transformation in Variance Component Estimation

Presents an efficient algorithm needing only about twelve lines of FORTRAN or PL/1 code to compute the W-transformation. Each step in the maximum-likelihood or restricted-likelihood procedures requires the W-transformation for estimation of the parameters of a mixed A.O.V. model.

12 pages.
