Selecting the Processing Mode for the Load

Two modes of processing are available when you are loading data with the DBLOAD procedure: insert mode and optimized load mode. Optimized load mode is a fast, efficient way to add new logical entries to the database. Insert mode must be used to add data records to existing logical entries. For details about these processing modes, see the S2KLOAD statement discussion in Procedure Statements.
