Derwin Skipp
Systems Programmer, Principal
Arizona State University
Enterprise Dynamic Network Authorization

Bio

Derwin Skipp is a Systems Programmer, Principal with Information Technology at Arizona State University. He is the lead programmer and leader of the university level network authentication and authorization services development team.

Presentation Abstract

Project EDNA provides a scalable enterprise management system for managing authentication and authorization of user access to university computing resources at Arizona State University. The Enterprise Distributed Network Authorization interfaces with university access control systems to provide dynamic, real-time account management to more than 200,000 users and 1,000,000 subscriptions. EDNA also provides automated authorization decisions as a network service to IT applications via a client API.
