Web Catalogs: PDF and HTML Postings

Catalog text postings

Note: The Web catalogs (PDF and HTML), as well as the print catalogs, are now managed by AAD.

Although the “beauty” of the Web is that updated information can be communicated frequently, catalog-related information is posted on the Web only once a year.

The print catalogs are converted and posted in the spring of each year.

PDF catalogs appear first. The PDF version precedes the arrival of the print catalogs on campus. PDF text and courses match the print catalog.
HTML catalogs primarily match the print edition. The HTML version is posted in pieces and is completed approximately one month later.
Because the Web is interactive, the following are updated:
•incorrect e-mail or Web addresses
•text references to courses (e.g., BIO 467) that don’t exist
•incorrect phone numbers and fax numbers and room locations
Only incorrect e-mail or Web addresses in the text of the most currentn converted catalogs will be changed. Any substantive change made to the printed catalogs (e.g., to e-mail or Web addresses) must be accompanied by a paper printout to University Archives for the official record. These issues are supported by General Counsel and University Archives.

Exceptions should be submitted in writing to the Office of the Executive Vice President and Provost of the University.

E-mail addresses

Working departmental e-mail addresses are not changed. However, if the e-mail address is currently a personal one changing to another personal one, or if the address was incorrect (regardless of whether it was a personal or departmental address), then the requested changes are made.

Our goal is to present a professional image to the Web audience and to minimize Web updates when e-mail or Web addresses change. Therefore, departments and programs are encouraged to choose an e-mail (or Web) address that will be stable, at least for six months but preferably a year and longer.

One thing you can do is create an alias that looks like a professional or departmental address, even if it is to a personal account (similar to “web.catalogs@asu.edu” going to a personal e-mail account). If that person leaves, then the same alias can be used for the next person (if a true departmental account isn’t obtained yet for that purpose).

You can create your own alias by going to the secure site, www-sec.asu.edu/epo/index2.html to access the EPO Account Update screen. It may take a day for the created alias to take effect.

Web addresses

It’s best to design your Web site with the idea that file names and file locations will remain stable. Renaming or moving your files and folders in your Web site changes the Web address. Whenever a file or folder name or location changes, links from other pages or sites to your file become invalid unless also changed. Some Web surfers like to type in Web addresses rather than find pages with links, so in general, a shorter Web address is better than a longer one.

Archived copies of Web catalogs

Current and past catalogs are on the Web in PDF and HTML formats.

Archived copies of catalogs and courses can be found from the main catalog page.

Note that course information is now available in a database. To find information dating back to spring 2002, select the appropriate semester and year.
