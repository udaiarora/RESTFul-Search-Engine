DATEw.

DATEw.


Writes date values in the form ddmmmyy or ddmmmyyyy

Category: Date and Time
Alignment: right

Syntax
Syntax Description
Details
Examples
See Also

Syntax

DATEw.

Syntax Description

w
specifies the width of the output field.
Default: 7
Range: 5-9
Tip: Use a width of 9 to print a 4-digit year.

Details

The DATEw. format writes SAS date values in the form ddmmmyy or ddmmmyyyy, where

dd
is an integer that represents the day of the month.

mmm
is the first three letters of the month name.

yy or yyyy
is a two- or four-digit integer that represents the year.

Examples

The example table uses the input value of 15415, which is the SAS date value that corresponds to March 16, 2002.
