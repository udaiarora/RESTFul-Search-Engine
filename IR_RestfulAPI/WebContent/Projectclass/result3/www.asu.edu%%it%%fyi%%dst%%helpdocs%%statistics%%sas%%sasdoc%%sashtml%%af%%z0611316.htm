Creating a Chart Control

To create a Chart control, open a frame and select a Chart Control from the Components window. This instantiates the control for you. You can name the control by opening the Property sheet and assigning a name on the name attribute.

You can then perform the following tasks:

  • Specifying Data in a Chart Control
  • Specifying the Chart Type in a Chart Control
  • Specifying Variables and a Viewing Mode for a Chart Control

Before a graph is drawn, you must specify the data set and at least two variables. You can set other attributes before this is done, but no graph is drawn until the data set and variables are specified.

If you enter the data set and variables through the Property sheet in build mode, the graph is displayed as soon as you enter the minimum information. If you use SCL to enter the information, the graph is displayed as soon as the minimum information is set and a Return statement is encountered. If you specify a data set or variable that does not exist, a warning is written to the SAS log, and no graph is drawn.

In build mode, using the Property sheet:

  • as soon as you enter the minimum information the XDataMin, YDataMin, ZDataMin and XDataMax, YDataMax, ZDataMax attributes are filled in with the minimum and maximum values from the data set. Also, if the Y variable is character data, then the YVariableType value is set to Categorical and is no longer editable until you change the variable. This is because character data cannot be treated numerically.

  • as you change attributes in the Property sheet, variable types do not change automatically every time. For example, if you assign a character variable to YVariable, the variable type will change to Categorical. If you then assign a numerical variable to YVariable, the variable type will still be categorical because it is valid to treat numerical data categorically.
