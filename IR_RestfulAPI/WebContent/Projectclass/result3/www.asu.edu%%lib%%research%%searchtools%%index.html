Web Search Tools

Search Engines and Directories

 

Search Engines

Search engines allow us to search the vast information-rich World Wide Web and provide full-text access to billions of web pages. Many search engines allow you to limit your search by media type such as image, movie, or sound files.

  • Altavista.com
  • AlltheWeb.com
  • Teoma.com

 

Web Directories

Web subject directories, also known as gateways or portals, are web site listings organized by subject.

 
  • Yahoo!
  • Librarian’s Index to the Internet
  • Open Directory Project

If you’re having trouble finding a web directory specific to your area of research, it’s always helpful to check the ASU Libraries' Subject Guides.

 

Meta-Search Engines

Meta-search engines search several search engines at once. These tools are useful for simple keyword searches.

  • MetaCrawler.com

  • Mamma.com
    •
