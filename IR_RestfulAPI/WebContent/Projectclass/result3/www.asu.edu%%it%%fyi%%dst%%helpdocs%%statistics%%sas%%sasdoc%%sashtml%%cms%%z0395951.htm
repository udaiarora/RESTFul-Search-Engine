CLOCK

CLOCK


Displays the current time according to a 24-hour clock

CMS specifics: all

Syntax
Details

Syntax

CLOCK

Details

The time is shown as hh.mm in the lower-right corner of the display. For example, 1:00 p.m. is displayed as 13:00. Repeat the command to toggle the clock on and off, or issue the command CLOCK OFF.
