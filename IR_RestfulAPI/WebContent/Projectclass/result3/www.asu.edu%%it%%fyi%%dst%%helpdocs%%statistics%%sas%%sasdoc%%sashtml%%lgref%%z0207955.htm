FLOOR

FLOOR


Returns the largest integer that is less than or equal to the argument

Category: Truncation

Syntax
Arguments
Details
Examples

Syntax

FLOOR(argument)

Arguments

argument
is numeric.

Details

If the argument is within 10**(-12) of an integer, the function returns that integer.

Examples

SAS Statements Results 
var1=2.1;
a=floor(var1);
put a;
 
 

2
 
var2=-2.4;
b=floor(var2);
put b;
 
 

-3
 
c=floor(3);
put c;
 
 
3
 
d=floor(-1.6);
put d;
 
 
-2
 
e=floor(1.-1.e-13);
put e;
