FTEXT

FTEXT


Selects the default font for all text.

Used in: GOPTIONS statement
Default: Default hardware font (except the first title)
See also: FTITLE

Syntax
Details

Syntax

FTEXT=text-font

text-font
specifies the font for all text on the graphics output. See SAS/GRAPH Fonts for information about specifying fonts.

Featured in: Example 6. Enhancing Titles

Details

The FTITLE= graphics option overrides FTEXT= for the first title.
