How to find books

Searching

Searching by author

If you know the author of a book, type it into the search box, last name first.

For Example: If you are looking for the book written by Stephen Ambrose, you would type "Ambrose Stephen" into the search box and click "Search".

If the library owns books by that author, a screen with information about them will appear.

If not, you will see a list of similar book author names, and a message "Your entry would be here." If that happens, the library does not have the book you are looking for or you do not have the correct author. Ask a librarian for help.
