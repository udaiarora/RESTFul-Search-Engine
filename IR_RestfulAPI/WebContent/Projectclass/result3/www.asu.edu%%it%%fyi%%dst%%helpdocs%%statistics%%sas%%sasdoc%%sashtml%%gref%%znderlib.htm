RENDERLIB

RENDERLIB


Specifies the SAS data library in which rendered font files are stored.

Used in: GOPTIONS statement
Default: WORK
See also: RENDER

Syntax

Syntax

RENDERLIB=libref

libref
specifies a previously defined libref that identifies the SAS library. The default library is WORK. See LIBNAME Statement for more information on assigning a libref.
