Roles and Personnel Responsibilities

Functional Techs

  1. Assist functional groups in their IDP sessions.
  2. First group of technicians who respond to customization requests from functional groups.

Development

  1. First group of technicians to implement customizations that are owned by functional groups without technical representation.
  2. Assist functional techs with customization requests as necessary.

Interfaces

  1. Implement interfaces from current ASU legacy systems to PeopleSoft systems as necessary.

Workflow

  1. Assist functional groups develop and implement workflow requirements.

Security

  1. Develop security strategy.
  2. Develop functional requirements for security.
  3. Develop technical specifications for security.
  4. Configure application and database security.

Helpdesk

  1. Develop and implement Helpdesk plan.
