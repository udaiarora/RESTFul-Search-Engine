Benefactor/Colleague Client Software

Back to IRM
 

Required Software:

  • User Interface 1.0
  • SecuRemote

 

Optional Software:

  • Acrobat Reader
  • Print Key
  • Barcode Fonts
With appropriate client software, Benefactor and Colleague may be accessed from your Microsoft Windows workstation. IRM will assist your computing support staff by providing access to the software along with instructions for installing it. If IRM is your primary support for PC hardware and software, we will install it for you. Some Benefactor and Colleague functions are available via web browser. Visit the ASU Foundation web site for details.

User Interface 1.0 - UI 1.0 is the required MS Windows-based client software for accessing ASU's Benefactor and Colleague Systems.

SecuRemote - The Benefactor/Colleague host is secured with a CheckPoint Internet Firewall. This firewall participates in the campus-wide firewall system.

Adobe Acrobat Reader - Adobe Acrobat Reader is the free viewing companion to Adobe Acrobat. Acrobat Reader lets you view, navigate and print Portable Document Format ( PDF) files. Benefactor and Colleague print formatted reports delivered via the Web are prepared in the PDF format, and require Adobe Acrobat Reader for display.

PrintKey2000 - This is a freeware print screen utility for Windows that facilitates screen printing in the graphical mode of User Interface 1.0 (which does not include a print screen feature).

Bar Code Fonts - Correspondence produced through Benefactor can include bar coded merge fields to expedite transaction entry. These fields can be incorporated into solicitations, acknowledgements, reminders, and receipts. Benefactor operators that will be producing correspondence from the system must install these fonts on their workstations, in order to correctly print a document that includes bar coding. These fonts were purchased under a site license agreement and are intended for use on ASU and ASU Foundation owned equipment.
