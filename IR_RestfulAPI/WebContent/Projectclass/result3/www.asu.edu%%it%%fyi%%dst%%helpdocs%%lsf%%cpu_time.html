How can I check the CPU time consumed by my current jobs?

There is a command currently available for doing just that. Simply enter the command:

chk_cpu user_id
Where, chk_cpu initiates a job on the batch server and returns with the CPU time information for your job(s) in a file called "chk_cpu.o.e". The CPU time is reported as hh:mm:ss (hours:minutes:seconds). The parameter "user_id" is your logon id (ASURITE id).

Part of the information you will see in the "chk_cpu.o.e" file will look similar to: 

The output (if any) follows: 
                                                
Please note that your job information is at the bottom. Where:

UID=UserId, PID=ProcessID, STIME=StartTime, TIME=CPU_Time, CMD=JobName

     UID   PID  PPID   C    STIME    TTY  TIME CMD
 user_id 34550 57596  91 10:11:17      - 273:01 sas -nocenter driver50.sas   
 user_id 47226 63076  90 10:08:41      - 274:46 sas -nocenter driver25.sas   
 user_id 50800 39908   1 14:45:26      -  0:00 grep user_id
Where 273:01 is the CPU time for the SAS job driver50.sas and 274:46 is the CPU time for the SAS job driver25.sas.
