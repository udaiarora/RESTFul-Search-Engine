Webapp Development Resources - Reference Links

General

  • The Web and the Internet
    • The World Wide Web Consortium (W3C) home page, with everything Web. These are the guys who define it, still led by Tim Berners-Lee, who invented it.
  • Internet glossary
    • A glossary of Internet-related terms.

Architecture

  • Web applications overview
    • Great tutorial on Web applications architecture by Selena Sol. Diagrams the four basic layers of Web architecture and the way that individual technologies like CGI, Corba, XML, Java, and a bunch of others fit into them.

Philosophy and culture

  • Transitioning from legacy to objects
    • A 1998 book, "Transition to Object-Oriented Software Development".
    • A 1995 book, "Business Engineering with Object Technology" by David Taylor.
    • Also, "Object Technology: A Manager's Guide" by David Taylor.

Languages

  • HTML help
    • The Bare Bones Guide to HTML in cheat-sheet format.
    • Another HTML cheatsheet, this one from HotWired's "Webmonkey".
    • The NCSA's HTML Primer.
  • XML info
    • Introduction to XML for Web developers, by Selena Sol.
    • What XML means to the Enterprise (paraphrased title), an article from Sun Microsystems.
    • The SGML/XML Web Page, a big fat up-to-date list of XML resources.
    • The W3C's XML page from the guys who write the specs.
  • Java and J2EE links
  • JavaScript resources
    • Netscape's online "JavaScript Reference".
    • Netscape's online "JavaScript Guide".

OS's, editors, utilities

  • UNIX help
    • UNIX quick-ref.
    • A simple UNIX guide, also from "Webmonkey".
    • Geekgirl's compendium of UNIX resource links.
    • Introduction to UNIX for Web developers ("for liberal arts hackers gone Webmasters"). By Selena Sol.
  • Help with UNIX editors
    • Quick-ref page for Vi.
    • An introduction to Vi, written by the guy who wrote Vi.
    • The Vi-lover's home page.
    • PICO tutorial.
    • Emacs quick-ref page from geek-girl.
  • AFS help
    • AFS Beginners Guide from MIT.

Security

  • General Web security info
    • The W3C's security page. Contains links to resources for secure protocols, e-commerce and encryption. Maintained by Lincoln Stein.
    • The NIH's security page. Good links to articles, advisory groups, Web sites, discussion lists, etc.
    • Javaworld's Web security nuts and bolts. Printable overview with good definitions.
  • Web Application security
    • The OWASP Guide to Building Secure Web Applications. Contains security information for Web application developers using various tools/languages.
    • Introduction to SQL Injection Attacks. Explanation of a common Web application security issue. Somewhat specific to Oracle but concept applies to all databases.
    • Injection Protection. Basic steps for protecting your Web application from SQL Injection Attacks.

Site, Page design

  • Accessibility guidelines
    • The W3C's accessibility guidelines for the Web.
    • IT's own IS Lab's accessibility guidelines. Very nicely done.

Higher Ed IT and Web dev

  • Groups, lists, conferences
    • Site for the annual WebdevShare conference in Indiana which focuses mostly on administrative applications. Includes archives of some presentations.
    • The Java in Administration Special Interest Group (JA-SIG).
    • CUMREC, the College and University Computer Users Association, which holds an annual conference.
    • EDUCAUSE, an association for higher ed and information technology organizations and professionals. Big annual conference.
