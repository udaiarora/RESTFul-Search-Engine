Frame Class: _getSelectedWidgets


Fills an SCL list with all widgets that are currently selected in the frame

Syntax

Syntax

objectName._getSelectedWidgets( widgets );

Argument Type Use Description
widgets List Update specifies the identifier of the SCL list that when returned contains all currently selected widgets in the frame
