Engineering and Scientific Subroutine Library (ESSL)

ESSL is a set of high performance mathematical routines for use by FORTRAN and assembler language programs. There are two versions of the library: Scalar and Vector. Only the Scalar library is usable at ASU.

DOCUMENTATION

  • IBM Manual "Engineering and Scientific Subroutine Library - Guide and Reference" No. SC23-0184. This document is available for reference at the Customer Assistance Center.

An example program is shown below. This examples uses the short-precision routine: SNDOT.

   // JOB
   // EXEC FORV2CLG,
   // LIBB='SYS2.ESSL.SCALAR.LIB'
   //FORT.SYSIN DD *
   C  test of essl routine SNDOT: short-precision
   C  SNDOT computes special dot product n times
   C
         REAL*4 X(4,3),Y(4,3),S(3)
              |1.0 2.0 3.0|
         X=   |2.0 3.0 4.0|
              |3.0 4.0 5.0| <-- data for X matrix
              |4.0 5.0 6.0|  (arrange in FORTRAN statements)
                    ...
              |4.0 3.0 2.0|
         Y=   |3.0 2.0 1.0|
              |2.0 1.0 4.0| <-- data for Y matrix
              |1.0 4.0 3.0|
         N=3
         M=4
         INCS=1
         ISW=1
         INCXI=1
         INCXO=4
         INCYI=1
         INCYO=4
         CALL SNDOT(N,M,S,INCS,ISW,X,INCXI,INCXO,Y,INCYI,INCYO)
             ...
            i/o statements
             ...
         STOP
         END
   /*
   //
