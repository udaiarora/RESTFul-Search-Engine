INTERPOL

INTERPOL


Sets the default interpolation value for the SYMBOL statement.

Used in: GOPTIONS statement

Syntax

Syntax

INTERPOL=interpolation-method

interpolation-method
specifies the default interpolation to be used when the INTERPOL= option is not specified in the SYMBOL statement. See SYMBOL Statement for the complete syntax of all interpolation methods.
