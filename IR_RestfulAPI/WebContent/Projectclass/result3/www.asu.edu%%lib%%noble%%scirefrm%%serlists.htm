Journal Abbreviation and Serial Lists

 

The following journal/serial lists may be used to find the full title for journal abbreviations and/or determine if a database indexes a specific journal(s).

  • General/Multidisciplinary Lists
  • Specific Database Lists

General/Multidisciplinary Lists

  • Acronyms, Initialisms, and Abbreviations Dictionary.
    P365 .A3x Science Reference (located at Reference Desk)

  • All That JAS (Journal Abbreviation Sources)
    A list of Web resources that list or provide access to the full title of journal abbreviations; arranged by subject.

  • CISTI (NRC's Canadian Institute for Scientific and Technical Institute)
    • Catalogue
  • Genamics.com
    • JournalSeek
  • jake
    jake is a tool designed primarily for locating where particular journals are available in electronic form. However you can search by journal abbreviation, and a list of potential journal titles are returned.

  • Periodical Title Abbreviations.
    PN4832 .A4x Science Reference (located at Reference Desk)

  • World List of Scientific Periodicals Published in the Years 1900-1960.
    Q158.5 .W68x Science Reference (located at Reference Desk)
