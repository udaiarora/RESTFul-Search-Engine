Generic Keyboard Mapping

A 3270-type terminal has the following special keys: PF1, PF2, ..., PF24, tab, backtab, reset, insert, delete, newline, enter, erase-eof, and others. Generic escape-key and control-key sequences have been defined to emulate most of these keys. A customized keyboard mapping (created with the key utility) may be necessary if you want to use 3270 keys not listed below.


     generic  3270            generic  3270
     -------  ----            -------  ----
     ctl A    insert          ctl L    newline
     ctl D    delete          ctl R    reset
     ctl E    erase-eof       ctl T    insert tab character
     esc 1    PF1             esc !    PF13
     esc 2    PF2             esc @    PF14
     esc 3    PF3             esc #    PF15
     esc 4    PF4             esc $    PF16
     esc 5    PF5             esc %    PF17
     esc 6    PF6             esc ^    PF18
     esc 7    PF7             esc &    PF19
     esc 8    PF8             esc *    PF20
     esc 9    PF9             esc (    PF21
     esc 0    PF10            esc )    PF22
     esc -    PF11            esc _    PF23
     esc =    PF12            esc +    PF24

If you include the option -NOKEYBIND on the command line when invoking the editor, the generic key sequences will not be available; only the customized keyboard mapping will be available.

If you omit the -NOKEYBIND option, the generic key sequences and customized keyboard mapping will both be available for use in the editing session.
