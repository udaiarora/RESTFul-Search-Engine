Adding Libraries

When you load and link objects, you need to link it with the appropriate library so all the references (calls, to functions and routines) can be resolved. By default, the loader loads some of the standard libraries, but sometimes libraries such as netlib, you need to add them yourself.

For example, to link with the math library in C (libm.a) which is usually located in /usr/lib and/or /usr/local/lib , you simply type

cc program.c -lm

The header is math.h, and only including the header #include <math.h> is not adequate if you wish to use math function such as sqrt, sine, cosine.
