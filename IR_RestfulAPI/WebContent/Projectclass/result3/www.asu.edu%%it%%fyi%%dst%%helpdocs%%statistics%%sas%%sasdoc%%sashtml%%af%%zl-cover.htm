Overview

The Text Label Control creates a text field which is not editable by users. Text labels typically describe the type of information expected in an input field. They can also be used to provide helpful information not associated with an input field.

Text labels are limited to one line and 200 characters.

You can set the label attribute at build-time, or based on user input, use SCL to set the attribute at run-time.
