Learning Outcomes: Find Books

This module explains how to use an online library catalog to locate books. It also discusses how to find books available in the ASU Libraries.

Why Books?
  • The student can recognize when they should use books.
Using the Catalog
  • The student can recognize that the catalog is the tool to use to find books.
Basic Ways to Search Catalog
  • The student will identify the three basic ways to search the catalog.
Breaking Research Questions into Keywords
  • The student will reduce research questions into concepts and then identify appropriate keywords for each concept.
Refining the Search Strategy
  • The student will recognize when it is necessary to refine their search.
  • The student demonstrates techniques to refine their search strategy.
Truncation
  • The student will use truncation to broaden their search.
Catalog Records and Information Screens
  • The student will interpret the location, call number and status information in order to locate the book.
  • The student will recognize the bibliographic elements from the catalog.
Requesting Books
  • Student will select appropriate action to obtain a book that is checked out (or at West, etc.).
Renewing Books
  • The student will select the appropriate action to renew books themselves.
Who Can Help?
  • The student will know to ask a librarian for help when needed.
