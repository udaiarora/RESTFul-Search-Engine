SYMBOLGEN


Controls whether the results of resolving macro variable references are written to the SAS log

Valid in: configuration file, SAS invocation, OPTIONS statement, Systems Options window
Category: Macro: SAS macro
PROC OPTIONS GROUP= MACRO

Syntax
Syntax Description
Details

Syntax

SYMBOLGEN | NOSYMBOLGEN

Syntax Description

SYMBOLGEN
displays the results of resolving macro variable references.
Tip: Use SYMBOLGEN for debugging programs.
CAUTION:
Using SYMBOLGEN can cause the log to become large.   

NOSYMBOLGEN
does not display the results of resolving macro variable references.

Details

SYMBOLGEN specifies that SAS display the results in this form: 

SYMBOLGEN: Macro variable name
      resolves to value
