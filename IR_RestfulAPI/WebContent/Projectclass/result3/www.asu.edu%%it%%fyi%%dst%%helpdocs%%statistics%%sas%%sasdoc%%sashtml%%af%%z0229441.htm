Graphic Text: _setAspect


Assigns the text's aspect ratio (the ratio of width to height)

Syntax

Syntax

CALL NOTIFY (grtext-name, '_setAspect', aspect-num);

Argument Type Description
aspect-num
 N assigns a value that describes how to scale the text:


0 use defined aspect for the font


1 scale text string to region height


2 scale text string to region width


3 scale text string to both region height and width
