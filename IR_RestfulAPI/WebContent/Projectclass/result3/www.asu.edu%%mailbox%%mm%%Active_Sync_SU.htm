Setting Up Your Active Sync

These step by step directions will assist you in setting up your windows device for synchronization with your Outlook E-Mail. This document makes the assumption that you have already installed Microsoft Active Sync Software that has come with your device. If you do not have this software installed, install it now. If you do not have a copy of the software, you may go to www.microsoft.com/downloads and search for the keyword ActiveSync.

If you already have a device which is syncing properly, and you are just looking to take advantage of the Microsoft Always up to Date notifications, click here to be taken to those instructions.  The Always up to Date Notifications are designed to help your device stay current with your Exchange account until the release of Microsoft's MSFP.

You must already have installed Active Sync 3.7.1 or later to complete these steps.

  1. To dock for the first time, you must first set up a new partnership.



    Select to set up a new Standard Partnership and click next

     
  2. Select to synchronize with a "Microsoft Exchange Server and/or this desktop computer"




     
  3. Enter the information required.  The server name exchange.asu.edu (note that this server DOES require an SSL connection); your ASURite User ID for the User Name; your ASURite Password; and the domain of ASURITE.  Check the box that says "Save Password" and then click Next.




     
  4. The next window will ask you which folders you would like to synchronize with.  Place a checkbox in any folders that you would like to sync with.  Additional "inbox" folders and other subfolders will only be available after your first sync.




     
  5. In the next window, select the radio button that says "Delete any items on my device and replace them with the items from the server" and click next.



     
  6. Finally you will need to name your device.  This name can be any name that you will recognize. 




     
  7. The next window "Synchronization Settings" will allow you to specify which programs you will want to sync with.  The defaults on this window should be accurate.




     
  8. Then your setup is complete, and you  may press finish and sync for the first time.



     



    9.  

  9. Set up your SMS notifications using these instructions.

You device can also be set up without the use of the Active Sync software.  Follow these instructions for this type of installation.

  1. Turn on the Windows Mobile Device.
  2. Click on Start in the upper-left corner of the screen.
  3. Then choose Programs from the menu.
  4. Click on Active Sync & in the Active Sync window choose to set up a new server connection.
  5. Enter the server address exchange.asu.edu.
  6. Be sure to check the box titled This server requires an encrypted (SSL) connection.
  7. Enter your ASURite ID, password, and the domain ASURITE.
  8. Put a checkmark in the box titled Save password.
  9. Then set up your device to receive SMS notifications that will keep your device always up to date with your outlook.
