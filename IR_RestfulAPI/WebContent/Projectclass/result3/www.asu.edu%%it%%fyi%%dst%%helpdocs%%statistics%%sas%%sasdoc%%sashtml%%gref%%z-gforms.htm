GFORMS

GFORMS


Specifies the JES form name for IBM printers.

Used in: GOPTIONS statement
Default: STD
Restriction: used only with IBM3287 and IBM3268 device drivers on OS/390 systems only

Syntax

Syntax

GFORMS='forms-code'
