TAN

TAN


Returns the tangent

Category: Trigonometric

Syntax
Arguments
Examples

Syntax

TAN(argument)

Arguments

argument
is numeric and is specified in radians.
Restriction: cannot be an odd multiple of /2

Examples

SAS Statements Results 
x=tan(0.5);
 
0.5463024898
 
x=tan(0);
 
0
 
x=tan(3.14159/3);
 
1.7320472695
