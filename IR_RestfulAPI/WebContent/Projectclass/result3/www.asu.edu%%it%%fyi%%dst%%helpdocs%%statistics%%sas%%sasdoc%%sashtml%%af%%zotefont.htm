Print Manager class: _getFootnoteFont


Returns the footnote font used in the document

Syntax
Details

Syntax

return = objectName._getFootnoteFont( );

Details

Font is validated as much as possible during the method invocation. However, the font is not bound to the printer until the _render method is called. If a problem is encountered at that point, then the font is set to the printer's default font.
