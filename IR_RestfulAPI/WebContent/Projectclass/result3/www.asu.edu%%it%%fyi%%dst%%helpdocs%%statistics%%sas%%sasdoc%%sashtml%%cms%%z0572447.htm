MSG=


Specifies the path of the SAS message directory

Default: none
Valid in: configuration file, SAS invocation
CMS specifics: all

Syntax

Syntax

MSG='directory-specification' | ('dir-spec | '...' dir-spec n')

directory-specification
identifies one or more directories in one of the following forms. Refer to Identifying an External File for additional details.

'filemode ' | 'SFS-directory'
specifies the filemode or the SFS directory.
