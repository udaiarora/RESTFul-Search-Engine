REP_MGRLOC=


Specifies the location of the repository manager for common metadata

Valid in: configuration file, SAS invocation
Category: Environment control: Files
PROC OPTIONS GROUP= ENVFILES

Syntax
Syntax Description
Details

Syntax

REP_MGRLOC="SCL-libname" | 'SCL-libname' |
(SCL-libname) |libref

Syntax Description

SCL-libname
any valid SCL (SAS Component Language) libname, with the following syntax:(libref <,SAS-data-library<,engine<,options>>>). The SCL libname can be enclosed in single or double quotation marks, or in parentheses. When you specify two or more locations, you must enclose the specifications in parentheses. The maximum length for the SCL libname is 200 characters.

libref
a previously defined libref.

Details

REP_MGRLOC= specifies the location of the repository manager for common metadata. Normally, this option is a site-wide option that is set by the SAS Installation Representative.
