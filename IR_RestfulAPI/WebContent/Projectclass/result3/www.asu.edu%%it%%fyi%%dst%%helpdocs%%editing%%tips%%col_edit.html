Column Editing with the Cut Command

There is probably a way to do column deletions in the many editors available on UNIX (vi, pico, xedit, emacs, etc.). But this document shows a way to delete columns throughout an entire file using the "cut" command. For example, if you had a file like this:

98 stats > cat test.data
1234567890
1234567890
1234567890
1234567890
1234567890

you could delete (cut) columns 4 and 8-9 like this:

99 stats > cut -c1-3,5-7,10 test.data
1235670
1235670
1235670
1235670
1235670

(or try "cut -c1-3,6- test.data" for a different example)

You can redirect the output of these commands to a file instead of standard output too. For example to direct output to a file called "new.data", you can use command similar to the following:

cut -c1-3,6- test.data > new.data

For more information on the cut command, enter "man cut" at the system prompt.
