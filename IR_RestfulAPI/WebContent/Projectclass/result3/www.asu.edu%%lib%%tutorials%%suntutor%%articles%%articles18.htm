Truncation

Truncation is another technique used to broaden your search, and increase the number of articles retrieved.
Truncation allows you to search for words that begin with the same letters, but which have different endings, by using a punctuation mark called a truncation symbol. For example, violen*
media and violen* and children and behavior
The above search uses the asterisk (*) as the truncation symbol to search for the words ?violence,? ?violent,? ?violently.? Truncation symbols can vary in each index. Some common truncation symbols are the asterisk(*), the question mark(?), and the exclamation point(!)
