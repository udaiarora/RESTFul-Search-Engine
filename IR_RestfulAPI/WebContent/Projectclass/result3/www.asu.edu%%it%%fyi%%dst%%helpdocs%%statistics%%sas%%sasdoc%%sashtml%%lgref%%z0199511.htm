DFLANG=

DFLANG=


Specifies language for international date informats and formats

Valid in: configuration file, SAS invocation, OPTIONS statement, Systems Options window
Category: Environment control: Language control
PROC OPTIONS GROUP= LANGUAGECONTROL

Syntax
Syntax Description
Details
See Also

Syntax

DFLANG=' language'

Syntax Description

'language'
specifies the language for international date informats and formats. The default value is English.

These are valid values for language:

  • Afrikaans

  • Catalan

  • Croatian

  • Czech

  • Danish

  • Dutch

  • English

  • Finnish

  • French

  • German

  • Hungarian

  • Italian

  • Macedonian

  • Norwegian

  • Polish

  • Portuguese

  • Russian (Host-specific)

  • Slovenian

  • Spanish

  • Swedish

  • Swiss-French

  • Swiss-German.

Details

You can change the value during a SAS session, but you can use only one language at a time. The values for language are not case-sensitive.
