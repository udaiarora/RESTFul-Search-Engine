Sort/Merge Operations

Sort work data sets usually need to be allocated when a program using the SORT or MERGE verbs is executed. The following form of a jobstep is used:

     //   EXEC COB2CG
     //COB2.SYSIN DD *
          .
          .
          .
          SELECT SORT-IN  ASSIGN TO S-INFILE
          SELECT SORT-OUT ASSIGN TO S-OUTFILE
          SELECT SORT-WK  ASSIGN TO S-SORTWK
          .
          .
          .
          SORT SORT-WK USING SORT-IN GIVING SORT-OUT.
          .
          .
          .
     //GO.INFILE  DD  DSN=WYL.xx.yyy.INDATA,DISP=OLD
     //GO.OUTFILE DD  DSN=WYL.xx.yyy.OUT,DISP=(NEW,CATLG,DELETE),
     //    UNIT=unit,SPACE=tracks,DCB=format,etc.
     //GO.SORTWK01 DD  UNIT=SORTWK,SPACE=(CYL,(1,1))
     //GO.SORTWK02 DD  UNIT=SORTWK,SPACE=(CYL,(1,1))
     //GO.SORTWK03 DD  UNIT=SORTWK,SPACE=(CYL,(1,1))
     //GO.SORTPRNT DD  SYSOUT=A
     //GO.SYSOUT   DD  SYSOUT=A

The last 5 DD statements should be used exactly as they are shown.
