Coeus
Technical Information

Coeus was developed at MIT and is used at many universities throughout the country. "Coeus" is not an acronym. It is the name of the ancient Greek titan of knowledge.

Client/Server

Coeus is a three-tier client/server application, meaning it runs as a program on a PC that connects to Java Web Application server, that connects to a database; all through the network.

Client

The client is built in Java and is a web application, but it does not run in your browser. It is installed on your computer and uses Java WebStart to run and connect to the Java Application Server.

Application Server

The Coeus server handles all the communication between the Coeus client and the Database Server. This includes negoatiting firewall and database user privileges. The client never talks directly to the Database Server.

Server

The data is kept on an Oracle database server. To use Coeus, you don't need to have Oracle Net8 software, because the Coeus Application Server handles all this communication.

Firewall

ASU has firewalls to protect the secure systems of the University. The Coeus databases are secured behind firewalls.

VPN software

The PC must have VPN SecuRemote installed to negotiate the firewall connection from off campus. Your Coeus client will negotiate the firewalls within ASU, you just need to get into the ASU Border Firewall from off-campus. VPN must be configured with the ASUFW site.

Firewall Permission

You request firewall permission from the Computer Accounts Office. The Production Coeus database and the Training Coeus database are in the production firewall zone known as "Admin Computing Accesses". The QA database is in the develop firewall zone known as "VPN access to dev ports in DMZ". See the trouble shooting guide for more details on firewall access.

Coeus Client Installation

Installation

  • installs and configures the Java Coeus client
  • places a Coeus icon on your desktop

You can run the installer by visiting the Coeus 4.0 Installer.

The installer will place the Coeus client on your local hard disk in your user space. You should not need to have local administrator privileges on your PC to install Coeus using Java WebStart. You will have the option to place a Coeus icon on the desktop. The Coeus 4.0 Installer will not create a Coeus program group on your Start Menu.

Uninstallation

Coeus can be uninstalled by reviewing your Coeus WebStart configuration.

Upgrades

Because Coeus runs as a web application, any updates to the application will be applied as you load the application. Sometimes the Coeus application will load slower from your desktop; this probably means it is retrieving updates to the application.
