How to Find Syntheses in Chemical Abstracts/SciFinder Scholar

 

 




 

 

 Search Strategy:
  1. Click on the link above to access Chemical Abstracts/SciFinder Scholar.

    Access to Chemical Abstracts/SciFinder Scholar is restricted to ASU faculty, staff and students only and requires special software. All Tempe campus library workstations have this software available.

  2. Select LOCATE (Find a specific reference or substance) from the New Task screen.

  3. Select LOCATE SUBSTANCES

  4. Type in the name of the compound. To search two or more compounds, put each name on a separate line.

  5. Click on the "pages" icon to view references.

  6. Click in the radial button for "References associated with" and then select the "Preparation" category.

    The above search strategy will limit results to 1967 to the present. To find synthesis articles prior to 1967, do not limit to the Preparation category in step 5 - instead take the "All References" option. Refine the results to Publication Years "1900-1967" and Research Topic "preparation"; there could be many "false hits" in the results.

  7. Use the page icon to find the actual article or see How To Find Your References in the ASU Libraries for other options.

    Access to Chemical Abstracts/SciFinder Scholar is restricted to ASU faculty, staff and students only and requires special software. All Tempe campus library workstations have this software available.

  6. Click on the "Analyze/Refine" button at the bottom of the screen.

  7. Click on the "Refine" button.

How to Determine the Commercial Availability of a Substance via Chemical Abstracts/SciFinder Scholar:

 

 

 

 

 Search Strategy:
  1. Click on the link above to access Chemical Abstracts/SciFinder Scholar.

    Access to Chemical Abstracts/SciFinder Scholar is restricted to ASU faculty, staff and students only and requires special software. All Tempe campus library workstations have this software available.

  2. Select LOCATE (Find a specific reference or substance) from the New Task screen.

  3. Select LOCATE SUBSTANCES

  4. Type in the name of the compound. To search two or more compounds, put each name on a separate line.

  5. Click on the flask icon to view commercial sources.

     If a flask icon is not displayed, this indicates that Chemical Abstracts has not found this substance to be commericially available.

  6. A list of suppliers will be displayed; to view detailed information from a specific supplier, click on the microscope icon.

How to Find Reaction Information in Chemical Abstracts/SciFinder Scholar:

 

 

 

 

 

 Search Strategy:
  1. Click on the link above to access Chemical Abstracts/SciFinder Scholar.

    Access to Chemical Abstracts/SciFinder Scholar is restricted to ASU faculty, staff and students only and requires special software. All Tempe campus library workstations have this software available.

  2. Select LOCATE (Find a specific reference or substance) from the New Task screen.

  3. Select LOCATE SUBSTANCES

  4. Type in the name of the compound. To search two or more compounds, put each name on a separate line.

  5. Click on the "A>B" icon to view commercial sources.

     If an "A>B" icon is not displayed, this indicates that this compound is not found in the CASReact database -- Use the "Get References" icon/button to display a list of references.

  6. Select the reaction role for the substance.
