NOTES

NOTES


Writes notes to the SAS log

Valid in: configuration file, SAS invocation, OPTIONS statement, Systems Options window
Category: Log and procedure output control: SAS log
PROC OPTIONS GROUP= LOGCONTROL

Syntax
Syntax Description
Details

Syntax

NOTES | NONOTES

Syntax Description

NOTES
specifies that SAS write notes to the SAS log.

NONOTES
specifies that SAS does not write notes to the SAS log. NONOTES does not suppress error and warning messages.

Details

You must specify NOTES for SAS programs that you send to SAS Institute for problem determination and resolution.
