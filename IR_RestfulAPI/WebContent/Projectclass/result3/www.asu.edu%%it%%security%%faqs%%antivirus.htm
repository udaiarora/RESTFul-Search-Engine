How do I check the current version of my VirusScan, VirusScan engine and VirusScan DAT/SDAT?

  1. Right click on the blue and red shield (VShield) in your task bar.
  2. Left click on About.
  3. A new window will appear with the words McAfee VirusScan. At the end of that line will be a v followed by some numbers. Those numbers represent the VirusScan version.
  4. In the center of this window will be a line that says Virus definitions: followed by some numbers. Those numbers represent the DAT/SDAT version.
  5. Another line will read Scan Engine: followed by some numbers. Those numbers represent the VirusScan engine version.

How do I scan my computer for viruses?

  1. Open up VirusScan. It is located under Start->Programs->Network Associates-> VirusScan
  2. In the VirusScan window there will be a text boxed labeled Scan in. This box is the location that you will manually be scanning. Enter C:\ to scan your hard drive.
  3. Make sure the box labeled Include sub folders is checked.
  4. There are also three circular buttons. One labeled Default files, one labeled All files and one labeled User specified files (you can ignore this one). Selecting All files will configure the scanner to scan every file on your computer. Selecting Default files will configure the scanner to look for viruses only in files that have file name extensions that are included with the virus definition (.DAT) files that are currently in use.
  5. Make sure that there is a check mark for Compressed files.
  6. Click on the Advanced.. buttons. Select Enable heuristics scanning (there should be a check mark to the left of this after you select it). Then click on Enable macro and program file heuristics scanning. Finally click on the OK button.
  7. Click on the button labeled Scan Now to begin the scan. Note: if you selected the All files option then the scan will probably take over an hour, depending on your system.

Why won't VirusScan scan my computer as automatically scheduled?

Your task is not running because the computer is in sleep mode. Turn the sleep mode off, leave your computer up and running, so scheduled tasks can be done. Your computer can also be locked and the scheduled tasks will be completed.

How can I tell if a virus warning is a hoax?

Often times, Virus warnings through E-Mails can be hoaxes. The best bet is to ignore all Virus warnings that do not come from an authorized source.

The Anti-Virus announcement list provides authorized information on new releases and updates to the ASU-supported anti-virus software. All subscribers will be notified of new software releases and updates to virus definitions and scan engine changes. This announcement service will also send notification about virus warnings and emergency patches.

For more information on Virus hoaxes or to check on a possible hoax, visit NAI's library of virus hoaxes: http://vil.nai.com/VIL/hoaxes.asp.

What types of viruses are there?

Essentially, three types of viruses exist: Boot Sector Viruses, File Infectors, and Macro Viruses.

Boot sector viruses:

This type of Virus infects the boot sector of hard disks and floppy disk. They are spread two ways: booting your system from an infected floppy disk will likely infect the hard drive, and using a floppy disk in an infected system will infect the floppy disk. The one way to avoid infection is to write-protect your floppy disk, when possible, and never leave a floppy disk in the A: drive of your PC when you shut it off.

File infectors:

This type infects executable program files, or files with a EXE or COM extension. They spread when an infected file is run, which loads the virus into memory. Any subsequent programs that are run risk being infected. This kind of viruses can be down loaded from the Internet or received in an E-Mail attachment. However, the file must be executed first to activate the virus.

Macro viruses:

These are fairly new Viruses that only began widespread distribution a few years ago. They are written using the macro language in certain productivity applications, primarily Microsoft Word and Microsoft Excel. They are spread by opening infected documents or spread sheets, which in turn infects the system and any subsequent files opened. Most do not do any damage and are merely an annoyance. However, the latest macro virus, Melissa, has been recently in the news for overloading mail servers with its macro capabilities.

The best way to avoid being infected by a Virus is to have up to date and properly install Anti-Virus software on your computer. ASU provides Anti-Virus software for all ASU students, faculty and staff. You can find out more about ASU's Anti-Virus software on the Anti-Virus WEB page.

What are the Quarantine and Infected folders? What should I do with the files found in them?

The Quarantine and Infected folders are made by VirusScan. Infected comes from version 4.x and Quarantine from version 7. These folders are where VirusScan moves emails that it has detected a virus in but was not able to clean.

If you are certain the emails in these folders are not infected then open them. If you are not certain then delete them. Additionally, VirusScan 7 will automatically Quarantine emails with attachments that have more than one period in them.
