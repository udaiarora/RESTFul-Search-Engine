Chart Control: colorSource


the color source to use when selecting colors (List or Range Object)

Type: Character
Valid Values: ColorList,ColorRangeObject

Details

Details

The colorSource setting is ignored unless colorVariable is specified. If the specified source is invalid, then the default colors are used.
