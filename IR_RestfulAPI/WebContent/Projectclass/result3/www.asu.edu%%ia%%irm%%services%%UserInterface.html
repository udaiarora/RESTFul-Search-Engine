User Interface 1.0

User Interface 1.0 is the required client software for accessing ASU's Benefactor and Colleague systems. This distribution will install a traditional character-based interface (Datatel - Terminal) and a dramatically improved graphical interface (Datatel). For detailed information on using either interface, reference Datatel's Guide to User Interfaces.

 

Installation Procedure:

You should remove any previously installed wIntegrate or FrontView client software before proceeding with this installation. These older clients ran independently of the Windows registry, and can be removed by deleting the program files and removing any shortcuts. FrontView may also be removed through the Windows Installer.

After retrieving the distribution file execute UI10. The installation file set will self-extract and begin the setup process.

Accept the default of a Full Local Installation.

Enter the User ID "guest" and select the account "Benlive".

The default installation path should be correct unless Windows is installed on a drive other than "C". If necessary correct the path to read "C:\DATATEL\UI".

The software installation is complete. The unzip dialog box may remain on the screen for a few moments while the temporary installation files are removed.

Re-execute UI10 to configure the user accounts.

Choose the Advanced Installation.

Add accounts as necessary. All users should have access to the three Benefactor accounts. Only ASU Foundation Finance employees will require the three Colleague accounts.

The configuration is complete. Again, the unzip dialog box may remain on the screen for a few moments while the temporary installation files are removed.

The UI10 instllation program is no longer necessary and may be manually removed from the system.

E-mail any installation questions to IRM-Q or call the IRM helpdesk at 5-7445.
