GET_DATA Data Vector: _setReverse

GET_DATA Data Vector: _setReverse


Sets the reverse status

Syntax
Details

Syntax

CALL SEND (data-vector-id, '_setReverse', reverse);

Argument Type Description
reverse
 C specifies the reverse status:


'Y' swap foreground and background colors


'N' do not swap foreground and background colors

Details

The _setReverse method is valid only if the viewer is a table editor.
