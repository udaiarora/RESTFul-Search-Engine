As a service to the ASU community and in an effort to reduce the number of phone books needed on campus, ASU Telephone Services and Qwest Dex are pleased to offer the Qwest Dex Electronic White and Yellow Pages on line for use by the ASU community.

This product is supplied to ASU by Qwest Dex. There is no charge for using this product.

Qwest Dex states that the currently supported platforms are Windows 95/98/2000/NT and Macintosh (OS 7.5.3+). Telephone Services has tested this with the following operating systems and browsers.

Operating Systems: Windows NT 4.0 Workstation with SP 6a
  Windows 2000 Professional with SP 2, SP 3
Browsers: Internet Explorer 5.5, 6.0
  Netscape 4.7+, 7.0, 7.1

This product will install software on your computer and place an icon on your desktop. Depending on how your PC is set up you may need to have administrative permissions in order to install it. If you're using Internet Explorer as your browser it will load the ActiveX component on your browser and the phone book will appear in an Internet Explorer window. If you're using Netscape it will install a stand-alone reader on your PC that will connect with the phone book on the web server. In either case, after installation clicking on the desktop icon will connect you directly to the Qwest Dex White and Yellow Pages on their web site, there is no need to access the Telephone Services main web page or to bookmark its location. Qwest Dex will update the phone books on the web site as new versions are issued.
