NEdit Command

The current version of NEdit available on the General and Research/Stats servers is 5.3. The command to execute NEdit is:

     nedit &

To print from inside the editor. From the main menu, select:

     File   Print 

A window titled Printer Options appears and shows a command:

     lp -t...

change the command to:

     lpr -t...

and with the mouse click on the [Print] button. The contents of the windows will be sent to your default printer.
