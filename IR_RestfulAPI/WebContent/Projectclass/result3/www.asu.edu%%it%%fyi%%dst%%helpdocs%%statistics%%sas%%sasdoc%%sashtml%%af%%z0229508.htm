Image Color Map: _putCmap

Image Color Map: _putCmap


Writes the contents of the color map to the Message window

Syntax
Details

Syntax

CALL SEND (object-id, '_putCmap');

Details

This method is useful for debugging problems with color maps and is similar to calling _getColor on each color map cell and writing the values to the log.
