Print Manager class: _setTitleSpace


Sets the title space of the document

Syntax
Details

Syntax

return = objectName._setTitleSpace( );

Details

The default additional space for a title is 0. Any additional space is added after the title is printed. This method will program halt if numberofunits is < 0 or units is not one of the valid units.

For information on valid units of measure, see Using the Print Manager Class.
