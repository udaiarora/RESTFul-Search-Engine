    Use of Keywords

    The Help Facility allows for keywords that may be used to point directly to topics of interest. The command to skip directly to a help topic without navigating through the main menu, is:

    help <topicname>

    replacing <topicname> with the topic that interests you. For example, to skip directly to the section on General UNIX Commands, simply enter: help commands. For the section on C language, simply enter: help c, for SAS enter: help sas, and so on. The current list of available keywords includes:

    Use of Keystrokes

    Enclosed is a list of keys that can be used within the Help Facility and their corresponding responses. 

       Down arrow - Highlight next topic 	
     Up arrow - Highlight previous topic
Return, Enter - Jump to highlighted topic
  Right arrow - Jump to highlighted topic
   Left arrow - Return to previous topic
 
	 a - Add current link to bookmark file (lynx_bookmarks.html)
	 m - Return to main menu
	 p - Print to a file or screen, or mail
	 q - Quit 
	 Q - Quit without prompting
	 / - Search for a string within the current document
	 v - View your bookmark file
         z - Cancel transfer in progress
    CTRL-R - Reload current file and refresh the screen
    CTRL-W - Refresh screen
    CTRL-U - Erase input line
    CTRL-G - Cancel input or transfer

    How to Print Text-version of Help Material

    While viewing a document, first press the "p" key (lowercase) to save the document to your AFS space and then from outside the help facility use the "lpr" command to send the file to one of the ASU printers. Under Printing, the help facility has more instructions on how to print files. You may also "help printing" for more information.

    Must-read Section

  1. The "help" command uses a UNIX version of a web browser called "Lynx" which like any other interactive UNIX application behaves differently when used with a different combination of telnet clients and UNIX operating systems. For example, with a Windows telnet client called "EWAN", when Lynx is executed on "general.asu.edu", there appears to be some shifting of highlighted lines. However with EWAN, and with Lynx on "stats.asu.edu" or on "research.asu.edu", there is no obvious "shifting". Whereas, with a Kermit or a Hummingbird telnet client, Lynx appears to work well on all the above mentioned servers.

  2. In the "help" menu, there are some selections which point to pages on the World Wide Web. The selections include Electronic Mail and Electronic Forum. It is possible that by going through several levels of selections from either of these pages, one may easily feel lost in the World Wide Web system. In this case, it is better to press "m" to go back to the main menu. It is also possible that one of the selections may experience problems connecting to a server and in this case, it is better to press "z" to cancel the request.

    Use of the UNIX man pages

    The help facility contains a brief summary on using man pages. To see this information, enter "help unix" and then from the options displayed, select "General Unix Commands" and then "man".

    For more information on man pages, enter "man man". To search for keywords in the man pages, use the "man -k" command which is described in the output from "man man". These commands must be entered at the system prompt.
