Introduction

About This Book

This book is a user's guide to SAS/ETS software. Since SAS/ETS software is a part of the SAS System, this book assumes that you are familiar with base SAS software and have the books SAS Language: Reference and the SAS Procedures Guide available for reference. It also assumes that you are familiar with SAS data sets, the SAS DATA step, and with basic SAS procedures such as PROC PRINT and PROC SORT. Chapter 2, "Working with Time Series Data," in this book summarizes the aspects of base SAS software most relevant to the use of SAS/ETS software.
