Tree View Control Events

Events specified for the Tree View Control class are described here.

autoExpand changed
Occurs when the autoExpand attribute is changed

beginEdit
Occurs at the start of a node being edited

click
Occurs on a click

collapse
Occurs on a collapse

deleteNode
Occurs when a node is deleted

doubleClick
Occurs on a double-click

endEdit
Occurs at the end of a node being edited

expand
Occurs on an expand

font changed
Occurs when the font attribute is changed

labelColor changed
Occurs when the labelColor attribute is changed

popup
Occurs on a popup request

resizable changed
Occurs when the resizable attribute is changed

resize
Occurs on a resize

selectedNode changed
Occurs when the selectedNode attribute is changed

selectionMode changed
Occurs when the selectionMode attribute is changed

selections changed
Occurs when the selections attribute is changed

title changed
Occurs when the title attribute is changed

titleHeight changed
Occurs when the titleHeight attribute is changed
