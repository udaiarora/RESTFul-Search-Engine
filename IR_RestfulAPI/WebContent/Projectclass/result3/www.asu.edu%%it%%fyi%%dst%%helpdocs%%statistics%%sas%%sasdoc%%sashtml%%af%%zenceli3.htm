Scatter Control: _setReferenceLineLocation


Set the location for the specified reference line

Syntax

Syntax

objectName._setReferenceLineLocation( refId, value );

Argument Type Use Description
refId Numeric Input refId of the reference line whose location will be set
value Numeric Input the location for the specified reference line
