This work is a portion of Gesner's complete Historiae Plantarum, which covers descriptions of fifty plants.  Each description includes references to other sources.  Following the individual descriptions there is a series of 14 colored plates which contain illustrations of each of the fifty plants.  There is also a table of the plates, listing by plate number the Latin names of the plants depicted.

Subjects: Botany--Pre-Linnean works.
