REXXMAC


Enables or disables the REXX interface

Default: NOREXXMAC
Valid in: configuration file, SAS invocation, OPTIONS statement, OPTIONS window
Category: Host Interfaces: REXX
OS/390 specifics: all

Syntax
See Also

Syntax

REXXMAC | NOREXXMAC

REXXMAC
enables the REXX interface. This means that when SAS encounters an unrecognized statement, it searches for a REXX EXEC file whose name matches the first word of the unrecognized statement. The REXXLOC= system option specifies the DDname of the REXX library to be searched.

NOREXXMAC
disables the REXX interface. This means that when SAS encounters an unrecognized statement, a "statement is not valid" error occurs.
