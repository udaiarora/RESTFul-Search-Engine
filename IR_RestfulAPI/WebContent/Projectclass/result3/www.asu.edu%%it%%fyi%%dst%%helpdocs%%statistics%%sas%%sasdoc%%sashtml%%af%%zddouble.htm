SCL List Class: _addDouble

SCL List Class: _addDouble


Adds an SCL numeric value to the list at the position specified by index. If index is negative, the position to add the item is determined by counting down from the end of the list. The name of the item is set to name

Syntax

Syntax

objectName._addDouble( val, index, name );

Argument Type Use Description
val Numeric Input Numeric value to be added to the list.
index Numeric Input Position to add the item.
name Character Input Name of the item (may be blank).
