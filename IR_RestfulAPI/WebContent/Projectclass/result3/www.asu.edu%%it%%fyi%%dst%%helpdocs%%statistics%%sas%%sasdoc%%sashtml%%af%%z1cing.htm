Histogram Control: YAxisValueSpacing

Histogram Control: YAxisValueSpacing


the spacing between the labels in multiples of the font height

Type: Numeric
Valid Values:

Details

Details

If the spacing is set to 1, the tick marks are spaced using one unit of font height. Other values space the tick marks further apart. Changing the spacing causes the axis to be redrawn with differing numbers of tick marks and labels.
