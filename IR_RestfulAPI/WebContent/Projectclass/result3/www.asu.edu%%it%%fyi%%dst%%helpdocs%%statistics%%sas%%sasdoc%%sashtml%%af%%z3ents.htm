Check Box Control Events

Events specified for the Check Box Control class are described here.

click
Occurs on a click

conformSize
Occurs when the control is resized due to conformSize

conformSize changed
Occurs when the conformSize attribute is changed

font changed
Occurs when the font attribute is changed

label changed
Occurs when the label attribute is changed

labelColor changed
Occurs when the labelColor attribute is changed

popup
Occurs on a popup request

selected changed
Occurs when the selected attribute is changed

textOrientation changed
Occurs when the textOrientation attribute is changed
