SCL List Class: _save


Saves list to a catalog entry or file. See SCL reference for SAVELIST function for description of arguments.

Syntax

Syntax

return = objectName._save( type, target,attrlist,desc );

Argument Type Use Description
return Numeric Return
type Character Input
target Character Input
attrlist Numeric Input
desc Character Input
