SORT31PL

SORT31PL


Controls whether SAS software calls the host sort program by using the extended (31-bit) plist or standard (24-bit) plist

Default: NOSORT31PL
Valid in: configuration file, SAS invocation, OPTIONS statement, OPTIONS window
CMS specifics: all

Syntax
Details

Syntax

SORT31PL | NOSORT31PL

SORT31PL
specifies that SAS software will call the host sort program using the extended (31-bit) plist.

NOSORT31PL
specifies that SAS software will call the host sort program using the standard (24-bit) plist.

Details

If you are using SORT31PL, you must use NOSORTMSG if your host sort program does not support the PRINT=ALL option.

If you are using SORT31PL and your host sort is configured to display all messages by default, you might need to specify SORTPARM='PRINT=CRITICAL' to suppress messages.
