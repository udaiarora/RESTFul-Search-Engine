Benefactor Class Descriptions

Back to IRM

If the class you wish to register for is not currently scheduled, please contact the training coordinator to schedule a session.
Course Title
Course Content
Course Length
DGI - Demographic/Giving Inquiry This course provides a basic introduction to Benefactor. Students will learn where and how to find information on both people and organizations in the Benefactor database
Two hours
CORE - Data Entry This course will provide indepth instruction on how to add people and companies to Benefactor, as well as how to make changes to existing data. Class capacity is 10. Participants MUST be able to attend all four sessions.
Four hours per day for four consecutive days
Data Retrieval This class will teach students how to extract information from Benefactor using simple query language. Class capacity is 10. Participants MUST be able to attend all four sessions. This class is offered on an as-needed basis.
Four hours per day on four dates as arranged by the instructor
MP - Major Prospects This class provides instruction on how to enter Contact Reports into Benefactor. It will also cover other screens related to prospect management.
Two hours
A & E Overview This course will acquaint the user with the Activities & Events module. Students will learn how to set up an event, manage invitations and reservations, cancel... Class capacity is limited to 10.
Two hours
CC - Correspondence Control This class will cover topics related to Benefactor Correspondence including how to to batch and process correspondence, identifying correspondence pieces to run, and generating envelopes. Class capacity is 10.
Two hours
