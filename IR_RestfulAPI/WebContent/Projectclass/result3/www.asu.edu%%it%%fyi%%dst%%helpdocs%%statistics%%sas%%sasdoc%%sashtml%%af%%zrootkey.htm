Registry Component: _getRootKey

Registry Component: _getRootKey


Returns the handle of the USER ROOT or the SYSTEM ROOT key

Syntax

Syntax

return = objectName._getRootKey( which );

Argument Type Use Description
return Numeric Return returns either the handle of the root key or 0 for failure
which Character Input specifies which root key to return: SYSTEM or USER
