SAS Component Language: Reference, Version 8

Look no further for information about developing applications with SAS Component Language (SCL). For the novice applications developer, this book describes the structure of SCL programs, describes the basic elements of the SCL language, and provides guide lines for compiling and testing applications. The experienced applications developer will find detailed reference information, including examples, for all of the SCL functions, statements, and CALL routines. This book is designed for SAS/AF applications programmers who have some experience with the SAS System.
