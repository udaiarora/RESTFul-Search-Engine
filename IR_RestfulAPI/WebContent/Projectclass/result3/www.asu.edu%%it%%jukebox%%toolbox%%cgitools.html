CGI Tools

CGI stands for Common Gateway Interface.  CGI is a standard for interfacing external applications with information servers, such as web servers.  Java Servlets provide similar functionality as CGI but written in the Java language.  A variety of CGI's and Java Servlets are available for ASU Web space users to add dynamic or interactive features to their pages. A description of each is given here, with a link to full usage instructions and troubleshooting information.  If you want to learn specific details about CGI itself, please see the NCSA CGI Overview.

  • Counter (Count.cgi): Tracks and displays the number of visitors to a page. Can also be used to generate a "hidden" count that does not display.
  • E-mail form (Mail Servlet): Processes form entries and emails the results to any valid email address.
  • E-mail attachment form (MailAttach Servlet): Processes form entries and emails the results to any valid email address as a separate attached file.
  • Location push (loc.cgi): "Pushes" the browser to a Web address. Useful for pull-down location menus and "buttonized" links.
