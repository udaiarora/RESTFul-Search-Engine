How to find articles

Conclusion

This lesson has given you the basics about finding articles. While each of our journal indexes look different, these skills should work in all of them. The best way to master an index is to practice with it.

If you would like to use the indexes from home or from another location off campus, you will need to login using your ASURITE ID and password. For more information, see Accessing ASU Library indexes and electronic journals from off-campus.

Don't be afraid to ask the reference librarians for help when you need it. To find contact information for the librarians who can help you find articles, see Who can help you?
