TRANTAB=


Specifies a translation table for character conversions

Valid in: DATA step and PROC steps
Category: Data Set Control

Syntax
Syntax Description
Details

Syntax

TRANTAB=table-name

Syntax Description

table-name
specifies the name of a translation table that is used to translate the characters in a data set from foreign encoding to local encoding. SAS searches for translation tables in SASUSER.PROFILE first and then SASUSER.HOST.

Note:   The TRANTAB= data set option overrides the libname option for a specific file.  

Details

Translation tables are used internally by the SAS supervisor to implement National Language Support (NLS). If you are unfamiliar with the purpose of translation tables, do not change the specifications without proper technical advice. For details, see SAS/CONNECT Software: Usage and Reference or SAS/SHARE Software: Usage and Reference.
