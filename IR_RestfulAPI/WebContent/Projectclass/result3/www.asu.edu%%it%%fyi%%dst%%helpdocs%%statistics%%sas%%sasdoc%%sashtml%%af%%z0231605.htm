Extended Text Entry: _setMode


Sets the Keystroke Feedback mode

Syntax

Syntax

CALL NOTIFY (ext_text_name, '_setMode', string);

Argument Type Description
string C specifies the Keystroke Feedback mode:


'ALWAYS' runs the _feedback method for every modification made to the object


'NONE' causes _feedback not to run at all
