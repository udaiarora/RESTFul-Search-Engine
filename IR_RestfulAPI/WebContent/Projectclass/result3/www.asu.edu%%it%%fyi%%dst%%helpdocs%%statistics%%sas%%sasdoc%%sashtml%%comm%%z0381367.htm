Communications Access Methods for SAS/CONNECT and SAS/SHARE Software

What Is a Communications Access Method?

A communications access method is the interface between SAS software and the network protocol that you use to connect two platforms. You must specify a communications access method for both SAS/CONNECT software and SAS/SHARE software.

The access method that you choose is determined by the network protocols that you have available at your site and the host platforms that you are connecting.

Note:   Do not confuse Communications Access Method with FTP Access Method. The FTP Access Method is a facility that is used with the FILENAME statement in SAS software that allows you to access remote files by using the FTP protocol. For more information, see SAS Language Reference: Dictionary.
