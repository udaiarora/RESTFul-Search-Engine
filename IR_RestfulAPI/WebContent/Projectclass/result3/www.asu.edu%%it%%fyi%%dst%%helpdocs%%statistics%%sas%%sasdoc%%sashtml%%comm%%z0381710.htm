Communications Access Methods for SAS/CONNECT and SAS/SHARE Software

Sample Scripts

The content and explanation follow for typical
SAS/CONNECT scripts that sign on and sign off.

TSO.SCR
connects to an OS/390 remote host with the EHLLAPI 3270 access method.

TCPUNIX.SCR
connects to a UNIX remote host with the TCP/IP access method.

LOGCMS.SCR
connects to a CMS remote host with the EHLLAPI access method.

TCPWIN.SCR
connects to a Windows NT or a Windows 95 remote host with the TCP/IP access method.

TCPMVS.SCR
connects to an OS/390 remote host without TSO with the TCP/IP access method.
