Catalog Entry List Model Events

Events specified for the Catalog Entry List Model class are described here. See Selection Model Events for inherited events.

catalog changed
Occurs when the catalog attribute is changed

catalogEntryDisplayed changed
Occurs when the catalogEntryDisplayed attribute is changed

descriptionDisplayed changed
Occurs when the descriptionDisplayed attribute is changed

levelCount changed
Occurs when the levelCount attribute is changed

modifiedDateDisplayed changed
Occurs when the modifiedDateDisplayed attribute is changed

typeFilter changed
Occurs when the typeFilter attribute is changed
