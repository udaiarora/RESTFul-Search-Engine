SCL List Class: _getNamedDoubleAt

SCL List Class: _getNamedDoubleAt


Finds the named numeric item by searching for the nth occurrence of name, and starting the search at start.

Syntax

Syntax

return = objectName._getNamedDoubleAt( name,nth,start, default );

Argument Type Use Description
return Numeric Return The numeric value.
name Character Input Name of item to search for.
nth Numeric Input Occurence of name to search for.
start Numeric Input Index to start searching from.
default Numeric Input Default value to return if named item is not found.
