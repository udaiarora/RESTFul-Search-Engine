Image: _readClipboard


Reads an image from the host clipboard and displays it in an image object

Syntax
Details

Syntax

CALL NOTIFY (image-name, '_readClipboard');

Details

On some hosts, you can read the clipboard only after you use the WRITE_CLIPBOARD command.
