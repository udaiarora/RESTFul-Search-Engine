Computing Sites Software Request Form

This “Computing Sites Software Request Form” is for use by faculty of Arizona State University (ASU). By submitting this form, you are requesting the addition of a software package to every machine in the Information Technology (IT) Computing Sites and Classrooms on the main campus of ASU.

ASU’s IT sites and classrooms are for general computer use. Requests for installation are intended to make available general-purpose software to ASU students. Although requests for specialized software intended for a select group of students will be considered, it is general policy to decline such installation requests. In these cases, specialized software purchased by departments for student use may be more suitable for use from an IT Instructor Volume.

When a general software package is approved for inclusion in the IT sites, it may be possible to perform the installation during the next available break change if funding is available. On the whole, qualifying software purchased through IT funds requires a lead-time of at least six months prior to being made available. During certain fiscal years, IT funding may not be available for the purchase of new software. Software installation requests are then prioritized and added on the next fiscal year budget cycle. Funding for software packages is occasionally provided through other university or departmental budgets. These requests are considered on a case-by-case basis.

Once approved and funding is made available, IT’s installation cycle to add new software is during break changes during either May, August, or December-January.
