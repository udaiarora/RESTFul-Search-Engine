Available C/C++ Compilers

C and C++ are part of the base set of software that Information Technology makes available on its UNIX servers. As such, C and C++ are available on the following servers: general.asu.edu (Sun), stats.asu.edu (HP), and research.asu.edu (IBM RS/6000). In addition, each server offers multiple "flavors" of C/C++, for example: 

PLATFORM           COMPILER   DISTINGUISHING FEATURE
--------           --------   ----------------------
research.asu.edu      cc      extended language level
.                     xlc     ansi language level
.                     c89     ansi language level
.                     xlC     IBM C++ compiler
stats.asu.edu         cc      standard HP-UX C compiler
.                     c89     HP-UX POSIX-conformant C compiler
.                     CC      HP C++ compiler
To get specific details on these compilers, consult the online information on the appropriate server with the following commands: 
SERVER      COMMAND   INFO IT PRODUCES
------      -------   ----------------
research*   man CC    produces an overview of the IBM cc compiler, 
		      e.g.,"Key Concepts", file types, preprocessor 
		      description, etc.)
research*   man cc    gives info on IBM's cc, xlc, and c89
.                     "man cc", "man xlc", and "man c89" produce 
		      the same info. 
stats**     man cc    gives info on HP's cc and c89 compilers
research*   xlC       displays info on the IBM C++ translator/compiler
stats**     man CC    displays info on HP's C++ translator/compiler

* "research" refers to research.asu.edu

** "stats" is stats.asu.edu
