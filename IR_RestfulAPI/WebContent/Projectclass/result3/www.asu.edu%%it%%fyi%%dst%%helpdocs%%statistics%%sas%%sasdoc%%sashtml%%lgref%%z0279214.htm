SETINIT

SETINIT


Controls whether site license information can be altered

Valid in: configuration file, SAS invocation
Category: System administration: Installation
PROC OPTIONS GROUP= INSTALL

Syntax
Syntax Description
Details

Syntax

SETINIT | NOSETINIT

Syntax Description

SETINIT
In a windowing environment, allows the SETINIT window to be displayed so that you can make changes. In a non-windowing environment, SETINIT enables you to change license information by running the SETINIT procedure.

NOSETINIT
does not allow you to alter site license information after installation.

Details

SETINIT is set in the installation process and is not normally changed after installation.
