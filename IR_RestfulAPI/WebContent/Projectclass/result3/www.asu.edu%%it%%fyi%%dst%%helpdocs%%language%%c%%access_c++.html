Accessing C++

C++ is available on the UNIX servers. To access C++ on stats.asu.edu, enter:

CC source.C

For C++ on research.asu.edu, use:

xlC source.C

For all C++ compilers, there are two important requirements:

  • a capital C must be used as the extension on your C++ source code file
  • the source program must include stdio.h, i.e., you must have the following statement in your source code file:
    #include <stdio.h>

Additional information on C++ options is avaialble online. To display this info:

  • On research.asu.edu, type: man xlC
  • On stats.asu.edu, type: man CC
