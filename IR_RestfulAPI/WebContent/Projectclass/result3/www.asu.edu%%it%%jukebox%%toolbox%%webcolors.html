Web Safe Color Chart

Although colors can appear differently from machine to machine when designing for the web, the following 216 colors are the colors used by all versions of Netscape regardless of platform. You can't go wrong by using any of these in your web pages. The color codes are shown in each box and you can copy and paste the code to your HTML for use in your own pages.
