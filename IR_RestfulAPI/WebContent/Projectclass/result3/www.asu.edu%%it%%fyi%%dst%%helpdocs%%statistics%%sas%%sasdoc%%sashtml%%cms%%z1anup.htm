CLEANUP

CLEANUP


Specifies whether to attempt to recover from an out-of-resources condition

Default: CLEANUP
Valid in: configuration file, SAS invocation, OPTIONS statement, OPTIONS window
CMS specifics: default, usage

Syntax
Details
See Also

Syntax

CLEANUP | NOCLEANUP

CLEANUP
specifies that SAS software should attempt to recover from an out-of-resources condition.

NOCLEANUP
specifies that SAS software should not attempt to recover from an out-of-resources condition.

Details

By default, the requestor window is displayed only when SAS is invoked in interactive windowing mode or in interactive line mode. In other methods of running SAS software, when a resource-critical situation arises, SAS software performs automatic continuous cleanup. If not enough resources are recovered, SAS software responds as if you had specified NOCLEANUP.
