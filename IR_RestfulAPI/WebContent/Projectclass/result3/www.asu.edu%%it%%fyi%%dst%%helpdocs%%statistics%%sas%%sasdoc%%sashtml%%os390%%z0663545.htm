WIDGPREV


Moves the keyboard focus from one widget to the previous widget

OS/390 specifics: all

Syntax
Details
See Also

Syntax

WIDGPREV

Details

With the keyboard focus on a widget in a window, entering the WIDGPREV command moves the keyboard focus to the previous widget in the window, in a manner similar to that seen with the SHIFT+TAB keys. For example, issuing WIDGPREV in the SAS Explorer window moves the keyboard focus between the list view and the tree view.

The WIDGPREV command is best used by assigning the command to a function key. Enter the KEYS command to display and edit function key assignments.

To use a function key to issue the WIDGPREV command, position the cursor in a text entry field and press the function key.
