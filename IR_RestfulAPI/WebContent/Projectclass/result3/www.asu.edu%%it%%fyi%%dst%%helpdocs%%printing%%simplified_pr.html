Simplified Printing

Information Technology has set up a simple menu system that let's you select a default printer for printing from one of its UNIX servers, Pine E-mail, Tin Netnews Reader, Lynx World Wide Web, or Gopher under UNIX. The print facilities within the applications listed above will automatically send output to the selected printer. When you print using "lpr" from one of the UNIX servers, a special version of "lpr" will be used by default, which will also print to the selected printer.

To select a default printer, type:

menu
at the UNIX prompt. After it displays its list of software services, type the number zero (0) to proceed to Menu #2. This will list several options, such as:
  1. Edit Your Signature File
  2. Change Your Password
  3. Select Default Printer
  4. Pine Info Center (U. of Wash.)
Type the number three (3) to select a default printer. A list of printers will be displayed, for example,: 
     |=======================================================|
     | Name      Description                                 |
     |=======================================================|
     | UCC1     Computing Commons Site 1st floor (Landscape) |
     | WEST1    ASU West Fletcher Library                    |
     | ASUELAB  ASU East Computing Commons Lab Printer       |
     | ...      ...                                          |
     |=======================================================|
Scroll through the list until you find a printer you'd like to use. To select a printer, simply hit the RETURN key while it's highlighted. Then type the letter "E" to Exit to Menu #1, and then type the letter "E" again to Exit from the "menu" program. Note that if no printer is selected, all printouts will be sent to UCC1, the printer located in the Computing Commons Atrium.

Also, if you are selecting a non-computing site printer, please check at the location of the printer to make sure that it's acceptable for you to print there.

Furthermore, if you need to use the output class and bin number options in the lpr command, please refer to the next selection.
