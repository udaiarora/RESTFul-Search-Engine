Overview

The Tree View Control enables you to create a hierarchical list of different text elements, or nodes. Users can expand and contract the list, similar to how one might navigate a directory structure of files, an index, or a table of contents.

A tree view control is typically used to display information that is organized in a hierarchy. Nodes in the tree view can contain both a bitmap and a text label. The text is always displayed in a vertical hierarchy.
