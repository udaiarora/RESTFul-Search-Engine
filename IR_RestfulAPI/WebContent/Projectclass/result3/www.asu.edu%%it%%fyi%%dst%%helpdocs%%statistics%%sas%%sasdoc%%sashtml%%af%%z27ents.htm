External File List Model Events

Events specified for the External File List Model class are described here. See Selection Model Events for inherited events.

directoryDisplayed changed
Occurs when the directoryDisplayed attribute is changed

externalPathDisplayed changed
Occurs when the externalPathDisplayed attribute is changed

fileDisplayed changed
Occurs when the fileDisplayed attribute is changed

filerefDisplayed changed
Occurs when the filerefDisplayed attribute is changed
