Data Form: _getCurrentPage

Data Form: _getCurrentPage


Returns the number of the displayed page

Syntax
Details

Syntax

CALL SEND (form-id, '_getCurrentPage', page-num);

Argument Type Description
page-num
 N returns the number of the displayed page

Details

The _getCurrentPage method always returns a number greater than 0 because the data form always has at least one page defined.
