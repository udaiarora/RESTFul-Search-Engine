$REVERSw.

$REVERSw.


Writes character data in reverse order and left aligns

Category: Character
Alignment: left

Syntax
Syntax Description
Comparisons
Examples

Syntax

$REVERSw.

Syntax Description

w
specifies the width of the output field.
Default: 1
Range: 1-32767

Comparisons

The $REVERSw. format is similar to the $REVERJw. format except that $REVERJw. does not left align the result.

Examples

put @1 name $revers7.;
