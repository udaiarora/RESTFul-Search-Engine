Object Class: _getEvents

Object Class: _getEvents


Returns metadata for all registered events on a class or object

Syntax

Syntax

return = objectName._getEvents( metadata );
return = objectName._getEvents( metadata, inherited );

Argument Type Use Description
return Numeric Return returns a value indicating whether the events where successfully retrieved: 0 if successful, 1 if not successful
metadata List Update returns the list with sublists containing event metadata
inherited Character Input specifies whether inherited events are returned in the metadata list: 'Yes' returns INHERITED events, 'No' does not
