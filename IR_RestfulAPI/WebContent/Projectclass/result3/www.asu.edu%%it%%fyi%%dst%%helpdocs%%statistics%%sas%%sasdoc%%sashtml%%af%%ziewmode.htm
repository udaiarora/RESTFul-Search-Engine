Chart Control: viewMode


the view mode (2D, 3D or 3DPerspective)

Type: Character
Valid Values: 2D 3D 3DPerspective

Details

Details

Mode Description
2D Height and width
3D Height, width, and the illusion of depth. Graphs appear to have depth, but you cannot view them from below or above.
3DPerpective Height, width, and true depth
