Access from Macintosh

From an IT Computing Site

Following are instructions for access to an Instructor Volume from a Mac in an IT computing site.

Go to any Information Technology computing site. Start up a Macintosh in the normal fashion. If you have an ethernet-connected Macintosh on the ASU network, you can do this from your own Macintosh.

To connect to the IT Instructional file server in the Information zone with your Instructor account:

  1. Pull down the Apple Menu and open the Chooser.
    d
  2. Select the AppleShare icon in the box at the upper left.
  3. Highlight the Information Technology zone in the AppleTalk zones box at the lower left. A file server name will appear in the box to the right.
  4. Highlight the IT Instructional file server name.
  5. Click OK. A new dialog box will appear.
    d
  6. Type in your ASURITE UserID in the Name field.
  7. Tab to the Password field and enter your password (passwords are case sensitive).
  8. Click OK and a new dialog box will appear.
    d
  9. Select the Instructor Volumes server volume name and click OK.
  10. Close the Chooser by clicking in the box in the upper left corner.
  11. After you have connected to the file server with your account, double-click the Instructor Volumes icon on the desktop, then open up your instructor folder.

Note: Students will have access only to their specific instructor folder.

Note: It is not possible to apply disk space quotas to AppleShare networks. Please do not abuse this privilege by wasting the available disk space. Those who do so may see their account and instructor folder removed.

Be aware of copyright infringements. If you will be installing an application in your instructor folder for class instruction given on the Macintosh networks, you should have enough legal copies of a copyrighted application for as many as will use it simultaneously. You will also be unable to prevent the copying of your applications from your instructor folder on the network server.
