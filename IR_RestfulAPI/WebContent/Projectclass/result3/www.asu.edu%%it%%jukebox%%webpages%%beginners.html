Introduction for Beginners

Welcome! You probably just finished setting up your webspace or gaining access to space somewhere on campus and are now ready to dive in. This page is here to help you get started. The Introduction to the Internet Section is a great place to start if you're completely new to the web and want to learn more about how this whole thing works. If you have an idea of what's going on, then you might want to start with the Introduction to the ASU Web section to acquaint yourself with terms, policies, and resources. Good luck with your pages!
