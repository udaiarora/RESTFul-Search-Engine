Services

The following network services are available through the ASU wireless network:

Guest Login

  • Access is limited to web only (http and https)
  • The following VPN connections are supported:
    • CheckPoint
    • Cisco
    • PPTP
  • Systems are scanned externally when connecting to the ASU wireless network looking for known vulnerabilities. If a known vulnerability is found, you will be given information on fixing your system before being allowed onto the ASU network.
  • Keep your system up-to-date and secure with:
    • System patches & fixes
    • Application updates
    • Antivirus software (check for update definitions daily)
    • Check for spyware on a regular basis
  • For assistance contact the IT Help Desk at 480-965-6500

ASURITE Login

  • Full Access to the Internet and resources at the ASU (some university resources require VPN access)
  • VPN use is highly recommended for a more secure connection - for more information on ASU VPN see the Computer Security web site.
  • Systems are scanned externally when connecting to the ASU wireless network looking for known vulnerabilities. If a known vulnerability is found, you will be given information on fixing your system before being allowed onto the ASU network.
  • Keep your system up-to-date and secure (visit the ASU Computer Security web site for information on securing your workstation):
    • System patches & fixes
    • Application updates
    • Antivirus software (check for update definitions daily)
    • Check for spyware on a regular basis
  • For assistance contact the IT Help Desk at 480-965-6500
