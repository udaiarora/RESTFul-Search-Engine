Frame Class: _getDropReps


Returns a list of the object's currently defined drop representations

Syntax
Details

Syntax

objectName._getDropReps( representationsList );

Argument Type Use Description
representationsList Numeric Update specifies the identifier of the SCL list that contains the drop representations operations

Details

Note that you can modify and return reresentationList to the _setDropRep method using call Apply to perform global changes in the drop representations.
