Overview

The Catalog Entry Model class enables a text viewer to display test-based catalog entries, such as LOG, OUTPUT, SCL, and SOURCE entries. Text-based entries are typically generated using the text editor window. This model is automatically attached to a catalog entry viewer (see the Catalog Entry Viewer class), and may also be attached to a text viewer (see the Text Viewer class).

In addition, you can subclass the catalog entry model to create a custom model that provides more specific functionality or displays text from a source not already covered with the catalog entry model.

The Catalog Entry Model class is a subclass of the Text Model Template class.

The Catalog Entry Model and related classes use the model/view paradigm described in SAS/AF online help.
