SAS/EIS Software: Administrator's Guide, Version 8

Learn how to set up and customize the application development environment for your SAS/EIS users. This documentation describes the features available with SAS/EIS software to define and control user access to data and applications. Through a series of examples and answers to frequently asked questions, you will understand how to establish security controls for your applications and the data they use. An easy-to-use interface, described in detail, simplifies the administrator's access-related tasks. When you use the security access windows to customize your site's environment, online help is available to describe the window components and to assist you in completing the required tasks.
