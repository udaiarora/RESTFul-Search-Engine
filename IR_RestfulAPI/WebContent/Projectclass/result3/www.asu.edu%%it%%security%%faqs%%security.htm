How can I tell I am at a secure web site?


If you are on a secure site, the URL (Universal Resource Locator) path will start with https://. Notice the s after http, this signifies a secure site.

What is a firewall?


A Firewall is a system designed to prevent unauthorized access to or from a network or computer. Firewalls can be implemented in both hardware and software, or a combination of both.

Firewalls are used to prevent unauthorized users from accessing a network or computer. All messages entering or leaving pass through the firewall, which examines each message and blocks those that do not meet the specified security criteria.

Firewall protection is especially useful for users with "always-on" connections such as DSL or cable modem. Such connections use a static IP address that makes them especially vulnerable to potential hackers.

Should I open all E-Mail attachments?


It is not good security/anti-virus practice to open E-Mail attachments you are not expecting, even from a friend or coworker. If you receive an attachment from a known source but you were not expecting it E-Mail the sender to verify what the attachment is and that they actually sent it. You should never open an E-Mail attachment from an unknown source.

It is also a good idea to have Anti-Virus software installed and configured to scan your E-Mail and attachments. ASU provides Anti-Virus software for all ASU students, faculty and staff. You can find out more about ASU's Anti-Virus software on the Anti-Virus WEB page.
