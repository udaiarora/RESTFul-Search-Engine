Changes and Enhancements to SAS/ETS Software in Versions 7 and 8

The Output Delivery System

All procedures now incorporate the Output Delivery System (ODS). This is a system for managing the results of a procedure. By default, the results for a procedure are directed to the SAS listing file as in previous releases, but with ODS you can create HTML or RTF files, create SAS output data sets of any table in the output, select or exclude pieces of output from a procedure, or modify the organization and style of that output. Refer to Chapter 6, "Using the Output Delivery System," or The Complete Guide to the SAS Output Delivery System for more information.
