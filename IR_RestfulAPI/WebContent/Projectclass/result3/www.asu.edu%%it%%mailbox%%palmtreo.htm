Palm Treo Support

At this time, ASU is not able to support the Treo 600 and the Tungsten T5.  They will synchronize with your Exchange account, but only by using the IMAP or POP3 protocol. ASU IT does not recommend this and actually discourages it, as e-mail that leaves the ASU Exchange servers is no longer recoverable.

Palm has introduced the new Treo 650 & 700 which are able to synchronize with the Exchange servers similar to the Blackberry devices. But unlike Blackberry, which requires an account on the Blackberry server, wireless synchronization to the Exchange server is enabled by default and no additional IT licenses are required.

For more information on the Treo's and other Mobile Messaging information, please visit http://www.asu.edu/mailbox/mm/.
