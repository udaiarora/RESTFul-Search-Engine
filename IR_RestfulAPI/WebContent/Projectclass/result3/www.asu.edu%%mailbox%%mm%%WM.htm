Q. What is Windows Mobile?

A. Windows Mobile is the operating system Microsoft has created for Pocket PC's and Smartphones to give you the same look and feel as your desktop or laptop computer. Windows Mobile gives you Pocket Applications like Word, Excel, PowerPoint, Outlook, a .PDF reader and Windows Media player, just to name a few.

Microsoft has improved on its e-mail platform (Exchange) and its Mobile OS (Windows Mobile) to give you the next generation in Mobile Messaging using a wireless synchronization that, before this, was only available in a Blackberry Device. Earlier versions allowed a user to download their messages from the server using a POP3 service, but created issues in sending mail, keeping like folders updated, and in keeping mail on the server for disaster recovery (or a dead battery). With these issues being solved, Windows Mobile Devices can now be offered as a comparable alternative to a Blackberry solution.

Q. Why should I get a Windows Mobile 5 Device?

A. Windows Mobile 5 was released in mid-2005 and has proven to solve many of the issues that, in the past, have caused headaches with device owners.

We'll start with Persistent Memory, this tells the handheld to store its system settings and personal data in a less volatile memory space.  In earlier versions all your data was stored on RAM (phone numbers, personal settings, etc.), which would be lost if the device lost power or had to have a hard reset.   In Windows Mobile 5, the device uses it's ROM to store this information, leaving the RAM for application use.  When the PDA boots, the necessary program files are copied to the RAM to be used, this can also cause slight delays in booting a device but at least your phone numbers are still there when you get to the car charger!

Also included in Windows Mobile 5 are improved applications.  It comes with Pocket Media Player 10, specific enhancements to the original Pocket Office Suite, and it adds on a Pocket PowerPoint.

Can you upgrade???  Some NEW devices will allow you to upgrade this software.  The upgrade is device specific and if it is available would be provided by your wireless vendor or the manufacturer of the device.  Most upgrades are scheduled to be made available in the 1st quarter of 2006.  But in the meantime, we recommend buying the device that already has Windows Mobile 5 or the device that you know will be able to be upgraded.  Check with www.microsoft.com/windowswmobile for more detailed product release information.
