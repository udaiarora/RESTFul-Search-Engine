TABLE: DARS_STUDENT_DEGREE_PROGRAM

What is the DARS Student Degree Program Table?
A record is created in this table when a student submits a Declaration of Graduation (DOG) or if an advisor creates a "D" screen to record a concentration for a BIS student. There will be one record in this table for each student showing their degree program. If a student has declared a double major, the student will have 2 records, one for each degree program; or 3 records for 3 concurrent degrees.

What is the key (unique identifier for a row) for the DARS Student Degree Program Table?
The key for this table is the student's ASU ID.

What is the timing of the DARS Student Degree Program Table?
This table is completely replaced on a weekly basis. It is usually run early Monday morning. Updating this table is dependent on the update of the DARS Reporting database, which takes place on Sunday.

Why would you use it?
This table can be used to find all of the students in a particular degree program and/or catalog year. It also can be used to find students who have a dual major or have filed a DOG. This is the best source to obtain data for a DARS batch audit run.

Why would you NOT want to use DARS Student Degree Program Table?
If you are looking for students with minors, concentrations, or certificates, the information is not in this table; it is in the DARS Student Marker Requirement table.

What should you look out for?
· Degree Status Field is key to knowing if data is from a DOG on file, student is undergraduate or graduate, and if student has graduated.
· SIS college, major, and degree codes are from the student's official SIS record; the DARS college, major, and degree codes are those which are expected with the DARS degree program in that record.
· To obtain the data for a DARS batch audit run, request the following fields: ASU ID, DARS Campus Location, DARS Degree Program Code, DARS Student Catalog Year Sem, and if applicable, DARS Degree Program Mask Code.
· A small number of students still have a Program of Study (POS) on file in SIS, so there will not be a record for them in this table unless they have a dual degree and a DOG was filed for one of the degrees.
· A search for dual degrees will usually return at least two records for each ID, except if one of those degrees has a POS on file in SIS.
· DARS Dual Degree Flag: Y indicates student is pursuing or has concurrently pursued more than one degree.
· Currently Enrolled Flag: Set to Y to get only enrolled students.
