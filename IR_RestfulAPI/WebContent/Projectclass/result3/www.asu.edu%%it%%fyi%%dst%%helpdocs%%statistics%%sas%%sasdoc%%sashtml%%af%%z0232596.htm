GET_DATA Data Vector: _setText

GET_DATA Data Vector: _setText


Sets the text

Syntax
Details

Syntax

CALL SEND (data-vector-id, '_setText', text);

Argument Type Description
text
 C specifies the formatted text to be displayed in the viewer

Details

The _setText method is valid only if the viewer is a table editor. If the viewer is a form editor, use _setUnformattedText or _setUnformattedValue.
