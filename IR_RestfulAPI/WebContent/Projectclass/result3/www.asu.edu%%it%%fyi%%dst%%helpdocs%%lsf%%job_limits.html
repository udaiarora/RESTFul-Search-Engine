What are the current job limits on the batch server?

Number of clusters that have batch servers

There are a 2 clusters that with servers available for batch jobs.
  • Research/Statistics cluster has a server called Batch1 with 4 processors.
  • Medusa is a high performance computing system with 8 processors that can also be used for batch jobs.

Number of jobs per batch-server per user

  • The maximum number of jobs a user can execute on Batch1 is unlimited except its "compute.long" queue has a limit of 4.
  • The maximum number of jobs a user can execute on Medusa is 2.

Number of jobs per cluster

The maximum number of batch jobs that can execute on the Research/Statistics cluster is:
  • Maximum of 24 in the "compute.long" queues.
  • Unlimited in the "computer.medium" queues.
  • Unlimited in the "compute.short" queues.
The maximum number of batch jobs that can execute on the Medusa is:
  • Maximum of 5 in the "medusa.long" queues.
  • Unlimited in the "medusa.medium" queues.
  • Unlimited in the "medusa.short" queues.
