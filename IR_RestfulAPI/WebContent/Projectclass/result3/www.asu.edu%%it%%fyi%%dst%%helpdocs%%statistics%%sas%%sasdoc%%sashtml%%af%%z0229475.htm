Icon: _setColor


Assigns a specified color to an icon

Restrictions Some systems do not support using methods to change the color of an icon.

Syntax

Syntax

CALL NOTIFY (icon-name,'_setColor', color-name);

Argument Type Description
color-name
 C specifies the name of the color to assign. The color-name argument is not case sensitive. SASCOLOR window elements can also be used for color-name.
