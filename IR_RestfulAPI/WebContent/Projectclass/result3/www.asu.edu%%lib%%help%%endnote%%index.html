Accessing the ASU Online Catalog Using EndNote Software

 

Please note that ASU now licenses and supports RefWorks online citation management software. You can import your EndNote citations into RefWorks if you decide to switch to the licensed product.

 

The following procedure explains how to configure EndNote software to search and retrieve records from the ASU Libraries' Online Catalog using the Z39.50 protocol. Z39.50 access is available ONLY for the ASU Libraries' Online catalog.

 

A copy of the connection file you need to search in the ASU Online Catalog can be downloaded from this page. To save it to your computer RIGHT click on this link, then save it to the Connect folder in your EndNote directory. You should call the file "ASU LIBRARIES".

 

 

To use EndNote to search the ASU Libraries Catalog, go to the File menu, select "Connect". In the "Select a Connection file to Open" dialog box, open ASU Libraries.enz.

 

See EndNote documentation for specific information regarding searching Z39.50 databases and creating bibliographies.


Z39.50 access is not available for every database or index.


A Z39.50 gateway is not necessary to connect EndNote with online bibilographic databases. Generally, you can use EndNote's "filters" to import text files saved or downloaded from these sources. The only restriction is the data must be consistently "tagged" with each tag starting on a new line, and the file saved as ANSI or ASCII text. For example, EndNote supports importing references downloaded from SilverPlatter online databases such as WebSpirs. See the method to download SilverPlatter database records for use with EndNote in the manual accompanying the software. In general, when you have found references in WebSPIRS, you will use the Save button to save your search results to a file. Choose Options and set the options to show field names as short labels and to show all fields (Once you have set the options, click Back to Save). Next, Click the box to include record numbers, then click the Save Records button. The SilverPlatter import filter folder contains filters for various databases that can be used to import the data. Note: Before downloading references from a database, carefully check the copyright and fair use notices for the database. Various information providers may have different restrictions for each database.

 

EndNote is a product of Thompson Resarch. Questions regarding problems or the use of the software should be directed to EndNote Technical Support.
