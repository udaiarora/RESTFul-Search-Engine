UNIVERSITY SECURITY COMMITTEE (USC)

The committee is responsible to the executive vice president for administration and finance. It recommends policy for security infrastructure systems, processes, and program responsibilities; develops, reviews, and updates security plans as appropriate; prioritizes recommendations; serves as a clearinghouse for all security issues; identifies possible fund sources for each program or project implementation; and makes recommendations to existing administrative governance committees of actions to be taken and appropriate funds sources.
