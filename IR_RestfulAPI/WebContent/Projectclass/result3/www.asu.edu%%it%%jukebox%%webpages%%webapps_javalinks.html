Webapp Development Resources - Java Reference Links

Java Essentials

  • Java Tutorials and Style Guidelines
    • Sun Computer's Java tutorials - coding, swing, beans - everything Java.
    • Java style guidelines, a pithy set of rules for coding style compiled from Sun, Netscape and NASA standards.
    • Code Conventions for Java from Sun.
    • Fairly friendly Java tutorial titled "From the Grounds Up".
  • Java books
    • Thinking in Java, 2nd Edition: here's where to find the PDF version of Bruce Eckel's book. After loading, do a "save as" if you want a copy on your desktop. (5 meg.) Also check out author's Website for lots of OOP stuff. There's a Third Edition due out soon, apparently.
  • Servlets and JSPs
    • Fundamentals of Java Servlets, a short course by jGuru hosted by Sun. Includes exercises.
    • JavaServer Pages Fundamentals, another short jGuru course at Sun's site. They say to take the Servlet course first.
    • Servlet and JavaServer Pages 1.0: A Tutorial. By Marty Hall, who wrote "Core Servlets and JavaServer Pages" for Sun.
    • Sun's JavaServer Page home page. There are lots of resources here, but the following two may be particularly helpful:
      • Sun's JSP syntax quickref and related links.
      • Sun's JSP FAQ.
    • Sun's J2EE tutorial. Download some software and datafiles, then run thru the tutorial to learn how to write JSPs and JavaBeans and how to use XML.
    • More JSP resources at the Java Resource Index.
  • JSPs and Beans
    • Building Your Own JSP Components: What makes a Bean a Bean, and how do you use them with JSPs? From iPlanet.

Java security

  • General Web security info
    • Javaworld's Web security nuts and bolts. Printable overview with good definitions.

Discussion lists, users' groups

  • Local groups
    • The ITJAVA-L discussion list for Java- and Web-app-related discussion. Belongs to administrative developers and support staff. You'll need to log in with your ASU listserv account, or create one if you don't have it already.
  • Other Web-based groups
    • The comp.lang.java FAQ List. Scroll down to Section 2 for links to Java newsgroups and discussion lists.
