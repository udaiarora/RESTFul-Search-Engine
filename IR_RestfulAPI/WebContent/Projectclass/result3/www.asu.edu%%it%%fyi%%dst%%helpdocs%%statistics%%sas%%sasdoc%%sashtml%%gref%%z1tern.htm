CPATTERN

CPATTERN


Selects the default color for PATTERN definitions when a color has not been specified.

Used in: GOPTIONS statement
Default: first color in current colors list

Syntax
Details

Syntax

CPATTERN=pattern-color

pattern-color
specifies any SAS/GRAPH color name. See SAS/GRAPH Colors for information about specifying colors.

Details

CPATTERN= is overridden by any color specification in a PATTERN statement. For details on how CPATTERN= affects the PATTERN statement, see The Effect of the CPATTERN= Graphics Option.

If you specify CPATTERN=, the solid pattern is skipped and the patterns rotate through only the appropriate default hatch patterns for the graph. See PATTERN Statement for more information on pattern rotation.
