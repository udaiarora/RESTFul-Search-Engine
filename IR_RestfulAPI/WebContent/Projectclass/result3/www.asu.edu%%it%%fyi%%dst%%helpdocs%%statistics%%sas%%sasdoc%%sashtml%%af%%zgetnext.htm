Tree View Control: _getNext


Returns the next sibling node

Syntax

Syntax

objectName._getNext( siblingNode, nextSiblingNode );

Argument Type Use Description
siblingNode Object Input specifies the node to retrieve the next sibling from
nextSiblingNode Object Output returns the next sibling of the specified node
