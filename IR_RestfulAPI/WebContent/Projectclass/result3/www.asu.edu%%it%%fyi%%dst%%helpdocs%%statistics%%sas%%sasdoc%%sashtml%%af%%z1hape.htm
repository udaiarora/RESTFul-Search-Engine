Frame Class: _resetCursorShape


Resets the cursor shape (mouse pointer shape) of the frame

Syntax
Details

Syntax

objectName._resetCursorShape( );

Details

If the cursor shape had been changed through the _setCursorShape method, the original cursor shape can be restored using _resetCursorShape.
