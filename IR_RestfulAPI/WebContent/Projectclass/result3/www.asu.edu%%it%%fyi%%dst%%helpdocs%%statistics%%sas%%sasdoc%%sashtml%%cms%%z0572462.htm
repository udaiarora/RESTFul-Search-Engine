OPLIST

OPLIST


Controls whether to write the settings of SAS software options to the SAS log

Default: NOOPLIST
Valid in: configuration file, SAS invocation
CMS specifics: all

Syntax

Syntax

OPLIST | NOOPLIST

OPLIST
writes the settings of SAS software options to SAS log.

NOOPLIST
does not write the settings of SAS software options to SAS log.
