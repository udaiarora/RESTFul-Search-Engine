SciFinder Scholar


Chemical Abstracts
Use restricted to ASU faculty and students for ASU educational purposes only.
Click "EXIT" button after use to allow other user access. If all sessions are taken, please try again later.
The database is not accessible on some weekend hours due to system maintenance. For specific operating times please check http://www.cas.org/SCIFINDER/SCHOLAR/newhours.html.
CAUTION: You must exit or close down the browser after using SciFinder. Failure to do so may allow others to access the product using your session and you may be held liable for any violations that may occur.

Database Information

Description:
(1900+) Electronic version of Chemical Abstracts, the world's largest and most up-to-date collection of chemical information. Covers journal articles, patents, technical reports, books, conference proceedings, and dissertations from around the world.
Date Range:
1900+
Updated:
Daily
Includes:
bibliographic citations; abstracts
Subject Coverage:
Biochemistry; Chemical Engineering; Chemistry
Types of Materials Covered:
journal articles; patents; technical reports; books; conference proceedings; dissertations
Geographic Scope:
International
Content Provider:
Chemical Abstracts Service
Alternate Methods of Access:
Print Equivalent:
  • Chemical Abstracts; see the ASU Libraries' Catalog for volume availability and location.
Software Requirements:
Because of software requirements, this database can be accessed only from ASU Library workstations.
Tips:
  • Searching instructions are available at
    http://www.cas.org/SCIFINDER/SCHOLAR/resources/html
    • Brief instructions for how to conduct the different types of searches are located under "See How It Works"
    • In-depth, online seminars for using specific features or conducting specific tasks are available by clicking on SciFinder e-Seminar Listing. Seminars range from 8-60 minutes. If given a choice between "subscription" or "task" for a specific seminar, always select "subscription".

  • The Help Guide for SciFinder Scholar is available at: http://www.cas.org/SCIFINDER/help/2006/SCH_Help/Ctxt_eps/default.htm.

  • To download citations into RefWorks:
    • Mark the records in SFS
    • Click on Save As button
    • Save file in Tagged Format
    • Open RefWorks
    • Under References, click on Import
    • Set Import Filter/Data Source to CAS SciFinder
    • Set Database to either or CAPlus as appropriate
    • Specify the folder (if folder is not specified, references will go into the Last Imported folder)
    • Click on Browse to find the downloaded file
    • Click on Import

  • To download citations into EndNote, ProCite and Ref Manager, see http://www.cas.org/Support/scifinder2006/bibapps/bibapps.html
