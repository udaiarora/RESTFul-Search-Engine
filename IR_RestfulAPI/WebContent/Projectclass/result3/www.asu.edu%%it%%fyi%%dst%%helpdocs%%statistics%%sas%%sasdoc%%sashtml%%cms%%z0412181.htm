VERBOSE

VERBOSE


Controls whether SAS writes the settings of SAS software to either the terminal or the batch log

Default: NOVERBOSE
Valid in: configuration file, SAS invocation
CMS specifics: all

Syntax

Syntax

VERBOSE | NOVERBOSE

VERBOSE
writes configuration options.

NOVERBOSE
does not write configuration options.
