JCL Procedures to Compile, Link, and/or Run

The system provides several cataloged procedures which simplify the use of COBOL. All of the EXEC statements and most of the required DD statements are already included in the cataloged procedure.

The following sample COBOL batch jobs illustrate use of the various COBOL cataloged procedures.

COB2C
compile only
COB2CG
compile and go (execute using the loader)
COB2CL
compile and link edit
COB2CLG
compile, link edit, and go (execute)

Normally, COBOL programs are compiled using COB2C until they compile successfully (no error diagnostics). Then COB2CG is used to compile and execute the program to test for proper program execution. When the program is error-free it can be stored in a load module library, using either COB2CL or COB2CLG.
