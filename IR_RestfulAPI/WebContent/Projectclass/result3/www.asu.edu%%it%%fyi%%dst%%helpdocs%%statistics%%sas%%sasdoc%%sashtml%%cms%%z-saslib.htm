SASLIB=

SASLIB=


Identifies a text library to be made global by SAS

Default: none
Valid in: configuration file, SAS invocation
CMS specifics: all

Syntax

Syntax

SASLIB=text-library-specification

text-library-specification
is the filename of a CMS file that has a filetype of TXTLIB.
