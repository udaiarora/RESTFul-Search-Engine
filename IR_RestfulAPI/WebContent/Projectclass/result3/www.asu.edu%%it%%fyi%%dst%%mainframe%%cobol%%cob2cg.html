// JOB statement ==> replace this statement with a valid JOB statement
//* COB2CG - Compile and Go (Execute)
//*
//*     //   EXEC  COB2CG,optional parameters
//*     //COB2.SYSIN  DD *
//*          COBOL Source Statements
//*     //GO.ddname  DD  statements as needed (data for program)
//*     //
//*
//*   This procedure compiles and executes a COBOL program.  Neither an
//*   object module nor a load module is saved.  A GO DD statement must
//*   be given for each input and output data set that your program uses
//*   during execution.
//*
//*   The ddname on a GO step DD statement must correspond with the
//*   ddname in an assignment-name in the SELECT clause.  Refer to
//*   section 4 for details.
//*
//TIME       EXEC  COB2CG
//COB2.SYSIN  DD    *
       IDENTIFICATION DIVISION.
       PROGRAM-ID. TTIME.
       ENVIRONMENT DIVISION.
       INPUT-OUTPUT SECTION.
       FILE-CONTROL.
           SELECT OUTPUT-FILE  ASSIGN TO SYSOUT1.
       DATA DIVISION.
       FILE SECTION.
       FD  OUTPUT-FILE LABEL RECORDS OMITTED
           RECORDING MODE IS F
           DATA RECORD IS OUTPUT-REC.
       01  OUTPUT-REC                           PIC X(132).
       WORKING-STORAGE SECTION.
       01  DATE-RECORD.
           02 YR                                PIC IS 9(2).
           02 MN                                PIC IS 9(2).
           02 DY                                PIC IS 9(2).
       01  TIME-RECORD.
           02 HR                                PIC IS 9(2).
           02 MIN                               PIC IS 9(2).
           02 SEC                               PIC IS 9(2).
           02 HUNDREDS                          PIC IS 9(2).
       01  OUTPUT-DATE-RECORD.
           02 FILLER         PIC IS X(14) VALUE ' THE MONTH IS '.
           02 MN-OUT         PIC IS 9(2).
           02 FILLER         PIC IS X(12) VALUE ' THE DAY IS '.
           02 DY-OUT         PIC IS 9(2).
           02 FILLER         PIC IS X(13) VALUE ' THE YEAR IS '.
           02 YR-OUT         PIC IS 9(2).
       01  OUTPUT-TIME-RECORD.
           02 FILLER         PIC IS X(20) VALUE ' THE TIME OF DAY IS '.
           02 HR-OUT         PIC IS 9(2).
           02 FILLER         PIC IS X(1) VALUE ':'.
           02 MIN-OUT        PIC IS 9(2).
           02 FILLER         PIC IS X(1) VALUE ':'.
           02 SEC-OUT        PIC IS 9(2).
           02 FILLER         PIC IS X(1) VALUE ':'.
           02 HUNDREDS-OUT   PIC IS 9(2).
       PROCEDURE DIVISION.
       BEGIN-DATE-TIME-PARA.
           OPEN OUTPUT OUTPUT-FILE.
           ACCEPT DATE-RECORD FROM DATE.
           ACCEPT TIME-RECORD FROM TIME.
           MOVE MN TO MN-OUT
           MOVE DY TO DY-OUT
           MOVE YR TO YR-OUT
           WRITE OUTPUT-REC FROM OUTPUT-DATE-RECORD.
           MOVE HR TO HR-OUT
           MOVE MIN TO MIN-OUT
           MOVE SEC TO SEC-OUT
           MOVE HUNDREDS TO HUNDREDS-OUT
           WRITE OUTPUT-REC FROM OUTPUT-TIME-RECORD.
           CLOSE OUTPUT-FILE.
           STOP RUN.
