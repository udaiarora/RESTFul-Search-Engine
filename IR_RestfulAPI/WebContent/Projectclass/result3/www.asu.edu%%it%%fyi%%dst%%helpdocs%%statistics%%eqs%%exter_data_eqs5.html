How to Read an External Data File in EQS 5.1

The enclosed procedure should be followed before accessing an external data file. This example, assumes that you are working with an EQS syntax file called 'prog.eqs', an external data file called 'datafile.dat', and an output file called 'prog.out'.

  1. Rename the data file with upcase letters. In this case, 'datafile.dat' becomes 'DATAFILE.DAT'. This can be done with the command:
    mv datafile.dat DATAFILE.DAT

  2. Add the following to your EQS systax file:
    DATA='DATAFILE.DAT';

    For example:
    /TITLE
    This is just a Test
    /SPE
    CASES=93; VARIABLES=9; ME=ML,ROBUST;
    DATA='DATAFILE.DAT'; MA=RAW; AN=COV;

    Note: The use of single quotes for the DATA file.

  3. Using the name of the data file in the EQS command is optional.

    For example, instead of the command:

    eqs -i prog.eqs -d DATAFILE.DAT -o prog.out -m i

    You can simply use:
    eqs -i prog.eqs -o prog.out -m i
