WIDGNEXT

WIDGNEXT


Moves the keyboard focus from one widget to the next widget

CMS specifics: all

Syntax
Details

Syntax

WIDGNEXT

Details

With the keyboard focus on a widget in a window, entering the WIDGNEXT command moves the keyboard focus to the next widget in the window, in a manner similar to that seen with the TAB key.
