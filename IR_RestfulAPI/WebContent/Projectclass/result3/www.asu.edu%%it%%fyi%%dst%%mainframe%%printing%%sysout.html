SYSOUT Classes

Class Paper                        Comments
----- ---------------------------  -------------------------------
 A    plain white (8 1/2 x 11)     Computing Site printers, 8 lpi
      (UCC1 is landscape)
      (UCC2 is portrait)
 B    none                         "punched card" class
 D    plain white (8 1/2 x 11)     Upper/lower case, 8 lpi
      (UCC1 is landscape)
      (UCC2 is portrait)
 M    Mailing labels               User must supply blank forms;
                                     printed 8 lpi with 48-line
                                     printer control loop.
 S    Special                      Use requires prior permission
 W    None                         TSO "hold" class
 Z    None                         Output not kept

Notes

  • Historically, class A was used for output for which it did not matter whether the printer was capable of lowercase or not, and class D was used for guaranteed upper/lowercase. Now, both classes are upper/lowercase.
  • Historically, class B was used for punched cards (80-column). Card punchers and card readers are no longer available at ASU. Card services may be available from local service bureaus on a fee-for-service basis.
