Communications Access Methods for SAS/CONNECT and SAS/SHARE Software

Supported Communications Access Methods by Host

For SAS/CONNECT, the matrix shown in SAS/CONNECT: Valid Access Methods Between Hosts identifies the communications access methods that you can use to make a connection between a local and a remote host. Unless you are using the DOMAIN server, as described in Using a DOMAIN Server for Cross-Access Method Communication, the access method that you use must be available on both hosts.

CAUTION:
Only Version 6 supports the protocol gateway of the DOMAIN server.  

A similar matrix for SAS/SHARE is shown in SAS/SHARE: Valid Access Methods Between Hosts. It shows the communications access methods that are available to a client host for accessing a remote SAS/SHARE server.

Also, for a list of access methods that are supported by host and by release number for SAS/CONNECT and SAS/SHARE, see SAS/CONNECT and SAS/SHARE: Supported Access Methods According to Host Release.

Note:   This chart shows access methods supported in Version 6, Version 7, and Version 8.
