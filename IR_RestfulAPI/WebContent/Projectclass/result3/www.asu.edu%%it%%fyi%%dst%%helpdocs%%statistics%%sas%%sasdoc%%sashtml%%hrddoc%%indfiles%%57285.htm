SAS/ACCESS Interface to ADABAS Software, Version 8

This title provides complete details for using the ACCESS procedure in a non-windowing environment and contains reference information required by an applications programmer to write, access, and view descriptors to access ADABAS data and use it in the SAS System. Note: the ACCESS procedure is no longer available through a windowing interface.

160 pages.
