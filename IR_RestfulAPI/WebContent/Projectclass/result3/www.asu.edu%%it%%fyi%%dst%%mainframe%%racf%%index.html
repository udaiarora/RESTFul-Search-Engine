IT : Mainframe MVS/TSO —   RACF



RACF is a software security system which protects access to your account (interactive access and batch access), your disk data sets, and your tape volumes. You can control access to these by informing RACF who, if anyone, is allowed access and how much access they may have. People using TSO may use ISPF option 9.R (recommended) to perform these functions. Other methods include typing the RACF commands on a TSO command line or submitting the RACF command in a utility batch program.

  • Common RACF commands. (These commands are needed only if you do not use ISPF option 9.R.)
  1. ADDSD - Add a data set profile so you can change it's protection
  2. ALTDSD - Alter a data set profile; change data set protection
  3. ALTUSER - Alter information in user profile; change name or password
  4. DELDSD - Delete a data set profile; data set becomes fully protected
  5. LISTDSD - List existing data set profiles and protection settings
  6. LISTUSER - List attributes associated with your userid
  7. PASSWORD - Change password or password interval
  8. PERMIT - Add or remove data set access for particular userids
  9. RALTER - Add or remove tape volume access for particular userids
  10. RLIST - List existing tape volume profiles and protection settings
  11. SEARCH - List data set names which have been added to RACF
