Q. What devices can I use with our system?

A. The Treo 650 & 700 will work with our Exchange environment.  Each run's on a different platform; the 650 using the Palm OS, and the 700 running Windows Mobile 5.   The 650 will use a product called Microsoft Versa Mail installed on the device to synchronize with the Exchange server.  The Palm 700 will use Microsoft Active Sync and will be set up in the same fashion as the other Windows Mobile Devices.

*Note: Verizon customers will not receive a copy of VersaMail on your Treo's ROM, instead you will receive a copy of Verizon's Wireless Sync Client.  This will also work with Microsoft Exchange, but if you prefer to have the VersaMail installed, you must install it from the original CD-Rom that was provided in the original packaging.  Go to www.palm.com for instructions on loading this software.

Q. What features are available to sync with my device?

A. The following features are currently supported on your Treo 650:

  • E-Mail Synchronization with attachments
  • Calendar Synchronization
  • Accept, decline and tentative responses to meeting invitations
  • Secure: Default synchronization; Optional proxy server such as MS Internet Security (ISA); Support for various VPN clients.

Some of the Exchange Features NOT Available with your Treo 650 include:

  • Wireless Reconciliation of Contacts, Tasks, and Notes.
  • E-Mail Subfolder synchronization
  • Global Address List access
  • Creating meeting invitations

Your Treo 700 Device will have all the same functions as any Windows Mobile Device, such as:

  • Wireless Reconciliation of Contacts, Mail, Tasks, Calendar items.
  • E-Mail subfolder synchronization
  • Global Address List access.
  • Creating meeting invitations
  • Accept, decline and tentative responses to meeting requests.
  • Support for Various VPN clients.

Q. How do I get my Treo 650 setup on ASU's E-Mail system?

A. Follow the instructions below to set up your Treo 650 for wireless synchronization of your e-mail and calendar.

  1. From the installation software CD, a window will appear that shows Discover Your Device.  Choose to set up a new account at that time.
  2. Create a new VersaMail E-Mail Account and click Next.
  3. Select the options that allows you to choose your mail service from the list, and choose Exchange Active Sync from that list and click Next.
  4. Enter your account information which includes the following and click Next.
    Descriptive Name of the Account: "choose this name yourself"
    Your Account Username: "your ASURITE User ID"
    Your Account Password: "the password to your ASURITE Account"
    E-Mail Address for this Account: "Your E-Mail@asu.edu"
  5. Enter the Server information:  exchange.asu.edu; the Port Number (leave default); and the maximum size message to download (default) and click Next
  6. Leave the Proxy Server window on it's default settings and click Next.
  7. Test the account settings by clicking Test My Settings, click okay after they have been tested and click Finish.
  8. If you Account did not test okay, be sure to check all the information entered.  If you have any further issues, contact the ASU Team at email-q@asu.edu .

To set up the sync on your device without using the desktop software.  Note that VersaMail is not installed by default on Verizon phones.  You must run the installation CD to install the VersaMail Client to the device.

  1. Go to Applications and Select VersaMail
  2. Open the Account Setup screen
  3. Open the menus
  4. Select Account
  5. Select Account Setup
  6. Select New
  7. In the Account Name Field, enter any descriptive name
  8. Select the Mail Service pick list and choose Exchange ActiveSync
  9. Select Next
  10. Enter your ASURITE User ID for your username, and enter your ASURITE password for the password.
