FULLSTIMER

FULLSTIMER


Writes all system performance statistics to the SAS log

Default: NOFULLSTIMER
Valid in: configuration file, SAS invocation, OPTIONS statement, OPTIONS window
CMS specifics: all

Syntax
Details
See Also

Syntax

FULLSTIMER | NOFULLSTIMER

FULLSTIMER
writes all system performance statistics to the SAS log.

NOFULLSTIMER
tells SAS not to write system performance statistics to the SAS log.

Details

Under CMS, the FULLSTIMER system option specifies whether extended performance statistics of your computer system are to be collected and displayed after each step. They include the following:

  • real time

  • user CPU time

  • system CPU time

  • memory task switches.

Note:   This option should be specified for performance analysis only. Specifying the FULLSTIMER option for all programs will affect performance.
