Communications Access Methods for SAS/CONNECT and SAS/SHARE Software

OS/2: EHLLAPI Access Method

SAS/CONNECT

Local Host

The following example illustrates the statements that you specify in an OS/2 local host SAS session to connect to a remote host with the EHLLAPI access method. 

filename rlink '!sasroot\connect\saslink\logtso.scr';
options comamid=ehllapi remote=a;
signon;

The first line identifies the script file that you use to sign on to the remote host. The script file contains a prompt for a userid and a password that are valid on the remote host. The EHLLAPI communications access method is declared with a connection to the remote host A, which is the remote session identifier that is specified when configuring the emulation package on your local host. The SIGNON statement performs the sign-on process.

Remote Host

SAS Institute does not provide support for connections to the OS/2 remote host with the EHLLAPI access method.
