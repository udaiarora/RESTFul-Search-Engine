S2KMODE=

S2KMODE=


Optional statement
Applies to: New database

Syntax
Details

Syntax

S2KMODE|S2KMD= M|S;

Details

The S2KMODE= statement specifies your mode of accessing SYSTEM 2000 software. M means the database files are in the SYSTEM 2000 Multi-User environment. S means the database is in a single-user environment, that is, a database in your SAS program environment. S is the default.

S2KMD= is an alias for the S2KMODE= statement.

S2KMODE= is also a data set option for input data views for SAS procedures. However, you cannot use it as a data set option for the DBLOAD procedure. For more details about the S2KMODE= data set option, see Data Set Options.
