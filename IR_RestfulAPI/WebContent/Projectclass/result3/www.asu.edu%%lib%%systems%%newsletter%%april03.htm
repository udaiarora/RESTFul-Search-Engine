LIST's Newsletter
April, 2003

The latest release of III includes some new features you might want to know about, described below.


Changes in the Online Library Catalog

With the upgrade to INNOPAC Release 2002, phase 3 that was completed last week, there have been a few enhancements to the WebOPAC screens. Some of the most noticeable ones are:

Next Page, Previous Page
Next Page and Previous Page buttons have been replaced with a hyperlinked pagination scheme similar to Web search engine results navigation:

Marking and Saving Records
The Save All On Page button will mark and save all entries on the results page. The Save Marked Records Button and the Save All On Page Button will be on the top and bottom of the page.

Course Reserves
In Course Reserve, patrons can now see the author of the reserved books and articles, as well as sort the lists by title, author and call number, by clicking the buttons at the top of the columns. The format column is not a field we are using at this time.

Please contact Fred McIlvain at 5-9427 if you have questions about the Online Catalog.


New Database Available

A new database has been put online: http://www.asu.edu/lib/top-menus/new.html.

Photographic Libraries
Photographic Libraries is a Web-based visual communications directory that includes links to Stock Libraries; Archive Collections; Moving Images; Photo Agencies; Fashion Photographers; Stars and Celebrities; Clip Art-Free Photos; Digital Photography; Fine Art; Ad Agencies- Resources; Photojournalist; Library-Museums; Maps-Cartography; Graphic Visuals; Student Galleries; Generation Next; Exclusive Benefits.
