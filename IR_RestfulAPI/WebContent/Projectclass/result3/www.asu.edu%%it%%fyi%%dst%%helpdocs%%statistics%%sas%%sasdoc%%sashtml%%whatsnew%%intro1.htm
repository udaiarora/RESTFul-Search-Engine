Essential Information about This Document



What This Document Includes

This document describes changes and enhancements that have been made to SAS software products since Release 6.12--or, for the mainframe platforms (OS/390, CMS, and OpenVMS VAX), since Release 6.09E. If your site is upgrading directly from a Version 6 release to Version 8, then all of the "What's New" information in this document will be of interest to you.

If you are upgrading from Version 7 to Version 8, then look for the icon. This icon precedes information about features that were added or enhanced between Versions 7 and 8. For example, in the ODS and Web Enablement section of the SAS/OR chapter, the last paragraph is new for Version 8. Similarly, in the SAS/IntrNet Components section of the SAS/IntrNet Software chapter, you will see several items that are new or enhanced in Version 8.

Detailed information about the new or enhanced features can be found either in the product documentation elsewhere on this SAS OnlineDoc CD-ROM, or in the SAS Help System.
