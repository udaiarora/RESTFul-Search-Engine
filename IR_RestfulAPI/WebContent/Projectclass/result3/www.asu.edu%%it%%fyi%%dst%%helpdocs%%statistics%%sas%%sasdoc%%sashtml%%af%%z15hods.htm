Form Data Model Methods

Methods specific to the Form Data Model class are described here. Inherited methods are described in the Object class.

Because the Form Data Model class is an abstract class, most of the Form Data Model methods are abstract methods; that is, the methods are defined but disabled at the Form Data Model level. These methods should be overridden by the subclasses of Form Data Model in order to make a useful model and provide model-specific functionality.
