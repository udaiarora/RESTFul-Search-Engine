Frame Class: _getDragReps

Frame Class: _getDragReps


Returns the drag representations currently defined for this object

Syntax
Details

Syntax

objectName._getDragReps( representationsList );

Argument Type Use Description
representationsList Numeric Update specifies the identifier of the SCL list that contains the drag representations operations

Details

Note that representationsList can be modified and sent back to the _setDragRep method using callApply to perform global changes in the drag representations.
