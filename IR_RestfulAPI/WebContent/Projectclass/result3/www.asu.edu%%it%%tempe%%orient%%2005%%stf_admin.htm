ASU Administrative Systems

ASU has several different computing systems for use for administrative functions. Several of these require authorization for access. Your department will inform you of the various systems you will need to access. See the Admin Computing Web page at asu.edu/admincomputing for a full listing of computing systems. If you have problems with getting access to these systems, contact the IT Help Desk at 480.965.6500.
