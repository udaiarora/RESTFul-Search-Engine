Setting up Outlook to access an IMAP account

Please follow the instructions below to set up the Outlook client to access your IMAP account.

 

To set up your Outlook client to access to the Arizona State University IMAP Server, click the Start button, choose Settings, and then the Control Panel. From the Control Panel, select the Mail icon, as illustrated below in Figure 1. Please note that your Control Panel may either be displayed as a sub-menu of the Settings menu, or as its own window depending on your operating system and view settings.


Figure 1: Mail Icon Selection

 

After opening the Mail icon, you will see the Mail Setup window as illustrated below in Figure 2. Click the button that says E-Mail Accounts.



Figure 2: Mail Setup

After you click the E-mail Accounts button, the E-Mail Accounts window will open, as shown in Figure 3. Activate the radio button next to “add a new e-mail account” and click “Next”.


Figure 3: E-Mail Accounts Window

Next you must choose the type of server your new E-Mail account will work with, as shown in Figure 4.


Figure 4: Server Selection

Activate the radio button next to “IMAP” and click “Next”. You will then be taken to a screen that looks like Figure 5.

Figure 5: IMAP Server Settings


You must now enter the IMAP Server settings: enter imap1.asu.edu or imap2.asu.edu depending upon which server you were assigned at creation exactly as shown above in Figure 5. For User Name, input your own ASURITE UserID. For Password, input your ASURITE password. Leave Log on using Secure Password Authentication (SPA) unchecked. Your outgoing e-mail server should be set to smtp.asu.edu if and only if you are on an ASU IP address (on campus, PPP dial-in, or VPN). If neither of those situations apply to you, you will need to use the outgoing e-mail server provided by your internet service provider. For example, if you are on a Cox IP address in Arizona, you will need to use smtp.west.cox.net.


Then click Next.

Click Finish and the control panel portion of the setup is completed.


If you are still having trouble completing this portion of the setup, please submit a question online at the Ask IT Web page, or call the IT Help Desk at (480) 965-6500 for further help.


Next, launch Outlook by double-clicking icon on the desktop. You may then be prompted for a User Name (ASURITE UserID) and Passphrase, as shown below in Figure 6. Please note that the server you are connecting to may be slightly different from the one pictured.


Figure 6: Connection Dialogue

You should now be able to access your Arizona State University Microsoft IMAP e-mail account through Microsoft Office Outlook 2003.
