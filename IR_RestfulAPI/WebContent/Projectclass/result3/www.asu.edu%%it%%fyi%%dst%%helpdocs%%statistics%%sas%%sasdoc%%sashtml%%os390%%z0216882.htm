FSDEVICE=


Specifies the terminal device driver

Default: none
Alias: FSD=
Valid in: configuration file, SAS invocation
Catergory: Environment Control: ENVDISPLAY
OS/390 specifics: device-name

Syntax
Details
See Also

Syntax

FSDEVICE=device-name

Details

The value of the FSDEVICE= system option is needed to run windowing procedures. See Terminal Support in the OS/390 Environment for a list of all devices that are supported by the SAS terminal-based interactive windowing system under OS/390.
