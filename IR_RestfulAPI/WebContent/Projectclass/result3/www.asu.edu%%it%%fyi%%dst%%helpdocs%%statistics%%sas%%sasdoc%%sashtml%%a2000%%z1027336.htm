SCS

SCS


Optional statement

Syntax
Details

Syntax

SCS;

Details

The SCS statement puts the QUEST procedure into single-statement queuing mode (single command submission).

Note:   SCS erases the 32760-byte buffer (used with MCS mode) even if it contains statements not yet sent to SYSTEM 2000 software.
