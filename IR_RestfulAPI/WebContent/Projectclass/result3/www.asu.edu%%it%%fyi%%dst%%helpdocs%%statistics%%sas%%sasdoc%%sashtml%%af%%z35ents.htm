Format/Informat Class Events

Events specified for the Format/Informat class are described here. See Object Class Events for inherited events.

formatCode changed
Occurs when the formatCode attribute is changed

formatDecimal changed
Occurs when the formatDecimal attribute is changed

formatName changed
Occurs when the formatName attribute is changed

formatWidth changed
Occurs when the formatWidth attribute is changed

informatCode changed
Occurs when the informatCode attribute is changed

informatDecimal changed
Occurs when the informatDecimal attribute is changed

informatName changed
Occurs when the informatName attribute is changed

informatWidth changed
Occurs when the informatWidth attribute is changed
