DEFWORKUNIT

DEFWORKUNIT


Specifies whether or not to use a unique CMS work unit for each output file in the Shared File System

Default: NODEFWORKUNIT
Valid in: configuration file, SAS invocation
CMS specifics: all

Syntax

Syntax

DEFWORKUNIT | NODEFWORKUNIT

DEFWORKUNIT
specifies that a single work unit is to be used for all files in the Shared File System. Fewer IUCV connections to the SFS server are used.

NODEFWORKUNIT
specifies that a unique CMS work unit is to be used for each output file in the Shared File System. This allows the SAS session to have better control over each SFS file opened for output.
