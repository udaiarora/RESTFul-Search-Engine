Data Set Data Vector: _getLabel

Data Set Data Vector: _getLabel


Returns the current label for a column in the data vector

Syntax

Syntax

CALL SEND (data-vector-id, '_getLabel', col-num, label);

Argument Type Description
col-num
 N specifies the column number
label
 C returns the label for the column
