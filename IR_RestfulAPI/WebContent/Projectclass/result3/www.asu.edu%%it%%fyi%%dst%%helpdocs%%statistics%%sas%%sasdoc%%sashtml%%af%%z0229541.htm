Image: _print


Prints the current image to a file or to a hardcopy device using the default system printer

Syntax
Details
See Also

Syntax

CALL NOTIFY (image-name, '_print');

Details

On UNIX systems, the _print method works only if host printing is enabled. To verify or change the setting, from a SAS windowing environment window select

Tools
Options
Preferences
and the DMS tab; then turn on Use Host Printing. Once host printing is enabled, the _print method sends the image to the file or device selected in the Printer Setup dialog.

For more information on using the Printer Setup dialog to specify a destination for the image, see the help topics for using SAS with your operating system.

On PC systems, host printing is enabled by default and can be turned off only by selecting Use forms from the Print Setup dialog under the File menu.
