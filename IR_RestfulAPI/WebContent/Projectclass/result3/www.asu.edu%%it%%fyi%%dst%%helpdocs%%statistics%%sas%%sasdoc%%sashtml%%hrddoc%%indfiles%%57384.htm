SAS/OR Software: The netWorks Application, Version 8

Learn how you can use the SAS/OR software netWorks Application to design a new communications network, document an existing network, or investigate network capacity, reliability, and performance issues. This report details a network resource planning (NRP) tool designed to help you manage and exploit your network investments. The feature descriptions and examples are written for users who are familiar with telecommunication and local area network (LAN) technologies.
