Professional SAS Programming Secrets: Updated with New Features of Releases 6.08-6.10

by Rick Aster and Rhena Seidman

Regardless of your level of skill or experience, you'll be operating like a pro after reading this highly acclaimed guide to SAS application development. Packed with ingenious tips, techniques, and strategies, Professional SAS Programming Secrets provides you with a collection of time-tested SAS programming methods, as well as hundreds of ready-to-use code routines. This book offers essential coverage of: observation loops; programming for efficiency; memory and management; word wrap; undocumented SAS features; report writing techniques; structures, unions, and bitfields; binary file programming; data security and encryption; and macro programming. Source code for examples is included on a handy, ready-to-use disk.

588 pages.
