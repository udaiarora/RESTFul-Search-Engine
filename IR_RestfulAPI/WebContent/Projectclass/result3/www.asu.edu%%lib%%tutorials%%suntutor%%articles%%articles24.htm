Understanding your results

Getting your articles: In the library

The journal record will look like this:

  • Holdings information: tells you what years and volumes the library owns
  • Location: tells you which library has the journal
  • Call number: gives you the number you need to locate the journal on the shelf

All of this information is important to locate the journal in the library. Do not leave the online catalog without collecting all of this information first.

To find the journal on the shelf, go to the library identified in the "Location" section. Ask a librarian in that library for the location of the journals. If the article was published within the last year, start with the current periodicals area; if it is older you will want to look in the bound journals area. Once you have found the journals in the library, use the call number to find the journal you need.

Some journals also are available electronically on the web. If a journal is available in this manner, the catalog record will include a link to the web site. See example below.

If you would like to view journal articles electronically over the web from off campus, please see Accessing ASU Indexes and Electronic Journals From Off Campus.
