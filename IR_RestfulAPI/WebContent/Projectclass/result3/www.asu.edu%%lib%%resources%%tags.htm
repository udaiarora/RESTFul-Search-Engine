Explanation of Restriction Tags
Used on the ASU Libraries' Web Pages


Due to technical and contractual constraints, many of the computerized databases purchased or leased by the ASU Libraries have restrictions on access. These restrictions range from limiting access to ASU faculty, staff and students to no access outside the libraries. In some cases, special software is needed. The following tags are used on the ASU Libraries' web pages to indicate restrictions on who may access databases and how to access them.


Tempe campus only


The "Tempe Campus only" tag indicates that access is restricted to computers physically located on the Tempe campus. There is no access from the West or Polytechnic campuses nor access from off-campus via PPP.


The general public may access "Tempe campus only" databases from within the Architecture, Hayden, Law, Music, and Noble Libraries. Some of these databases may have been purchased separately by the libraries on the West or Polytechnic campuses; please call the reference desks at the West and Polytechnic campus libraries for availability.


Use the browser's "BACK BUTTON" to return to the previous page.


ASU only or


The "ASU only" tag indicates that the database is restricted to ASU faculty, staff and students. The restrictions may be enforced through either of the following methods:

  1. Access only from ASU computers; no access from non-ASU Internet Service Providers such as AOL;
  2. Require the use of an ASU PPP account or an ASURITE account.


For more information on accessing these resources from outside of the library, consult our guide on Remote Access to the ASU Libraries' Network. The general public may access "ASU only" databases from within the Architecture, Hayden, Law, Music, and Noble Libraries. Some of these databases may be also be accessible from the libraries on theWest or Polytechnic campuses; please call the reference desks at the West and Polytechnic campus libraries for availability.


Use the browser's "BACK BUTTON" to return to the previous page.


Library only or


The "Library only" tag indicates that the database is restricted to use within the Architecture, Hayden, Law, Music and Noble Libraries. Some databases may be restricted to only one library.


Use the browser's "BACK BUTTON" to return to the previous page.


PDF File or


The "PDF File" tag indicates you are about to view a PDF formatted file. This type of file must be viewed using the Adobe(R) Acrobat(R) Reader. The plug-in is pre-loaded on the library workstations. For home or office use you may obtain a free copy of the plug-in for download from the Adobe(R) web site: http://www.Adobe.com/prodindex/acrobat/readstep.html


Use the browser's "BACK BUTTON" to return to the previous page.


Web


The Web tag indicates that the database is accessible on the World Wide Web. Please note that many web accessible items have ASU only restrictions. Some Web databases, especially electronic journals and books, require a graphics browser as well as additional software such as Adobe Acrobat Reader to read and/or print from the database. Additional requirements are noted along with the Web tag.


Use the browser's "BACK BUTTON" to return to the previous page.
