SAS Reference Documentation
(aka. SAS OnlineDoc)

The SAS reference documentation is now available online. The current version is 8 and includes description of SAS language elements (functions, statements, procedures, options, and so on), product interfaces, and displays, as well as information about concepts. It also includes documentation for all SAS products.

Please note that item #1 and item #2 are the same except one has Search capabilities. If one becomes temporarily unavailable then please use the other one.

  1. Documentation (aka. SAS OnlineDoc Version 9.1 via SAS web site)
    (Has search capabilities)

  2. Reference Documentation (aka. SAS OnlineDoc Version 8 via SAS web site)
    (Has search capabilities and requires registration)

  3. Reference Documentation (aka. SAS OnlineDoc Version 8 via ASU web site)
    (Search capabilities not available but has a master index. Limited to ASU IP addresses)

  4. What's New in SAS Software for Version 8.1 and 8.2
    (Detailed changes and enhancement for most products are in the SAS Help System i.e. main menu Help -> SAS System Help -> under Contents tab, Welcome to the SAS System -> expand and it shows What's New in Release 8.1 and What's New in Release 8.2)

  5. How to Create Painless Graphics in a Minute

  6. New features in SAS/GRAPH software with Version 8

  7. Statistics-related Technical Reports

  8. SAS Software Tutorials for Version 8

  9. Compatibility of Version 8 with Earlier Releases

  10. Version 6 and Version 8: A Peaceful Co-Existence

  11. In-depth summary of Version 8 (University of Leeds)

  12. The Complete Guide to the SAS Output Delivery System, Version 8

  13. Everything about Version 7 (almost!)

  14. ODS XML Primer
