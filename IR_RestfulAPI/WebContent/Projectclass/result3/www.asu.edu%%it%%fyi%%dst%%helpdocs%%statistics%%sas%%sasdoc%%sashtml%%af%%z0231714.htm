SAS/AF Software: Class Dictionary

Overview

Use the Extended Input Field class when you need additional functionality not provided in the Input Field class. This class supports multiple lines, and you can specify different fonts and colors.

An extended input field can contain a maximum of 200 characters because the contents of an extended input field are stored in the object's SCL variable. This limit also applies to text entry fields. If you need an input field that is more than 200 characters, use the Extended Text Entry class.

Parent:
sashelp.afclass.input.class

Class:
sashelp.afclass.ExtInput.class
