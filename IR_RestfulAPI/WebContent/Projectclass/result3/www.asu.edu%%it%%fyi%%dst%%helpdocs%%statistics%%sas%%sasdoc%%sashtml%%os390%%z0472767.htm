FILEEXIST


Verifies the existence of an external file by its physical name and returns a value

OS/390 specifics: file-name

Syntax
See Also

Syntax

FILEEXIST(filename)

filename
is a fully qualified operating environment data set name or a fully qualified path (for UNIX System Services files).
