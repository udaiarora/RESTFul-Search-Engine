Getting a Network Card

To utilize the wireless network an IEEE 802.11b or IEEE 802.11g compatible wireless network card is required. For a review of various network cards or for recommendations on selecting a card, please visit our Network Card Review page.

After obtaining an IEEE 802.11b compatible network card for your computer, follow the steps below to get connected to ASU's wireless network:

Installing the Network Card

Install the network card into your computer. The details for doing this will vary depending on the particular card you are using. Installation instructions will come with your network card to aid you in this process.

Configuring the Network Card

To configure the network card to communicate with the ASU wireless network the default settings will usually suffice. The card should be configured as follows:

  • The SSID should be set to asu, in lowercase.
  • The network mode must be infrastructure, not adhoc. Some cards will refer to this as 'Access Point' or 'computer to base station' as opposed to infrastructure.
  • WEP must be disabled. Also, if available, the authentication method must be set to Open System and not Shared Key

Accessing the Network


Existing ASURITE Login

The final step in using the wireless network is to switch your computer over to DHCP and register your network card with the DHCP server. The ASU campus wide DHCP service provides the ability to have your computer's IP information automatically set when your computer starts up.

To register your network card with the DHCP server go to https://author1.asu.edu/dhcp using your wireless network connection.

New Guest/ASURITE Login
(Only available in the MU, Hayden Library, BA/BAC, ADMIN A/B)

A new logon process for connecting to ASU's computing network was implemented in selected areas of Tempe Campus. The computer owner now just needed to open a Web browser and complete an automatic safety scan before being connected to the network. A MAC address registration will not be needed for connection. Guest users will have access limited to selected VPN service, and HTTP and HTTPS Web browsing.

If you are an ASU faculty, staff or student enter your ASURITE UserID and passphrase and click "Continue."

If you are a visitor at ASU, please click on "ASU Guest Access".





Troubleshooting

If you have any problems connecting to the wireless network at ASU, contact the Help Desk.
