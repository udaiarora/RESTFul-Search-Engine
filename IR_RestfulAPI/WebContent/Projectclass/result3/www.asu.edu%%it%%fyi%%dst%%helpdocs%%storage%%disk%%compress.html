Compressing and Uncompressing Files

To compress a file, the UNIX command "compress" can be used. For example, to compress the raw data file "sipp93t3.xxx" the command used is:

compress sipp93t3.xxx

This creates a compressed file with the name "sipp93t3.xxx.Z". The compression can be quite substantial. For example in an actual case, the uncompressed file "sipp93t3.xxx" was approximately 68.5 MB, whereas the compressed file "sipp93t3.xxx.Z" was only 2.6 MB.

uncompress sipp93t3.xxx.Z

For more information on the compress and uncompress commands, please enter "man compress" at the system prompt.

Note: Some UNIX applications such as SAS and SPSS can read compressed data files. Hence in such cases, there is no need to uncompress.
