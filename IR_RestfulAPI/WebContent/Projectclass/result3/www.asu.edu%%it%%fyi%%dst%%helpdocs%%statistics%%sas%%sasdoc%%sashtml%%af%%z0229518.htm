Image Icon: _getLabel

Image Icon: _getLabel


Returns a label for an image icon

Syntax

Syntax

CALL NOTIFY (icon-name, '_getLabel', line1, <line2>);

Argument Type Description
line1
 C the first line of text for the label
line2
 C the second line of text for the label
