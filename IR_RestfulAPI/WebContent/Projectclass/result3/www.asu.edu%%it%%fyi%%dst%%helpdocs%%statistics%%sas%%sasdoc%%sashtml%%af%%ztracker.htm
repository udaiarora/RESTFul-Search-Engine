Frame Class: _cursorTracker

Frame Class: _cursorTracker


Invoked when cursor tracking is enabled and the mouse pointer is moved across the frame

Syntax

Syntax

objectName._cursorTracker( x, y );

Argument Type Use Description
x Numeric Output returns the X (horizontal) coordinate location of the mouse pointer
y Numeric Output returns the Y (vertical) coordinate location of the mouse pointer
