How to delete print jobs

The command:

npdel lpq_4711
Deletes the print job with the ID "lpq_4711".

The command:

npdel -S "owner=agjjc"
Deletes all print jobs belonging to userid "agjjc".

The command:

npdel -S "(owner=agjjc) and (queue=Qcc102pr1)"
Deletes all print jobs belonging to userid "agjjc" in the queue "Qcc102pr1".

For more information, enter "man npdel" at the system logon prompt.

One of the times you should delete a print job is when you check the status of the print job and find that its status is listed as "suspend". It is necessary to delete this request because, without the system administrator's intervention, neither this nor subsequent print requests from you and from other users will print until the top-most (suspended) request is deleted.
