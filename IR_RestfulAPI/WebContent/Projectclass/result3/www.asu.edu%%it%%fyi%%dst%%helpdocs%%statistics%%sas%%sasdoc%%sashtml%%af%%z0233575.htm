SAS/AF Software: Class Dictionary

Overview

The Icon class enables you to use icons, which are pictorial representations of tasks or actions, to replace or supplement text in your SAS/AF applications. SAS/AF software supplies a library of icons that you can incorporate in your FRAME entries to represent things such as push buttons, menu choices, or windows. In addition to providing SAS supplied icons, some hosts provide interfaces that support using user-defined icons.

Parent:
sashelp.fsp.widget.class

Class:
