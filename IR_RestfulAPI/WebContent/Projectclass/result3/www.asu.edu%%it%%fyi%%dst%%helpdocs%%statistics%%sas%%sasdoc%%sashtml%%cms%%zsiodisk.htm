SIODISK=

SIODISK=


Forces all WORK files from a SAS program or session to be written to the specified disk

Default: none
Valid in: configuration file, SAS invocation
CMS specifics: all

Syntax

Syntax

SIODISK=disk-specification

disk-specification
is the CMS filemode letter or an SFS directory name.
