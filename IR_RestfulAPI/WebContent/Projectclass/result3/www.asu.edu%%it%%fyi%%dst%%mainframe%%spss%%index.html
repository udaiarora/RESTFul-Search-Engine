IT : Mainframe MVS/TSO —   SPSS



SPSS RELEASE 4.1 is the production version. To access this version, use the following Job Control Language:

     // JOB
     // EXEC SPSS
        ...
     SPSS COMMANDS
        ...
     //

Further help may be obtained with the command "HELP SPSS" and by including the command INFO OVERVIEW or the command INFO LOCAL in the list of SPSS commands in the job stream above.

The "SPSS - Using SPSS via TSO" write-up is available in rich text format (rtf). If your browser cannot display the document, you can save it on your workstation's disk and then view it with a word processor, such as Word or Word Perfect.

A list of known errors in SPSS is also available.

SPSS, Inc has reorganized their documentation and produced a new manual set in conjunction with Release 4. These manuals include:

SPSS Base System User's Guide
SPSS Reference Guide
SPSS LISREL 7 and PRELIS User's Guide and Reference

These manuals are currently available in the Customer Assistance Center and in the ASU BOOKSTORE.

VERSION 3.0 OF SPSS-X IS NO LONGER AVAILABLE.
