Form Data Model: _getEntry


Returns the name of the DATAFORM catalog entry associated with the model

Syntax
Details

Syntax

CALL SEND (model-id, '_getEntry', entry);

Argument Type Description
entry
 C returns the name of the DATAFORM catalog entry associated with the model

Details

The DATAFORM catalog entry name can be a two-level or four-level name. The _getEntry method returns a blank if no DATAFORM catalog entry name is specified.
