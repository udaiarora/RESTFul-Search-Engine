VSAMLOAD

VSAMLOAD


Enables or disables the loading of records into an empty VSAM data set

Default: NOVSAMLOAD
Valid in: configuration file, SAS invocation, OPTIONS statement, OPTIONS window
CMS specifics: all

Syntax

Syntax

VSAMLOAD | NOVSAMLOAD

VSAMLOAD
enables load access to VSAM data sets.

NOVSAMLOAD
disables load access to VSAM data sets.
