Graph Output Control: _setMagnification

Graph Output Control: _setMagnification


Sets the magnification factor with which the graphics output is drawn

Syntax

Syntax

objectName._setMagnification( magnifyPercentage );

Argument Type Use Description
magnifyPercentage Numeric Input specifies the percentage by which to magnify the graph. The default is 100 percent.
