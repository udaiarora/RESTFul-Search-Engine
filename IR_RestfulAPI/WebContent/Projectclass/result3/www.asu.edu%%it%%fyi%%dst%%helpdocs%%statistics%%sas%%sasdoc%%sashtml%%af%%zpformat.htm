Format/Informat Class: _setupFormat


Sets the specified format

Syntax

Syntax

return = objectName._setupFormat( format );

Argument Type Use Description
return Numeric Return returns the format code fo the the specified format. 0 if not successful
format Character Input A format that is supplied by SAS or proc FORMAT
