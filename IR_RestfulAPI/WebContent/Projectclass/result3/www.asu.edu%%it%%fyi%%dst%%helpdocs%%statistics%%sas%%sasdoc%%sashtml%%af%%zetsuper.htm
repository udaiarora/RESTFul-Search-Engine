Class Class: _getSuper


Returns the identifier of the super class

Syntax
Details

Syntax

objectName._getSuper( superClassID );

Argument Type Use Description
superClassID Numeric Update returns the object identifier of the super class

Details

If superClassID is the Object class SASHELP.FSP.OBJECT, the _getSuper method returns 0.
