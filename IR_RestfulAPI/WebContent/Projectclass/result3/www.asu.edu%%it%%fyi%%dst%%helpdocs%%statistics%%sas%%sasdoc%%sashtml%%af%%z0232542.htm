GET_DATA Data Vector: _getWrapping

GET_DATA Data Vector: _getWrapping


Gets the text-wrapping information

Syntax
Details

Syntax

CALL SEND (data-vector-id, '_getWrapping', wrapping<, split-char>);

Argument Type Description
wrapping
 C returns the wrapping status


'Y' the text will be wrapped


'N' the text will not be wrapped
split-char
 C character on which to split lines

Details

The _getWrapping method is valid only if the viewer is a table editor. If wrapping is set to 'Y', the character set by split-char wraps the text over multiple lines. For example, if split-char is set to the backslash (\) character, the text One\Two\Threewill be split over 3 lines.
