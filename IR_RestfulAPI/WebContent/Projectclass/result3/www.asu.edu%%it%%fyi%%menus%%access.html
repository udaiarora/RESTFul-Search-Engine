Accessibility and Browser Obsolescence

The FYI Web pages, designed and maintained by ASU Information Technology's Information Dissemination Team, are designed to be accessible to all users, regardless of disability. As such, they conform at level Triple-A to the Web Content Accessibility Guidelines 1.0 (WCAG 1.0) Recommendation of the World Wide Web Consortium (W3C) Web Accessibility Initiative (WAI).

Because many obsolete or obsolescent browsers fail to conform to W3C standards, conforming pages may be poorly rendered in those browsers. Specifically, poorly implemented browser support for the Cascading Style Sheet, Level One (CSS1), standard can, in extreme circumstances, render a conforming page unreadable. If display of FYI pages appears "jumbled," disable style sheet use in your browser. For example, in Netscape Communicator 4, go to Edit | Preferences | Advanced and deselect Enable style sheets.

For best results, the ID Team recommends use of a standards-compliant browser to view the FYI pages. Use of versions of Nescape earlier than 6.0 and versions of Microsoft Internet Explorer earlier than 5.0 is discouraged.

If you have difficulty accessing the FYI pages using a standards-compliant user agent (browser) or adaptive technologies, please do not hesitate to contact the ID Team by e-mail.
