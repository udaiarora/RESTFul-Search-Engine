Communications Access Methods for SAS/CONNECT and SAS/SHARE Software

SAS Support for VTAM LU 0 on OS/390

Note:    The VTAM LU 0 access method applies only to SAS/SHARE. It was implemented in Version 6 for backward compatibility with previous releases of SAS/SHARE. However, beginning with Version 7, VTAM LU 0 is not supported. It is recommended that you use the APPC (LU6.2) access method instead of VTAM LU 0. See OS/390: APPC Access Method.
