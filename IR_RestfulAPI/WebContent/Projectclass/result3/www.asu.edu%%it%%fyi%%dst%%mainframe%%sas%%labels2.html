PROC FORMS without Sorting

Modified Format Without Sorting

In this job, two sets of 3-line labels are printed. (Note the use the SETS option on the PROC FORMS statement.) The value for ADDRESS1 is assigned in the DATA step via SAS IF-THEN and ELSE statements, and the value for ADDRESS3 is given in a SAS assignment statement. These statements minimize coding redundant addresses in the data lines. Note that values of the variables COLLEGE and DEPT are not printed in the labels. COLLEGE is used in a "subsetting IF statement" in the DATA step to select the observations for which labels are to be printed. DEPT is used to select values to assign for the first line of the address (variable ADDRESS1). 

 

//  JOB
/*JOBPARM LINECT=0
//  EXEC  SAS
//LABELS  DD  SYSOUT=M,DEST=LOCAL,FCB=M
DATA LBLDATA;
  INPUT COLLEGE 1 DEPT $ 2-3 NAME $ 4-28
    ADDRESS2 $ 29-48 ;
  IF COLLEGE=3;
  IF DEPT=' ' THEN ADDRESS1='COLL OF';
              ELSE ADDRESS1='DEPT OF';
  ADDRESS3='CAMPUS';
  CARDS;
3  LIBRARY,BIMSON           BUSINESS ADM
1MAANDERSON,BRUCE           MATHEMATICS
3FIAPILADO,VINCENT          FINANCE
4EEBASS,ROBERT              ELECTRICAL ENG
4IEBEDWORTH,DAVID           INDUSTRIAL ENG
1SOBINDICK,MARILYN          SOCIOLOGY
2EEBITTER,GARY              ELEMENTARY EDUC
4AIBRADY,WARD               AGRICULTURE
2SEBROOK,WESTON             PROF FIELD EXP
0SCBULEY,JERRY              SPEECH AND THEATRE
3ECBURGESS,PAUL             ECONOMICS
1MACANARY,JIM               MATHEMATICS
;
PROC FORMS DATA=LBLDATA DDNAME=LABELS WIDTH=28
     LINES=3 SKIP=5 DOWN=5 INDENT=1 SETS=2 ALIGN=24;
  LINE 1 NAME/LASTNAME;
  LINE 2 ADDRESS1 ADDRESS2/PACK;
  LINE 3 ADDRESS3;
//
