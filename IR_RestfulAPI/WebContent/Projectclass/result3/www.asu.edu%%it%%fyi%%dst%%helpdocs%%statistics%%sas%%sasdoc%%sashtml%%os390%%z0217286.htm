Concatenating External Files

Multiple sequential data sets can be concatenated via JCL DD statements, a TSO ALLOCATE command, a FILENAME statement, or a FILENAME function. (When accessing concatenated files, performance is better when either of the first two methods is used.) See the examples in Using the FILENAME Statement or Function to Allocate External Files, Using the JCL DD Statement to Allocate External Files, and Using the TSO Allocate Command to Allocate External Files. Also see Reading Concatenated Data Sets.
