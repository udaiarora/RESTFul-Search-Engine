Z Variable

Z Variable


Identifies the z coordinate of where a graphics element is to be drawn.

Type: numeric
Length: 8
Default: none
Restrictions: Used only with output from the G3D procedure.

Syntax
Functions
Details

Syntax

Z=depth-coordinate;

Functions

You can use the Z variable with these functions:
BAR PIE POLY
DRAW PIECNTR POLYCONT
LABEL POINT SYMBOL
MOVE

Details

The Z variable uses the units that are specified in the ZSYS variable.
