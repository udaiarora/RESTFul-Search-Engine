Webapp Development Resources - Getting Started

So you want to write a Webapp? Well, you're in the right place... Since the earliest inception of the World Wide Web, ASU has looked to Web applications to extend (and sometimes push the boundaries of) the functionality of the ASU Web. In recent years, the pace of new projects has picked up considerably in the Webapp arena with ASU now providing online class registration and fee-payment services.

Who can write Webapps at ASU?

At this time, Web application development is limited to projects specifically focused on providing enterprise-level service to the Web community of ASU. At some point in the future, we all agree that it would be nice if students, faculty, and staff would be able to write their own personal Web applications and have them be hosted in their personal Webspace, but unfortunately, at the current time, the support cost would be too great.

With that said, if you are interested in learning more about enterprise Webapp development at ASU, please continue exploring this site by navigating to our Documentation page.
