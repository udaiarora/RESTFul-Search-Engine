Compiler Options

Compiler options allow the normal mode of compiler operation to be altered temporarily. Some options affect output produced by the compiler, others affect the internal workings of the compiler itself. Generally suitable default values have been chosen for each option and are built into the compiler.

Specification of compiler options can be done via the COPTS symbolic parameter in conjunction with any of the COB2 cataloged procedures containing a C (for compile) in the name. For example,

     //  EXEC  COB2CG,COPTS='LIST,NOFLAG'

requests that the two compiler options LIST and NOFLAG be used.
