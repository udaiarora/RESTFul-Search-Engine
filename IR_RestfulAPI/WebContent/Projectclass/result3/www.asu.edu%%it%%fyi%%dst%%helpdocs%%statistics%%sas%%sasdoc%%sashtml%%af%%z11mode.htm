Histogram Control: sortMode

Histogram Control: sortMode


determines the sorting order (formatted or unformatted) for the category variable

Type: Character
Valid Values: None FormattedAscending UnformattedAscending

Details

Details

Valid values are:
None Do not sort values
FormattedAscending If the variable has a format associated with it, convert the variable value to this format and then sort the values in ascending order.
UnformattedAscending If the variable has a format associated with it, sort the variable value and then convert the value using the format.
