Future Development

Information Technology is currently working to implement an enterprise-wide authorization management system called EDNA (Enterprise Dynamic Network Authorization). This system will also allow departments to manage authorization of access to their web applications. Logically, the University web authentication system (WebAuth) and the authorization system (EDNA) are very tightly coupled. A future version of WebAuth will be integrated with EDNA, using XML as the method of client-server communication.
