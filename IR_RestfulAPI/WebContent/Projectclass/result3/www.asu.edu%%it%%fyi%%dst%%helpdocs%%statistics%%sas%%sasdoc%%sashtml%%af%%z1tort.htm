Graph Output Control: _setContort

Graph Output Control: _setContort


Turns on or off automatic resizing of the graphics output to fit the containing region

Syntax

Syntax

objectName._setContort( resize );

Argument Type Use Description
resize Character Input specifies whether to turn on or off automatic resizing: Y turns it on, N turns it off
