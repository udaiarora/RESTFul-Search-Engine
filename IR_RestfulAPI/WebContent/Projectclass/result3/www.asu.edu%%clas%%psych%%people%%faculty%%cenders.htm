downloads:

Fitting Multilevel Models Using SPSS Mixed Pull-Down Menus
This document contains detailed instructions on fitting multilevel models using the pull-down menus in SPSS. The document is an adjunct is the Peugh and Enders paper that is in press at Educational and Psychological Measurement.

Implement the Bollen-Stine Bootstrap With Missing Data
This Winzip archive (Bollen Stine Files.zip) contains SAS programs that can be used to generate model-based (i.e., Bollen-Stine) bootstrap samples with or without missing data.  The archive also contains the Amos, EQS, Lisrel, and Mplus code needed to analyze the data and obtain a bootstrap test statistic and standard errors.  A manuscript that is currently in press in Structural Equation Modeling is also included, and the manuscript contains detailed programming instructions for using the macro and the associated SEM program files.

Combining Likelihood Ratio Chi-Square Statistics From a Multiple Imputation Analysis
This SAS program combines likelihood ratio chi-square statistics from an analysis of multiply imputed data sets using the method proposed by Li, Meng, Raghunathan, and Rubin (1991).  An application of this method can be found in the missing data book chapter listed above.

Little's (1998) Test of Missing Completely at Random (MCAR) Missing Data
This SAS program implements Little's (1998) MCAR test that was proposed in JASA.  Because of the reliance on PROC MI, this macro can only be used with SAS version 8.2 or higher.  An application of this method can be found in the missing data book chapter listed above.

Implement the Bootstrap for Obtaining EM Algorithm Standard Errors
This SAS program implements the bootstrap for obtaining standard errors for EM algorithm estimates of a mean vector and covariance matrix. Details can be found in Little and Rubin (2002, p. 196).
