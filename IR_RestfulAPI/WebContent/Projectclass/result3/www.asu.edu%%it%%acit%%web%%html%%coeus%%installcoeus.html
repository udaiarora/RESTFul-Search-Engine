Coeus Desktop Application
Download

To install the Coeus desktop application, follow the 4 steps below. Detailed, step-by-step instructions are available in an HTML version or for printing in Microsoft Word version or PDF version. (Note: You must have the correct version of Java WebStart on your computer to install Coeus. Contact your local computer support desk for more information.)

  1. Close all open documents and click on the Coeus installer link below.
  2. If the page does not launch the Coeus Installer, click the "Launch Coeus 4" link.
  3. The installation process will begin. Follow the prompts to finish the installation.
  4. When prompted, click "Yes" that you want a new Coeus icon to appear on your desktop. Double-click on this icon to start Coeus.
