Using the List View Control

List view controls are used to display SCL lists graphically. Typically, list view controls are used to display the contents of a model. For example, dropping a color list model onto a list view control (such as a list box) enables the list box to display a list of color choices.

How to use a list view control

  1. Drag a list view control from the Components window and drop it onto a frame.
