SAS/OR User's Guide: Project Management, Version 8

Discover how you can use SAS/OR software to efficiently manage your projects! This book serves as the primary documentation for all the project management procedures in SAS/OR, namely, the CPM, GANTT, NETDRAW, and PM procedures. In addition, this guide includes documentation for the DTREE procedure, which is used for Decision Analysis. The book also includes a description of the PROJMAN application which is a user-friendly GUI for project management. This book is written for end users who are computer-literate and have experience with the SAS System. This title is included in the SAS OnlineDoc CD-ROM, which is shipped free of charge with the SAS System.
