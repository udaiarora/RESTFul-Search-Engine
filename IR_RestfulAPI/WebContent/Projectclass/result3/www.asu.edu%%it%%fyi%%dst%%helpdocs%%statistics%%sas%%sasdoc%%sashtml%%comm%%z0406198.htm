Communications Access Methods for SAS/CONNECT and SAS/SHARE Software

New SAS/CONNECT Remote Host Spawners Available

A spawner program resides on a SAS/CONNECT remote host and listens for client requests for connection to the remote host. After the spawner program receives a request, it invokes the remote SAS session.

New spawner programs support these remote host types:

  • OpenVMS Alpha

  • OS/390.

Both of these spawners support only the TCP/IP access method for communicating with a supported local host.
