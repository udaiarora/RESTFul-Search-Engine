uni-KEY 1.10           curses/TERMINFO Keyboard Utility
 Please enter your uni-XEDIT profile filename
 (or just press ENTER to return to the main menu)


In the above figure, you type the name of the file which is to contain the keyboard mapping. The file is created in the current directory unless you specify a path name as part of the filename. Do not use an environment variable ($HOME) or the tilda as part of the name. If you use an incorrect name, no error message is issued until after you exit the utility; i.e., you'll lose whatever settings you made.

Use ".profile.xedit" as the filename in your home directory if you need only one keyboard mapping file; otherwise, use it for the most commonly used keyboard mapping file. To make an additional keyboard mapping file, run the key utility again but specify a different filename, preferably a short name with an extension of ".xedit", e.g., "stats.xedit".
