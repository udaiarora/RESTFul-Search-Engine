System Options in the CMS Environment

Portable SAS system options are documented in SAS Language Reference: Dictionary. Only the system options that are CMS-specific or that have host-specific aspects are documented in this section. However, Summary Table of SAS System Options includes all SAS system options that are available under CMS. The table specifies the default values for all options and describes when and where each option can be specified: at SAS initialization, in the SAS configuration file, or in the SAS OPTIONS statement, for example.

For information about using SAS system options under CMS, see Specifying System Options in the SAS Command.
