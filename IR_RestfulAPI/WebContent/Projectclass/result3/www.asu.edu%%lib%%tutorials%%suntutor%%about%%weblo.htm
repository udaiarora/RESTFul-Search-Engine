Learning Outcomes: Find Web Information

Why Use the Web?

  • Given a topic, the student will identify that the web is an effective tool for research.

Web Accuracy

  • The student will recognize that information found on the web can be inaccurate.

Browsing and Searching

  • The student will recognize the difference between browsing and searching.

Breaking Questions into Keywords

  • The student will reduce research questions into concepts and then identify appropriate keywords for each concept.

URLs

  • The student will distinguish which part of the URL indicates the host of the site.

Why Evaluate?

  • The student recognizes the need to evaluate information from the web.

Evaluating Pages

  • The student will judge the quality of a web page based on certain criteria.

Who can help?

  • The student will know to ask a librarian for help when needed.
