Doing More with SAS/ASSIST Software

Profiles items

User

The User item opens the User Profile window. For more information, see Customizing SAS/ASSIST Software.

Master/group

The Master/group item opens the Master/Group Profile window. Master and Group Profiles are used to set SAS/ASSIST software options for entire sites or groups within sites. Master and Group Profiles are usually set up by system administrators.
