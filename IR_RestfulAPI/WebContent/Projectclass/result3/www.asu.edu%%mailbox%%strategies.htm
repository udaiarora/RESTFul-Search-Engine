Strategies for Exchange Mailbox Management

It is becoming increasing important to manage space on the mail server. You will receive a message from the system administrator when your mailbox reaches its size limit of 250 mg. To keep from reaching this limit, it is recommended that you archive the old mail messages that you want to keep and delete the rest. When you archive items in Outlook, you move the folders containing the items to an archive file that you can easily retrieve when you need the information. The archive file can be stored on your hard drive or a network drive.

Please see the following instructions on how to archive mail and other space saving tactics.

Manual Archive

To manually archive a folder:

  1. Select the folder.
  2. Choose File from the menu.
  3. Select the Archive menu option.
  4. Give the archive file a path and name. (location)


AutoArchive

The AutoArchive feature can simplify the archiving of your Outlook folders. When Outlook is installed, several of the Outlook folders already have AutoArchive turned on, where folder contents are archived automatically each time you start Outlook. The AutoArchive settings for each folder are checked by date. Any items that have aged past the archiving date are automatically moved to you archive file.

There are two steps in setting up AutoArchive:

  • Establish the Outlook AutoArchive properties.
  • Establish the settings for individual folders.


Set Up AutoArchive Properties


To set up the Outlook AutoArchive properties:

  1. Click Tools on the menu and select Options.
  2. Select the Other tab.
  3. Then click AutoArchive. The AutoArchive dialog box appears.
  4. Be sure that the AutoArchive Every check box is selected, and insert a number and time period. AutoArchive will run the first time you start Outlook each day.
  5. Put a check in the Prompt Before AutoArchive check box. You will be asked to confirm the archiving before it starts.
  6. Click in the Default Archive File box and give your archive file a name or accept the default.


Set Properties for a Folder

To set the AutoArchive properties for a folder:

  1. Right-click the folder that you want to setup.
  2. Choose the Properties from the menu.
  3. Choose the AutoArchive tab.
  4. Fill in the Clean Out Items Older Than box.
  5. Be sure the Move Old Items To options is selected.
  6. Then name your archive file or accept the default name.


Retrieve Archived Folder

To retrieve an archived folder, use the Import and Export wizard.

  1. Choose File from the menu, then choose Import and Export...
  2. In the Choose An Action to Perform box, click Import from another program or file, and then click Next.
  3. Select File type to import from box, scroll down, choose Personal Folder File (.pst), and then click Next. The File To Import dialog box appears.
  4. Use Browse to select the file to import and click Next.
  5. Select Archive Folders from the Select the folder to import from box. Click on Finish.


Other Space Saving Tactics

Archiving and deleting old mail are excellent ways to save space on the mail server, but there are a few other ways to accomplish the same result.

  1. Don’t let your Sent Items folder become too large. Especially be on the lookout for messages with attachments. Every time you forward a message, the original message is included in the reply and saved to your folder. If the original message has a large attachment, you will be making an unnecessary copy of the large attachment.
  2. Instead of keeping mail messages with attachments, save the attachments as files on your hard drive or network drive and delete the mail message.
  3. Be careful when turning on the Journal feature. The Journal will record your activities and become a very large, unmanageable file in a short time.
