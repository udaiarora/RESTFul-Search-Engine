Text Entry Control: _getMark


Returns the position of the marked text

Syntax

Syntax

objectName._getMark( startPosition, endPosition );

Argument Type Use Description
startPosition Numeric Output returns the starting position of the mark
endPosition Numeric Output returns the ending position of the mark
