SASAUTOS=

SASAUTOS=


Specifies an aggregate external file that contains the automatic call source macros

Default: SASAUTOS
Valid in: configuration file, SAS invocation, OPTIONS statement, OPTIONS window
CMS specifics: library-specification, operating environment behavior

Syntax
Details
See Also

Syntax

SASAUTOS=aggregate-specification | ('aggregate-specification-1, ...'aggregate-specification-n')

aggregate-specification
can be a the name of a MACLIB or SFS directory, or a fileref associated with a MACLIB or SFS directory, or a DDname associated with a MACLIB. For details, see Identifying an External File .

Details

To use the autocall facility, the SAS system option MAUTOSOURCE must be in effect.
