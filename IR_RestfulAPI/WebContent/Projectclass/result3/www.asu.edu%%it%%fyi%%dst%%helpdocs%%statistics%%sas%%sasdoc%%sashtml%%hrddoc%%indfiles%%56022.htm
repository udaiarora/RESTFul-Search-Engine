SAS/GRAPH Software: Examples, Version 6, First Edition

Learn how to use the power of SAS/GRAPH to produce attention-getting graphs! This task-oriented book presents thirty practical examples of useful business graphs and walks you through the code for each example. In addition, portions of each program are singled out for a closer look to help you understand better how the program (and SAS/GRAPH) works. This book is ideal for users who want to learn how to get the most out of SAS/GRAPH.

320 pages.
