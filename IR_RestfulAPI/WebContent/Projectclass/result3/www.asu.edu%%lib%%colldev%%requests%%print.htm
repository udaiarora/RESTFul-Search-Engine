Request for Print Journals


If you are an Arizona State University Librarian subject selector, or area specialist, use this form to request print journals that you believe should be added to the ASU collections. This is for print subscriptions only - if the journal is available electronically, please fill out the Electronic Resource Request Form.

If you are a member of the ASU faculty or an ASU student, please use the Online Request Form to recommend items for the library to purchase. Please remember this is a recommendation only; it will be reviewed in terms of content and price by the Collections and Scholarly Communication Office. For questions, contact the CSC Office at 965-5250.
