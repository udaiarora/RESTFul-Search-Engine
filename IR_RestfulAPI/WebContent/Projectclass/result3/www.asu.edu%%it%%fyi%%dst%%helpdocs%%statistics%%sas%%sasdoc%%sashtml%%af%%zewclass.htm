Object Class: _newClass


Creates a new class based on the current instance

Syntax

Syntax

return = objectName._newClass( entryName );

Argument Type Use Description
return Object Return returns the identifier of the new class created from the instance
entryName Character Input specifies the four-level name of the CLASS entry to create
