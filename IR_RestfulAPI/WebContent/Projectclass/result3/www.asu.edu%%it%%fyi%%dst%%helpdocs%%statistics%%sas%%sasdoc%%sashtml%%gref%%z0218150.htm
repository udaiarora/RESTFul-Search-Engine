%SEQUENCE

%SEQUENCE


Specifies when to draw Annotate graphics elements, relative to the procedure's graphics output or relative to the other Annotate graphics drawn.

Variable affected: WHEN

Syntax

Syntax

%SEQUENCE (when);

The value of when is not case sensitive, must be one of these values, and must not be enclosed in quotation marks:
AFTER literal; specifies that subsequent Annotate graphics will overlay procedure graphics. A is an alias for AFTER.
BEFORE literal; specifies that subsequent Annotate graphics will be overlaid by procedure graphics. This value is the default. B is an alias for BEFORE.
