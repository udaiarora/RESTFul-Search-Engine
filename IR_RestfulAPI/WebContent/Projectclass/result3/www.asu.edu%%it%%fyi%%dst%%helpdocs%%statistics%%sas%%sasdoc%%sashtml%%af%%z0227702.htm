Catalog Entry Model: _getTitleColor

Catalog Entry Model: _getTitleColor


Returns the color and optionally the display attribute of titles in an OUTPUT catalog entry

Syntax
Details
Example

Syntax

CALL NOTIFY (viewer-name, '_getTitleColor', color<, attr>);

Argument Type Description
color
 C specifies or returns a color
attr C specifies or returns a display attribute:
'NONE'
'REVERSE'
'UNDERLINE'
'REVERSE UNDERLINE'


Details

By default, title lines in the viewer are the color assigned by the SASCOLOR window element, TITLE.

Example

If the _setTitleColor method has never been called, the _getTitleColor method returns the name of the SASCOLOR element, 'TITLE'. 

call notify('textview', '_get_title_color_',
            color);
