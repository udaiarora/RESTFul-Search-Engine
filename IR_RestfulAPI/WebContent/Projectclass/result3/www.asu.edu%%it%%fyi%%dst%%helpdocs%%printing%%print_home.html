How to Print to Local Printer in Pine

It is possible to print to a local printer in your home or office (or to a network attached printer) from within Pine, provided the terminal emulator you are using is capable of doing so.

At the Pine main menu, type "s" to select the Setup menu item, then type "p". Type 1 and press enter. Note that this may not work with all attached printers, and will depend on the terminal emulation/communications software in use.
