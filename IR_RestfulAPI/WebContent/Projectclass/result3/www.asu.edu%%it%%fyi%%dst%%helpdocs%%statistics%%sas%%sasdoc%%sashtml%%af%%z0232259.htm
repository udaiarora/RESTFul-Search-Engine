Data Form: _getAttributes


Returns the current attribute settings for the data form

Syntax
See Also

Syntax

CALL SEND (form-id, '_getAttributes', attr-list);

Argument Type Description
attr-list
 N specifies the identifier of an SCL list to contain the current values of all the named attributes listed in the table provided with the _setAttributes method.

_setAttributes.
