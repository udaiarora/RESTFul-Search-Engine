EPA

The EPA data set describes the results of measuring sulfate concentrations.

The following table shows the variables in EPA and how they were assigned to SAS/SPECTRAVIEW variables. Note that the optional BY variable is included.

Variable Dimension Description
HOURS BY Variable hour that the reading was taken
LEVEL Z Variable layer above ground
LNGITUDE X Variable longitude coordinate
LATITUDE Y Variable latitude coordinate
SULFATE Response sulfate concentration percentage
OZONE not used ozone concentration percentage
