SAS/AF Software: Class Dictionary

Overview

The Check Box class enables you to create check boxes. A check box, which consists of a marker character and an associated label representing a choice, acts like a switch. When the user selects a check box, it becomes active; when the user selects the check box again, it becomes inactive. When the check box is active, it appears to be highlighted.

Parent:
sashelp.fsp.widget.class

Class:
sashelp.fsp.CheckBox.class
