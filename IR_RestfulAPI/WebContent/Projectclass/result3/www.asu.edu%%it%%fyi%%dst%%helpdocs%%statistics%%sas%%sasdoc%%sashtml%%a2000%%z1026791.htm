CREATE


Creates a database definition

Optional statement
Applies to: New database

Syntax
Details

Syntax

CREATE;

Details

The CREATE statement indicates you want to create a SYSTEM 2000 database definition, but not load any data. By default, the DBLOAD procedure expects you to load data.
