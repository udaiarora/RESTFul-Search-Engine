SAS/ACCESS Interface to CA-DATACOM/DB Software: Reference, Version 8

This book provides complete details for using the ACCESS procedure a non-windowing environment. It contains reference information required by an applications programmer to write access and view descriptors to access CA-DATACOM/DB data and use it in the SAS System. Note: The ACCESS procedure is no longer available through a windowing interface.

150 pages.
