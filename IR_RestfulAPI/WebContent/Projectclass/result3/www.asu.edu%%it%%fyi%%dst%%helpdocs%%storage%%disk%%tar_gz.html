Working with .tar.gz Files

Q. I am not sure how to deal with a *.tar.gz file? It seems UNIX wants a *.Z file when I use the 'uncompress' command.

Let us suppose you have a file called 'sasfiles.tar.gz'. The .tar.gz filename extension implies that an archive file (.tar) was first created with the 'tar' utility and then a compressed file (.gz) was created with the 'gzip' utility. Hence the reason for the .tar.gz file.

To uncompress, first use the 'gunzip' command, for example:

gunzip sasfiles.tar.gz

This creates an uncompressed file called 'sasfiles.tar'. Now this can be unarchived using the command given in the document titled "Archiving/Unarchiving using the Tar Command".
