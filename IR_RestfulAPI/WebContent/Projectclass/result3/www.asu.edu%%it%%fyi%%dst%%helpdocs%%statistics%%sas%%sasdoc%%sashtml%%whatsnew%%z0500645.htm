Production Status: PROJMAN (new version) and QSIM

The new version of PROJMAN features a completely new graphical user interface, replacing the former menu-based interface. PROC PM is now the primary tool used by PROJMAN for editing activity data in project models. PROJMAN now incorporates all of the latest project modeling features added elsewhere in SAS/OR software, and it is fully compatible with both the earlier version of PROJMAN and the beta product, XPROJMAN. Project models defined using these products can be imported into the new PROJMAN for further work.

QSIM is an application that enables you to construct models of queueing systems and to analyze their performance through the use of discrete event simulation. QSIM's interactive model builder provides a graphical user interface so that you can use drag-and-drop to create your queueing system model. You can monitor and debug the functioning of your queueing model visually by animating the flow of transactions through the model graphically. You can sample data generated from queueing models and analyze the data using any of the statistical procedures of the SAS System.

QSIM offers a large number of built-in components for sampling, queueing, logical routing, and monitoring, all of which can be modified extensively by varying their individual options and parameters. Additionally, you can assemble customized components by encapsulating sections of your model. Saving and replicating these assembled components reduces your workload and helps avoid clutter in large, complex models.
