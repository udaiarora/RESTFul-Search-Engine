Feature Presentation
  • Exchange Update
    By Nate Wilken

    • Exchange Environment Growth
      • 25% more mailboxes
      • 500% larger quota (250MB)
      • 400% larger mailbox size (65MB)
    • Infrastructure Improvements
      • www.asu.edu/mailbox/experformance.htm
      • Increase storage system performance, capacity, reliability
      • Increase & upgrade Exchange servers
      • Secondary storage email archiving
      • Microsoft Product Support Services Health Checks
      • Mirroring & clustering enabled
    • Customer Best Practices ? see www.asu.edu/mailbox/experformance.htm
      • Use Outlook 2003 SP1 on Windows XP SP2 system
      • Keep Outlook updated (office.microsoft.com/officeupdate)
      • Publish less free/busy data & don?t retrieve calendar data in planner
      • Use Exchange Cached mode (only available in Office 2003 & is the default)
      • Use auto-archiving
      • Delegates & Owners should use the same Outlook version
      • Home VPN/Outlook users should change router IP address
    • Email Archiving
      • Automated nightly process, policy-controlled archiving to online secondary storage
      • Archived items will appear as shortcut but seamless access
      • Reduce mailbox size
    • Exchange 2003 Features
      • Improved Outlook 2003 and Outlook Web Access
      • RPC/HTTP reduces VPN need
    • Exchange 2003 Pilot
      • Email group will identify pilot members
      • Hope to begin upgrade to 2003 by end of summer
