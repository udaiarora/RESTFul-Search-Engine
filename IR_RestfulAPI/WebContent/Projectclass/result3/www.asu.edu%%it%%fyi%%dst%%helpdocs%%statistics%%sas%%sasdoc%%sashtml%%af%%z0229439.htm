Graphic Text: _isColor


Reports whether the specified graphic text color is a valid SAS/GRAPH color

Syntax
Example

Syntax

CALL NOTIFY (grtext-name, '_isColor', color-name, value);

Argument Type Description
color-name
 C returns the name of the color or the hexadecimal representation of the color.
value
 N is a flag indicating whether the color is a valid SAS/GRAPH color:


0 color is invalid


1 color is valid

Example

_isColor returns a value of 1 in the variable val, indicating that the color specified by name or hexadecimal value is a valid SAS/GRAPH color: 

   call notify('grphtxt1','_is_color_',
               'blue',val);
   call notify('grphtxt1','_is_color_',
               'CX00A200',val);
