PGMPARM=


Specifies the parameter that is passed to the external program specified by the SYSINP= option

Default: none
Valid in: configuration file, SAS invocation
Category: File Control: EXTFILES
OS/390 specifics: all

Syntax
Details

Syntax

PGMPARM='string'

string
can be up to 255 characters long. The quotation marks are optional unless the string contains blanks or special characters.

Details

The PGMPARM= option specifies the parameter that is passed to the external program specified by the SYSINP= option. For more information about using the PGMPARM= and SYSINP= options, contact your lcoal SAS Support Consultant.
