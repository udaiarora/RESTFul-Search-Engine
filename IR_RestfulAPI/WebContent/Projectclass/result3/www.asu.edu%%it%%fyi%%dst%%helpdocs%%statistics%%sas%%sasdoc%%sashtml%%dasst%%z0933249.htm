Selecting Other Data

In the Report Engine window, you can select new tables and queries and retain modified column information. For example, you may have a query that is based on the AIRLINE.MARCH data file. If you change the format for the Departure date column from DATE7. to DATE9. for the report and then select another query with a Departure date column, the new report also uses the DATE9. format.

To select another query or table, fill in the Data and Type fields as described in Opening and Using the Report Engine Window.
