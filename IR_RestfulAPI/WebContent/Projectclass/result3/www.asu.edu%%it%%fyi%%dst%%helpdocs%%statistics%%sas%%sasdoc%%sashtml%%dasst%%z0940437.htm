Opening the SQL Editor Window

To open the SQL Editor window, follow this path:

Tasks
Data Management
Query
Query and Reporting
SQL Editor

The SQL Editor window appears.

SQL Editor Window

If the window is filled when you open it, it contains a last query that was loaded into this window. To clear the window, select New from the File menu.
