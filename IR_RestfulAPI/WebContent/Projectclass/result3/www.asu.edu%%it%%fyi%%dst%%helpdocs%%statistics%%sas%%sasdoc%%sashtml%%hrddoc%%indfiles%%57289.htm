SAS/ACCESS Interface to IMS-DL/I Software, Version 8

Consider this documentation your source for complete details on accessing IMS-DL/I data with the SAS System. The ACCESS procedure has been enhanced to include new statements and options. The UPDATE, DELETE, INSERT, and REPLACE statements let you easily modify, access, and view descriptors. Information on using the DATA step interface is also included. This documentation contains information required by an applications programmer. Note: the interactive windowing interface is no longer supported.
