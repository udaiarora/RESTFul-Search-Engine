X


Enters host-system mode or enables you to issue a host-system command without ending your SAS session

OS/390 specifics: portable version of the TSO command

Syntax
Details

Syntax

X <command>

Details

The X and TSO commands are identical, with one exception: under an operating environment other than OS/390, the TSO command has no effect, whereas the X command is always processed. See TSO for more information.
