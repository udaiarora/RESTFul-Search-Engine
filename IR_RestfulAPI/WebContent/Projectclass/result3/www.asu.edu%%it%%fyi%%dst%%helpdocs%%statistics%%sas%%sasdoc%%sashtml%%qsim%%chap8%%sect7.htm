Analyzing the Sample Path

Time in the System

By default, the Bucket component accumulates the age or system time of the transactions arriving to it. However, it can accumulate any attribute of the transaction including user-defined attributes. The Bucket control panel has a check box which when set, records the attribute values in the SAS data set. Figure 2.15 shows such a control panel. If a simulation has been executing and the Collect Data check box has been set, then you can click the Analyze button to submit SAS code that calculates statistics on the sample.

Univariate statistics are calculated and histograms are printed for the attribute named in the Bucket control panel.
