Alternate Method to Import Queries

If you want to import just a few queries, you can use the Enhanced Editor in the SQL Editor window instead of the Import Queries window. To open the Enhanced Editor, follow this selection path:

Tasks
Data Management
Query
Query and Reporting
SQL Editor
Edit
Enhanced Editor
When you select Include from the File menu, the Include window opens and prompts you for the name of an external file. Type the name of the external file and query that you want to import. Enclose the external file in quotes if necessary.

Select OK to include the query in the Enhanced Editor. The query is "imported."
