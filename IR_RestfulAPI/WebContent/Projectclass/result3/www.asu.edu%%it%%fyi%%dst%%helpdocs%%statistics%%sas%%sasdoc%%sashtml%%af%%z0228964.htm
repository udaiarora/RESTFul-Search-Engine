Data Set Data Model: _deleteRow

Data Set Data Model: _deleteRow


Deletes the current row from the table

Syntax
Details

Syntax

CALL SEND (object-id, '_deleteRow');

Details

The _deleteRow method is an editing method and is not valid when

  • browsing a table or when the NODELETE option is specified in the _setDataset method, or

  • Deletion Allowed is disabled under the Table Characteristics of Data Form or Data Table.

CAUTION:
Deletions cannot be recovered. The _deleteRow method deletes rows from the table, not just from the display of the table. You cannot recover the contents of a deleted row.  

A row must be locked before it can be deleted. Once the deletion is completed, the model no longer has a current row. The _deleteRow method sets SYSRC for error, note, and warning conditions.
