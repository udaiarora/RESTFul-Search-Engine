SAS/AF Software: Class Dictionary

Overview

The Class class and any subclass of the Class class are metaclasses and enable you to

  • get information about an application, such as information about the application's structure, which classes are being used, and which classes are using certain instance variables

  • get information about a class' parent and child classes

  • list the instances of a class and process all of those instances in some way

  • create objects and classes at run-time.

By default, all classes loaded in SAS/AF software are instances of the Class class.
