Widget Class: _bupdate


Updates an object upon exiting the Object Attributes window at design time

Syntax
Details

Syntax

objectName._bupdate( );

Details

This method runs automatically by the _attributesDialog method when you select OK from the Object Attributes window. The default behavior updates the attributes of an object. For example, changing the color of a text entry object, updating the icon number, or displaying a new Grseg entry for a SAS/GRAPH Output object.
