"PERSONALLY IDENTIFIABLE"

Decorative Image: Text Box: "PERSONALLY IDENTIFIABLE" MEANS DATA OR INFORMATION WHICH INCLUDES: Decorative Image: Text Box: (1) THE NAME OF THE STUDENT (2) THE STUDENT'S CAMPUS OR HOME ADDRESS; (3) A PERSONAL IDENTIFIER (SUCH AS A SOCIAL SECURITY NUMBER OR STUDENT NUMBER) (4) A LIST OF PERSONAL CHARACTERISTICS OR OTHER INFORMATION WHICH WOULD MAKE THE STUDENT'S IDENTITY EASILY TRACEABLE

To determine what is "personally identifiable" FERPA provides these guidelines. Note that "personally identifiable" includes more than just a student's name that can include a "personal identifier" such as as a social security number. This is why it is illegal to post grades by ID or Social Security Number without obtaining the student's written permission.
