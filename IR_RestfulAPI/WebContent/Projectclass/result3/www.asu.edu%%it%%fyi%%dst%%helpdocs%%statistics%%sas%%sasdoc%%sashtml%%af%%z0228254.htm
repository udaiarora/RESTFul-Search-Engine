Critical Success Factor: _selectFormat

Critical Success Factor: _selectFormat


Selects the format for displaying minimum and maximum values

Syntax

Syntax

CALL NOTIFY (csf-name, '_selectFormat', type);

Argument Type Description
C specifies the type of format to use. Valid values are DATA for the format data set and RANGE for the format specified in the RANGE entry.
