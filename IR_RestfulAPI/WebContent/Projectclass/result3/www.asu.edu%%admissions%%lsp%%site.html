* - Note for Netscape Macintosh users: There are serious cache corruption issues with Netscape 4.x for Macintosh. If your browser does not correctly display images after reloading a previously cached page, you must clear the memory and disk caches before you reload. You can also turn off the disk cache entirely to fix the problem. This issue is fixed with Netscape version 6 and only is a problem with version 4.x for Macintosh. The Windows and Unix versions are not affected.
