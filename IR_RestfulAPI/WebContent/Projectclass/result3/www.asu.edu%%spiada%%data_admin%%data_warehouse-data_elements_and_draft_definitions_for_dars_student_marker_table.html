Currently Enrolled Flag

Char

1

This flag (Y/N) is set to Y if a student is currently enrolled in classes.  If the flag is an N it means that the student either completely withdrew from the current semester or they were not registered in the current semester.  This flag is set for fall and spring semesters only.  For the purposes of this flag, fall begins the first day of summer session.  Spring begins the first day of spring classes.

Char

3

The code for the campus that administers the marker (may be different than the campus of the degree program). If not different from the degree program it is taken from the DARS_DEGREE_PROGRAM_CAMPUS  in the DARS_STUDENT_DEGREE_PROGRAM table in the Student warehouse.  Values are ASM and ASW.

DARS Marker Campus Code

Char

1

The code for the campus that administers the marker (may be different than the campus of the degree program).  If not different from the degree program it is taken from the CAMPUS_CODE in the DARS_STUDENT_DEGREE_PROGRAM table in the Student warehouse.  Values are M and W.

DARS Marker Catalog Year Sem

5
