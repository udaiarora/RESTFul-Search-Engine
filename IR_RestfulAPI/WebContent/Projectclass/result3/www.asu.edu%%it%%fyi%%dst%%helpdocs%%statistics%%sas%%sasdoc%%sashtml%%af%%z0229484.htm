Icon: _setLabel

Icon: _setLabel


Assigns a label to an icon

Syntax
Details

Syntax

CALL NOTIFY (icon-name, '_setLabel', label);

Argument Type Description
label
 C specifies the label to assign

Details

_setLabel assigns the value specified in label as the new label for the icon. If the icon does not have a return value, the new label also becomes the new return value.
