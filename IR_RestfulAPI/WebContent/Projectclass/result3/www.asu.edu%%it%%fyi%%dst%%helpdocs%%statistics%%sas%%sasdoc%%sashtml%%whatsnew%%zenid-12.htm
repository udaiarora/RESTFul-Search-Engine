Introduction

SAS/IntrNet software provides a powerful set of components that enable you to create and deploy Web-enabled applications that use the power of SAS software. These applications can provide remote access to your organization's data and can bring the computing power of SAS software to any desktop.

Complete documentation for SAS/IntrNet software is provided on the SAS Web site at http://www.sas.com/rnd/web/intro.html. The documentation is also provided on the media on which you receive SAS/IntrNet software.
