Catalog Entry Model Methods

Methods specific to the Catalog Entry Model class are described here. Inherited methods are described in the Text Model Template class. These methods can be used when the text viewer is connected to a catalog entry model.

Note:    To call these methods from a viewer connected to this model, such as the catalog entry viewer, you can use the CALL NOTIFY routine as shown in the syntax. If you are using the model without a viewer, you should call these methods using a CALL SEND routine to the model identifier. The syntax and examples that follow assume you are using a viewer.  

Note:    Changes to colors or attributes made with the catalog entry model are not saved in the catalog entry.
