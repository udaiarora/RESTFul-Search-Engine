Saving and Restoring

Transaction Sample Paths

The state information on the components in a transaction's sample path can be saved to a data set in a Bucket. Figure 8.8 shows the Collect Data check box, which must be set to start saving data on the transactions arriving to that Bucket. These data are placed in a data set named for the number of Bucket's. You can see and change the default name in the Bucket control panel.
