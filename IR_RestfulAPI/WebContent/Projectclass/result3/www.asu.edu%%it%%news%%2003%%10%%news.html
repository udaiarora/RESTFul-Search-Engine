Latest VirusScan Update: SuperDat 4296
Latest Virex 7 Update: 030917
Latest Virex 7 Update to Scan Engine: 4160
Netshield Netware: 4240

PC users: VirusScan virus definition and scan engine upgrade (sdat) version 4296 is now available. Also available is VirusScan 4.5.1 Service Pack 1, which enables VirusScan to run under Windows XP. (Note that XP remains unsupported by Information Technology.) See the ASU Anti-Virus Web site.

Macintosh users: The Virex 7.2 is available for download or auto-updating, which works on OS X only. Virex 6.1 is available for all other versions. See the ASU Anti-Virus Web site.

Handheld Devices: VirusScan Wireless 2.0 is available which supports most types of handheld devices using Palm OS, Pocket PC, Windows CE, and EPOC operating systems. See the ASU Anti-Virus Web site.

If there are any questions on download or installation, please send an e-mail note to COMM-Q.

To receive the most current antivirus news by e-mail, subscribe to the Anti-Virus Announcement List.

Note: This software is valid to use on either private or ASU-owned machines, so please install and keep up-to-date on all machines you use.
