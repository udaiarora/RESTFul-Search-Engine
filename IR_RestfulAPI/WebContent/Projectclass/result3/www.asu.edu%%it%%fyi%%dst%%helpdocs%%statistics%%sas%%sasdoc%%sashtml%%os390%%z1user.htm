SASUSER=

SASUSER=


Specifies the location of the SAS data library that contains the user profile catalog

Default: SASUSER
Valid in: configuration file, SAS invocation
Category: Environment Control: ENVFILES
OS/390 specifics: library-specification

Syntax
Details
See Also

Syntax

SASUSER=library-specification

library-specification
can be any valid DDname, the name of the physical file that comprises a SAS data library, or a UNIX System Services directory; the DDname must have been previously associated with the SASUSER SAS data library using either a TSO ALLOCATE command or a JCL DD statement.

Details

If a UNIX System Services directory is being used, it must have been created beforehand.
