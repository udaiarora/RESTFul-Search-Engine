GWRITER

GWRITER


Specifies the name of the external writer used with IBM printers.

Used in: GOPTIONS statement
Default: SASWTR
Restriction: Used only with IBM3287 and IBM3268 device drivers on OS/390 systems

Syntax

Syntax

GWRITER='writer-name'
