Software Licensing at ASU

Students, faculty and staff at Arizona State University have the benefit of getting access to software for academic and administrative use. The Software Licensing Web site has information about site license software and volume licensing options from Microsoft.

Please note that software that is available for download from this site is restricted to ASU staff only, and requires a valid ASURITE UserID and password to download.
