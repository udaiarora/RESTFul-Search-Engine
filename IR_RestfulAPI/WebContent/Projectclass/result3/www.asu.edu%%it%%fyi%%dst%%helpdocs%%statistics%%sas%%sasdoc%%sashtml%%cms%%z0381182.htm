GCURSOR


Turns the graphics cursor on and off (3179G and 3192G terminals only)

CMS specifics: all

Syntax
Details

Syntax

GCURSOR <ON> | <OFF>

Details

This command is used only with 3179G, 3192G, IBM5550, and IBM3572G graphics terminals. When a mouse is attached, the default setting for GCURSOR is on. Without a mouse, the cursor movement keys are used to position the graphics cursor. The GCURSOR command acts like a toggle switch. Alternatively, you can use the ON and OFF operands.
