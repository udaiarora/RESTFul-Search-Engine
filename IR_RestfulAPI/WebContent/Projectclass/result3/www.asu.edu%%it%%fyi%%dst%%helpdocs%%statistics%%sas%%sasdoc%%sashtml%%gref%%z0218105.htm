%CNTL2TXT


Copies the values of the internal coordinates (XLAST, YLAST) to the text coordinate (XLSTT, YLSTT).

Variable affected: FUNCTION
Updates: XLSTT, YLSTT

Syntax
Details
See Also

Syntax

%CNTL2TXT;

Details

%CNTL2TXT is useful when you are calculating the position of labels on a graph. See CNTL2TXT Function for an example.
See Also

%SLICE for drawing a filled circle.
