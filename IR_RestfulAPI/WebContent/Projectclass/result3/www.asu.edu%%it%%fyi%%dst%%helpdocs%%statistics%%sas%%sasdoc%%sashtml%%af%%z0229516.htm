Image Icon: _getImage


Returns the IMAGE entry being displayed in an image icon

Syntax

Syntax

CALL NOTIFY (icon-name, '_getImage', IMAGE-name);

Argument Type Description
IMAGE-name
 C the one-level name of the currently displayed IMAGE entry
