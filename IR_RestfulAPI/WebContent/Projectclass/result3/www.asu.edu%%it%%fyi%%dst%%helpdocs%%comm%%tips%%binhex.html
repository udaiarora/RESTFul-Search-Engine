How do I Decode a File that is BINHEX Coded?

Question:
I received an email message from a friend with an attachment. The atttachment says:

17KB Applications/Mac-BINHEX40

I saved it onto my "m(?)" drive like the guys here told me to do but I can't see it. It's coded or something. For example, one line says:

PFL"SB@3jgf&ZG'9N)'%JBQpj)'&ZC#"PGQ9Z)(4SEh9RD#"*)(GKFb"K)'GTFQ'

How can I look at this document, save it on to my disk and/or print it?

Response:
BINHEX4.0 is a form of coding data that is transmitted over the internet. You can decode the file by using a BINHEX converter, however, the easiest way I know of converting the file and saving it to your disk is to use a File Transfer Protocol (FTP) program for the Mac called Fetch. Fetch will decode and download the file from your AFS space/"m-drive" and save it on a disk on the Mac.

To use Fetch in this manner, you will need to do the following:

Open Fetch. For the host name enter "general.asu.edu". Use your ASURite id and password for the appropriate fields. This should bring up a list of files on your AFS space. Go to the "customize" menu and select Preferences. Under "topic" go to "formats" and then check the BinHex4.0 box. Click on OK. Then highlight the name of the file and click on "get file". This should decode the file and download it to the disk of your choice.

Once the file is decoded, you can view it in or print from an application that is compatible with the application that created the file.
