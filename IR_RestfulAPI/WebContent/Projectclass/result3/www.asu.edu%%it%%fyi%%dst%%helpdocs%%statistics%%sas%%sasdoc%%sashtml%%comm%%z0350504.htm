Communications Access Methods for SAS/CONNECT and SAS/SHARE Software

SAS/CONNECT Scripts

SAS Institute provides sample script files that you may customize and use with SAS/CONNECT to sign on to and sign off a remote host. Names of script files and their locations on the operating system are provided in this documentation, where relevant. The content of selected script files is provided to draw your attention to statements that may need to be modified in order to work at your site. See Sign-on Script Files for these sample script files.

Note:   The script files were written to support Version 6 and Version 7.
