Widget Class: _unhide


Displays a widget that has been hidden

Syntax
Details

Syntax

objectName._unhide( );
objectName._unhide( all );

Details

Example

_unhide displays the buildIcon object if the variable catalog is not in error: 

 if field('error', 'catalog') = 0 then
    buildIcon._unhide();
