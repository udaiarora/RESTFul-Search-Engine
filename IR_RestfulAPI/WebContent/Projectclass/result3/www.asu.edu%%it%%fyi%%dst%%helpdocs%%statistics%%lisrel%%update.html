LISREL, PRELIS and MULTILEV commands

We were currently using LISREL version 8.54, PRELIS version 2.5 and MULTILEV version 5.4.

The command to execute lisrel is:

lisrel input_file output_file &

The command to execute prelis is:

prelis input_file output_file &

The command to execute multilev is:

multilev input_file output_file &

where 'input_file' is the name of the command file and 'output_file' is the output file which is to be created. The message:

[1] + Done ...

signals the end of the job.
