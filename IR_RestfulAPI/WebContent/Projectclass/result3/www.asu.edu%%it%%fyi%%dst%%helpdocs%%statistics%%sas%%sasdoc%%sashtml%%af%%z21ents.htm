Text Label Control Events

Events specified for the Text Label Control class are described here.

conformSize
Occurs when the control is resized due to conformSize

conformSize changed
Occurs when the conformSize attribute is changed

font changed
Occurs when the font attribute is changed

justification changed
Occurs when the justification attribute is changed

label changed
Occurs when the label attribute is changed

labelColor changed
Occurs when the labelColor attribute is changed

mnemonicCompanion changed
Occurs when the mnemonicCompanion attribute is changed

popup
Occurs on a popup request
