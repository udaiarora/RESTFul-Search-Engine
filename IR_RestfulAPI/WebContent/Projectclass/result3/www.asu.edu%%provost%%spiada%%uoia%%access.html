Accessibility of Institutional Analysis Pages

Institutional Analysis web pages provide a text equivalent for every non-text element. This aids users who listen to the content of the site by using a screen reader. Our pages use a consistent style of presentation allowing users to easily find navigational elements, and each page provides a "Go to Page Content" link at the beginning of each page to allow users to skip navigation links to reach the content of a page quickly. We use style sheets to specify text fonts and background colors so that users can override font and color choices with their own style sheets.


We do not use animated gif graphics, flashing text, or any colors to convey information. Our pages do not use any Java applets or programs that need special plug-ins. In addition to the features that make the graphics version of the site more accessible, a text-only version provides added convenience for users of screen readers, PDAs and web-enabled cellular phones.


We use Adobe Acrobat for our online publications. Users can get information regarding the accessibility of Adobe Portable Document Format (PDF) files from the Access Adobe website.
