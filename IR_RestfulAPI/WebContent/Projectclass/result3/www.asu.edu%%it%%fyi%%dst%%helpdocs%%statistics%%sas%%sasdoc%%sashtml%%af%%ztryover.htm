SAS/AF Software: Class Dictionary

Overview

The Registry Component class provides an SCL programming interface to the SAS Registry API and supplements PROC REGISTRY and the SAS Registry Editor. The SAS Registry holds persistent information about SAS system settings. You can use the registry class to programatically create, query, and delete keys in the registry. You can also query and assign named values to the keys.
