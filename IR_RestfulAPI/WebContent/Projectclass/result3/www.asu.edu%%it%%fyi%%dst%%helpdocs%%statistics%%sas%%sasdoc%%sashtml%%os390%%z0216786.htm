SAS Companion for the OS/390 Environment

System Options in the OS/390 Environment

Portable system options for base SAS are documented in the SAS Language Reference: Dictionary. Only the base SAS system options that are specific to OS/390 or that have aspects specific to OS/390 are documented in this chapter. However, Summary of All SAS System Options Available under OS/390 lists all the base SAS system options that are available under OS/390.

For information on system options that support a SAS product, such as SAS/ACCESS, SAS/CONNECT, or SAS/SHARE, see the documentation for that product.

For information about using SAS system options under OS/390, see SAS System Options.

For information on file specifications, see Referring to External Files and Ways of Allocating External Files.
