IT has made available printable information guides in the form of a downloadable PDF file.

Pocket Guide

The two-sided pocket guide contains basic information about IT services and computing sites, and a listing of libraries and copy centers on campus. Keep it with you for handy reference throughout the semester. This document is designed to be printed on 8.5 x 11" paper, double-sided and folded into 4 quarters.

Student Guide

The Student Guide is the printable version of the online Student Guide available from this Web site. The guide contains information about computer security, ASURITE services, Web and e-mail, computing sites and how to get computing help. This document is designed to be printed on 8.5 x 11" paper, single or doubled sided (12 pages.)
