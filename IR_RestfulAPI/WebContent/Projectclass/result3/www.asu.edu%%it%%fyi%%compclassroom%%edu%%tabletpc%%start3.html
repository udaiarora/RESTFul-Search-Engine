Writing

Using the tablet as a writing pad is simple:

  • On a convertible model, swivel the screen and lay it flat on the keyboard.
  • Select Notepad or Journal from the software menu.
  • Select the pencil icon at the top of the screen.
  • Begin writing.

Notes may be saved as typed text by selecting the lasso at the top of the screen. Lasso around the text, then select Action > Convert Handwriting to Text.

Windows Journal

Windows Journal is a program that turns the tablet into an electronic writing pad for taking and sharing notes. Notes are handwritten on the screen using the pen. Both print and cursive are recognized. Notes can be saved in handwritten form or converted to typed text.

Other features include the ability to change pen styles, erase, search notes, and use Sticky Notes. Saving notes written in Journal is similar to saving a Microsoft Word document. All notes can be saved and organized using folders and other desktop tools.
