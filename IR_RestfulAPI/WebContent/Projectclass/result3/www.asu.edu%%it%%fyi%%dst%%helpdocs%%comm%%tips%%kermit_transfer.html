Downloading and Uploading Files Using Kermit

To download files:

  1. Connect to the server (e.g. general.asu.edu).
  2. Start Kermit on the server with the command 'kermit'.
  3. If downloading a non-text file, enter 'set file type binary'.
  4. Enter 'send <host_filespec>'.
  5. Hit your [END] key to bring up a local kermit command prompt and enter 'receive <local_filespec>' or
    'receive binary <local_filespec>' if it's a binary file.
  6. When it's done, enter 'c' to connect and 'quit' to quit the remote kermit.

To upload files:

  1. Start Kermit on server
  2. Type 'receive' or 'receive binary' if non-text file.
  3. Escape to local kermit.
  4. Type 'set file type binary' if non-text file.
  5. Type 'send <local_filespec>' to upload the file.
  6. Reconnect and quit the remote kermit.

For further assistance, please send an email message to "comm-q@asu.edu".
