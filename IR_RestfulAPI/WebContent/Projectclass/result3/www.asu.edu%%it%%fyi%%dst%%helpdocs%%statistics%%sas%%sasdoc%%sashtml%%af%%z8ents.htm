Graphic Text Control Events

Events specified for the Graphic Text Control class are described here. See Widget Class Events for inherited events.

aspectRatio changed
Occurs when the aspectRatio attribute is changed

color changed
Occurs when the color attribute is changed

font changed
Occurs when the font attribute is changed

objectNameUsage changed
Occurs when the objectNameUsage attribute is changed

text changed
Occurs when the text attribute is changed
