The PDLREG Procedure

Syntax

The following statements can be used with the PDLREG procedure:

PROC PDLREG option ;
BY variables ;
MODEL dependents = effects / options ;
OUTPUT OUT= SAS-data-set keyword = variables ;
RESTRICT restrictions ;
