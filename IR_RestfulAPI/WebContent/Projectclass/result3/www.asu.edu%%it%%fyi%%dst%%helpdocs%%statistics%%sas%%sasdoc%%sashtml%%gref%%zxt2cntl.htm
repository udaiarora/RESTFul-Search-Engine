TXT2CNTL Function

TXT2CNTL Function


Copies the values (XLSTT, YLSTT) to (XLAST, YLAST), replacing previous values of (XLAST, YLAST).

Syntax
Details

Syntax

FUNCTION='TXT2CNTL';

Details

TXT2CNTL allows nontext functions to use the ending position of a text string as a starting or ending point.
