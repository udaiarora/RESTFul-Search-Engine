SORTWKNO=


Specifies how many sort work data sets to allocate

Default: 3
Valid in: configuration file, SAS invocation, OPTIONS statement, OPTIONS window
Category: Sort: SORT
OS/390 specifics: all

Syntax
Details

Syntax

SORTWKNO=n

n
can be 0 through 6. If SORTWKNO=0 is specified, any existing sort work files are freed and none are allocated.

Details

The SORTWKNO= option specifies how many sort work files are to be allocated dynamically by either SAS or the SORT utility. (See DYNALLOC.)
