Data Set Data Model: _overrideOnRequired

Data Set Data Model: _overrideOnRequired


Allows or disallows override for required columns

Syntax

Syntax

CALL SEND (object-id, '_overrideOnRequired'<, override>);

Argument Type Description
override
 C 'Y' allow override on required (default)


'N' do not allow override on required.
