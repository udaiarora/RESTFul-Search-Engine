Widget Class: _scrollbarsOff

Widget Class: _scrollbarsOff


Hides the scroll bars of an object

Syntax
Details

Syntax

objectName._scrollbarsOff( );
objectName._scrollbarsOff( which );

Details

These objects support scroll bars:

Catalog Entry Viewer Organizational Chart
Data Form Process Flow Diagram
Data Table Report Viewer
Extended Table SAS/GRAPH Output
Extended Text Entry Work Area
Extended File Viewer

This method has no effect on other objects.
