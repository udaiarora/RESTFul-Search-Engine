    Transferring EndNote Citations

    • Instructions for EndNote version 8
    • Instructions for earlier versions

     

    For EndNote version 8

     

    You can import your EndNote database files directly into Refworks:

     

    1. In the RefWorks menu bar, select "References" and then "Import EndNote V8 Database"

     


    2. In the Import window that appears, click the "Browse" button, select the the database file for you EndNote database. It will end with a ".enl" extension.

     

     


    3. Click the "Import" button (it may take as long as 45 secs. for a large database file to load.

     

     


    For versions of EndNote previous to v. 8:

    If you are an EndNote user, you can export your saved EndNote citations to a text fine, and import this file into Refworks. To do so, you must first make sure to export EndNote citations in a format ("RefMan (RIS) Export" format) which Refworks recognizes.

    Set the proper output style:

    1. Click Edit
    2. Select "Ouput Styles"
    3. Select "RefMan (RIS) Export"

     

    If you do not see "RefMan (RIS) Export" in the pull down menu, you must add it to the list of output styles. To do this:

    1. Go to the menu bar and select "Edit"
    2. Select "Output Styles"
    3. Select "Open Style Manager" (it may take up to two minutes for the Style Manager to open)


    4. When the Style Manager opens, scroll down and make sure the the "RefMan (RIS) Export" option is selected.


    5. Close the Style Manager (click the "X" control in the upper right-hand corner).

     

     

     

     

    Once the correct output style is selected, you can export your records:

    1. Select "File" and "Export".


    2. In the "Save As" box, select the directory you wish to save your records to.
    3. Type in a file name. The records will save as a text file with the proper RefWorks formatting.

     

     

     

     

    Now, within Refworks, you can import these EndNotes records as you would for any other database.
