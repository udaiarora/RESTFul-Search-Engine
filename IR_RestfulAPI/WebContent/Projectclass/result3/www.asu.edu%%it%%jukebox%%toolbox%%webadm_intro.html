Using WebAdmin

Webadmin is a menu-driven utility to make maintaining AFS directory permissions easier.  Departments with webspace served through www.asu.edu can use Webadmin to be sure that the right people have access to update and maintain their respective websites.

Step 1: Log on to general.asu.edu

Use a terminal program such as SSH Secure Shell to login to the general server using the address general.asu.edu.

Step 2: Change to Your Web Directory

Once logged in, change the current directory to your web AFS directory.
(Example: cd /afs/asu.edu/www/YOURDIRECTORY)

Step 3: Start webadm

Type /afs/asu.edu/dist/webadm/webadm
and you will see the menu below.

This screen shows the current options available for web administration. Click on any of the menu items on the screen below to review information and instructions on their use.
