The likelihood function is maximized via either the dual quasi-Newton or trust region algorithm. The default is the dual quasi-Newton algorithm. When there are autoregressive parameters in the model, the initial values are obtained from the Yule-Walker estimates. The starting value 1.0-6 is used for the GARCH process parameters.

The variance-covariance matrix is computed using the Hessian matrix.
