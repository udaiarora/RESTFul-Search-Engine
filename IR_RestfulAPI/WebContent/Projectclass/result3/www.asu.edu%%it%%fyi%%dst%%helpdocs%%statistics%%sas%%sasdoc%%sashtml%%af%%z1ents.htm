Object Class Events

Events specified for the Object class are described here.

CBTFrameName changed
Occurs when the CBTFrameName attribute is changed

attachedInterface changed
Occurs when the attachedInterface attribute is changed

classID changed
Occurs when the classID attribute is changed

contents updated
Occurs when the value of an attribute in the contentsUpdatedAttributes is changed

contentsUpdatedAttributes changed
Occurs when the contentsUpdatedAttributes attribute is changed

defaultAttribute changed
Occurs when the defaultAttribute attribute is changed

help changed
Occurs when the help attribute is changed

model changed
Occurs when the model attribute is changed

modelID changed
Occurs when the modelID attribute is changed

name changed
Occurs when the name attribute is changed
