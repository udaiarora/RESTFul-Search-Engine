Recommended Auto Local Logon Configuration:

SecuRemote offers an Auto Local Logon feature on most Windows platforms. This allows a user to automatically authenticate with the firewall using the authority of the Windows login.

To use this feature, choose Configure Auto Local Logon… from SecuRemote's Passwords menu. Supply the user's ASUrite ID and password as well as the user's Windows ID and password. The ASUrite ID and password will be encrypted and stored on the workstation. Next, select Enable Auto Local Logon … from the Passwords menu. Future authentication requests for the configured user will be handled automatically as long as the stored ASUrite password and Windows password remain valid.
