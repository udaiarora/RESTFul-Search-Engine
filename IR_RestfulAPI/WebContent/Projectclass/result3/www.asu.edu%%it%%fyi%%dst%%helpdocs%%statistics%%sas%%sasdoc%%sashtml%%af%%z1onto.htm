Widget Class: _respondToDragOnto

Widget Class: _respondToDragOnto


Changes the drop site object's visual appearance to indicate a drop is allowed

Syntax
Details

Syntax

objectName._respondToDragOnto( );
objectName._respondToDragOnto( objectID );

Details

The _respondToDragOnto method is called when a drag site object is dragged over a valid drop site object. A visual change in the object is recommended to show the user that a drop is possible.
