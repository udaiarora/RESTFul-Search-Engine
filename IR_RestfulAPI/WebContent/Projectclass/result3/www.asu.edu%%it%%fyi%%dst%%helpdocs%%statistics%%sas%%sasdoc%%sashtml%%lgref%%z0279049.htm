CMDMAC


Determines whether the macro processor recognizes a command-style macro invocation

Valid in: configuration file, SAS invocation, OPTIONS statement, Systems Options window
Category: Macro: SAS macro
PROC OPTIONS GROUP= MACRO

Syntax
Syntax Description
Details

Syntax

CMDMAC | NOCMDMAC

Syntax Description

CMDMAC
specifies that the macro processor examine the first word of every window command to see whether it is a command-style macro invocation.

NOCMDMAC
specifies that no check will be made for command-style macro invocations.

Details

If the MAUTOSOURCE option is in effect, the macro processor searches the autocall libraries for the word. If you specify MAUTOSOURCE and CMDMAC, you can use computer resources more efficiently by specifying NOMRECALL, which causes the macro processor to search the autocall libraries only once for each keyword that begins a command.
