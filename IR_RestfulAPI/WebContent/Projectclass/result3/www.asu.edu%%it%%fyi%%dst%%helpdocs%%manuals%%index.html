Online Manuals

Some vendors of UNIX systems provide a portion (possibly all) of their manuals and other documentation in a format suitable for display in an interactive session. The format may be one for display in a web browser session, an x-windows session, or a character-based session (i.e., vt100 terminal telnet session).

The process for accessing these online manuals is described below for each of the indicated systems.

  • general.asu.edu (SUN systems)

    The documentation for SUN systems is available in web browser format. Point your browser to the url http://docs.sun.com/.

  • research.asu.edu (IBM AIX systems)

    The documentation for AIX systems is available in web browser format. Point your browser to the url http://www-1.ibm.com/servers/aix/library/.

  • medusa.asu.edu (SGI systems)

    The documentation for SGI systems is available in an x-windows session. The command for accessing the online documentation is "iiv".
