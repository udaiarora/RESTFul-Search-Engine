Opening an Existing Report

To open an existing report from the Report Engine window, select Open from the File menu. The Open window appears, as shown in the following display.

Open Window for Reports

You can type S in the Cmd field and press ENTER to select and open a report in the Report Engine window. To sort the reports, select Sort By from the Edit menu and select Name, Description, or Date to reorder the reports listed.
