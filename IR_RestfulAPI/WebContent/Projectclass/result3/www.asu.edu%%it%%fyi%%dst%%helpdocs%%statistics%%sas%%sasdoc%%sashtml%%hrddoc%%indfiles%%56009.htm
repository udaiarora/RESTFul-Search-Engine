SAS/ETS Software: Applications Guide 2, Version 6, First Edition

Attention econometricians, sociologists, psychologists! This guide shows you through various examples how to use SAS/ETS software for modeling, simulating, and forecasting with economic data. Each of the chapters gives examples you can work through, showing features of SAS/ETS software. Methods for analyzing linear and nonlinear regression models, with single or multiple equations, are also discussed.
