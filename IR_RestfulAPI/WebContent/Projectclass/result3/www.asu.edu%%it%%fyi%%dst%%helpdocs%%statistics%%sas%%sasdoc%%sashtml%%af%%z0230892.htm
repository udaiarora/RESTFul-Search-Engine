SAS/AF Software: Class Dictionary

Overview

With the Control class you can create a control object, which is an arrow within a box, that displays additional information when selected. The additional information may be a menu, a display, text, or a list of values for a field. On non-graphics devices, the arrow is replaced with a character such as a question mark (?).

Parent:
sashelp.fsp.widget.class

Class:
sashelp.fsp.Control.class
