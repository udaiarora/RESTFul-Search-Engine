SAS/ETS Software: Applications Guide 1, Version 6, First Edition

The many applications and examples in this guide will help you use SAS/ETS software for time series modeling and forecasting, financial reporting, and loan analysis. Save time developing your models using the ARIMA, AUTOREG, FORECAST, SPECTRA, STATESPACE, and X11 procedures. Also, the book covers SAS/ETS software capabilities for printing financial reports and analyzing loans. This guide is written for data analysts who are familiar with basic SAS System concepts.
