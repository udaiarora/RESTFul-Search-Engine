SAS/OR User's Guide: QSIM Application, Version 8

Learn how to build and analyze models of queueing systems using discrete event simulation with the QSIM Application. These kinds of models are useful in a variety of applications including: network and telecommunications systems, manufacturing systems, and transportation networks. This title shows you how to build models using the simple point and click graphical user interface; how to run the models; and how to collect and analyze the sample data to give you insight into the behavior of the system.
