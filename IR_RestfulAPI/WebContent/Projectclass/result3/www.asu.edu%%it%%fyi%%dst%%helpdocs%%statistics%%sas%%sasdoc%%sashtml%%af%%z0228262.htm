Critical Success Factor: _setDsname

Critical Success Factor: _setDsname


Assigns the name of the data set containing the current value of the CSF

Syntax
Details

Syntax

CALL NOTIFY (csf-name, '_setDsname', data-set-name);

Argument Type Description.
data-set-name
 C specifies the name of the SAS data set

Details

  • If you changed the data set name for a previously drawn CSF, the CSF is redrawn using the new data set.

  • If you changed the data in the data set, call the _update method to get the CSF to redraw with the changed data.
