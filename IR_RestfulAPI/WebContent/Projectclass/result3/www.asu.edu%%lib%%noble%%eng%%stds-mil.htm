Military Specifications

 

  • DoD Index of Specifications and Standards
    Since ASU Libraries does not own any Military Specitfications this web site is the only means to search and access these documents. Use the keyword search feature to search multiple fields. Full text of documents from 1986 is available in .pdf format.

 

SAE - Society for Automotive Engineers

 

Online:

  • SAE
    Use the search feature at the top of the web page to search by keyword or standard number or browse by selected categories. This site offers a pay per view service for the full text. The SAE Handbook has text of ground vehicle standards.

 

Paper:

  • SAE Handbook. Annual. Warrendale, PA: SAE, 1965- .
    TL 151.S62 Science Reference
    Covers recommended practices, information reports, and standards for ground vehicles. Standards for other subjects can be identified in the ANSI catalog, and are filed with the ANSI standards.

 

  • SAE Aerospace Index. Warrendale, PA: SAE, Cooperative Engineering Program, 1980- .
    TL 671.1 .S24 Science Reference
    This index includes aerospace standards, recommended practices and information reports. Some of these standards may be found in the ANSI cabinets most however need to be purchased through the SAE.

 

  • AMS Index. Warrendale, PA: SAE, Cooperative Engineering Program, 1989- .
    TL 671.1 .A5x Science Reference
    This index is still a valuable resource for identifying AMS (aerospace material specifications) standards developed by the SAE. To search more current standards or to order AMS standards visit the SAE web site.

 

 

SEMI - Semiconductor Equipment and Materials Institute

 

  • Book of SEMI Standards. Annual. Mountain View, CA: SEMI, 1981- .
    TK 7871.85 .S46a Science Reference 
    SEMI develops standards and guidelines for semiconductor manufacturing, as well as standards for materials and equipment to produce flat panel displays. Each volume of this multi-volume set is divided into sections, each with designated focus for that group of standards, e.g. "Materials" or "Software." Science Reference keeps the latest annual only.

 

 

Other Standards at ASU

 

ASU libraries owns a small number of standards besides those noted above (ASCE, AASHTO, etc.). These standards are located in the book stacks and can be found using the library catalog. To search for a specific standard, do a keyword search using the term "specifications" and words decribing the type of standard you need (e.g. bridges).
