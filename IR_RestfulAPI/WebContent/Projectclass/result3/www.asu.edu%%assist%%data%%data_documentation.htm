ASSIST Data Documentation

The data documentation for each data table in the ASSIST database consists of: (1) a table description including the primary purpose and general use of the table; and (2) a chart listing each data element with definitions and code values. New data elements, renamed data elements, and new code tables recently added or in the process of being changed are highlighted in red.

ASSIST data availability is determined by a number of factors including the availability of historical data, the time and resources of the participating institutions, and the varying categories for data element submission (required, highly recommended, and optional data elements).  The ASSIST Community College Data Availability Chart is a summary chart listing by institution the terms that each data table is populated.  It is updated annually in late summer.
