Basic LSF Commands

How to check Batch System status:

1. bqueues

Displays information about batch queues in the LSF Batch System.

How to submit Batch Job:

2. bsub

Submits a job for execution in the LSF Batch system. If no commands are given then it will read job commands from the standard input. If the standard input is a controlling terminal, bsub will prompt for commands.

Some of its options are:

-q QUEUE_NAME
This submits the job to the queue assigned to QUEUE_NAME. The default is short.

-m HOST
This submits a job to a specified cluster/host. Where HOST can be "stats" for the Stats cluster and "compute" for the compute cluster. This option is only needed when submitting jobs from one cluster to another.

-o OUTFILE
Writes the standard output of the job to the specified file. The default sends it via email to the user.

-e ERRORFILE
Writes the standard error output of the job to the specified file.

-b START_TIME
Starts the job at a specific time after moving it from the Pending state. The START_TIME is in "military-time", e.g., 5:00 for 5am, 16:30 for 4:30pm and 5:25:18:50 for May 25 at 6:50pm.

-N -u YOUR_EMAIL_ADDRESS
-N notifies you by email when the job finishes and -u sends email to your_email_address.

-J JOB_NAME
Assigns JOB_NAME as a character string to the job.

If -o is used and -e is not, then the standard error output will be sent to the file specified for standard output.

If both -o and -e are not used then the standard output and standard error output are merged and sent via email to the user.

How to track Batch Job:

3. bjobs

Displays the status and other information about jobs in LSF.

One of its options is:

-u all
This displays jobs for all users. The default is to display jobs submitted by the user who invoked this command.

There is a local created command called "q" that can also be use and it displays more information about current jobs.

4. bkill JOB_ID

Sends a signal to kill the specified job.

For more information, please refer to the LSF User's Guide available in the Customer Assistance Center (formerly COMPASS) or to the MAN pages. For example, to access the man page for bsub enter "man bsub" at the system prompt.
