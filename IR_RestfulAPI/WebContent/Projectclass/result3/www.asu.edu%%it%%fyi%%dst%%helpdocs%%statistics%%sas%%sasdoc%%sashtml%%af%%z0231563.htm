Extended Text Entry: _getLine

Extended Text Entry: _getLine


Returns the text of the specified line

Syntax

Syntax

CALL NOTIFY (ext_text_name, '_getLine', text-string, line-num);

Argument Type Description
text-string
 C returns the value of the line of the object
line-num
 N specifies the line number to return
