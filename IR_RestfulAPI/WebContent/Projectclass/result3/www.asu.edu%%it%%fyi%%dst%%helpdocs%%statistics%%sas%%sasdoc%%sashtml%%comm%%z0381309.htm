Communications Access Methods for SAS/CONNECT and SAS/SHARE Software

VSE: APPC Access Method

CAUTION:
Version 6 Only Beginning with Version 7, the VSE platform is not supported. However, information about VSE is included here for Version 6 users.  

SAS/CONNECT

No messages.

SAS/SHARE

Error message: 

Supporting access method initialization failed

The APPC access method could not be initialized. This message is issued if the REMOTE SERVER= option specifies a VTAM-appl-ID that is not valid. Make sure that the APPCAPPL= option has not been specified.

Error message: 

Target application partner could not be located

The server whose name was specified by the REMOTE= option in the LIBNAME or the PROC OPERATE statement could not be found. Make sure the server that you want to connect to is running.
