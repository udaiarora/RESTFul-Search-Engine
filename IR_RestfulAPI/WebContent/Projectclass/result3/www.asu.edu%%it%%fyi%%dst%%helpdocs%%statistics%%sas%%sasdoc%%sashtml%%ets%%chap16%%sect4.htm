The SIMLIN Procedure

Syntax

The following statements can be used with PROC SIMLIN:

PROC SIMLIN options;
BY variables;
ENDOGENOUS variables;
EXOGENOUS variables;
ID variables;
LAGGED lag-var endogenous-var number ellipsis ;
OUTPUT OUT=SAS-data-set options;
