Form Data Model: _getMaxrow


Returns the number of rows

Syntax

Syntax

CALL SEND (model-id, '_getMaxrow', numrows);

Argument Type Description
numrows
 N returns the number of rows in the model information:


-1 the number of rows is unknown


0 the number of rows
