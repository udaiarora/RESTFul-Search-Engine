$KANJIw.


Adds shift-code data to DBCS data

Category: DBCS
Alignment: left

Syntax
Syntax Description

Syntax

$KANJIw.

Syntax Description

w
specifies the width of the output field.
Restriction: The width must be an even number. If it is an odd number, it is truncated.
Range: The minimum width of the format is 2 + (length of shift code used on the current DBCSTYPE= setting)*2.
