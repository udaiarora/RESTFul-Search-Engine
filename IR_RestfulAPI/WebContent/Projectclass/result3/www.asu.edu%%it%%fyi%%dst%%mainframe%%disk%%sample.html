Sample JCL for Disk Utilities

Numerous sample batch jobs for performing various disk utility functions (e.g., copying files, listing names of files) are available online in a TSO/ISPF editing session.

Use the ISPF editor to edit the file into which you wish to insert one of the sample jobs. Once you are in the editor, enter the following command on the editor command line to identify which set of source files you want to copy:

     MODEL CLASS JCL

Next, enter the following command on the editor command line to display a menu of available sample batch jobs. (You also need to put "A" or "B" on one of the line number fields, so the editor knows where to insert the sample code.)

     MODEL

A menu will be displayed from which you choose whether you want sample disk JCL or sample tape JCL. Select the one in which you are interested. Another menu will be displayed which identifies the sample batch jobs by function. Choose the one most appropriate for your needs. (Use PF7 and PF8 to see all the available choices.)
