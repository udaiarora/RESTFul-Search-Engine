SAS/AF Software: Class Dictionary

Overview

The SCL List class is used to pass data to SAS Component Language methods that have SCL list parameters. An instance of the SCL List class provides the same functionality as a SCL list. Lists can consist of named or unnamed numeric, character, object, or other list values. Methods in this class are named based on the corresponding SCL function.
