CRC Handbook of Chemistry and Physics
QD 65 .H3 Science Library 1st floor, shelved in Reference Section
ASU Library Network Database
See also: Tables Removed from the 71st thru the 79th Editions

A good place to start, particularily if looking for basic physical data such as boiling and melting points, densities, refractive indexes and solubilities for your labs. Also includes data on thermodynamics, fluid properties, biochemistry, spectroscopy, atomic and nuclear physics, and polymers.

Combined Chemical Dictionary
ASU Library Network Database

Covers basic physical data of boiling and melting points and densities; some spectral and LogP data.

Merck Index
ASU Library Network Database

Covers basic physical data of boiling and melting points and densities; some spectral and LogP data.

ChemFinder
http://chemfinder.camsoft.com/

Provides name, synonyms, molecular formula, structure, melting and boiling points, density, refractive index, vapor density and pressure, water solubility and flash point, plus links to other Internet sites with information on the chemical.

NIST Data Gateway: Property Search
http://srdata.nist.gov/gateway/gateway?search=property

Provides links to NIST databases that contain name, synonyms, structures, atomic/nuclear data, spectra, phase diagrams, plus thermodynamic, kinetic, solubility, polymer and biochemistry data.

Instructions

This page is an index to selected library and internet resources that contain chemical, physical, thermodynamic, mechanical, toxicological, and safety data.

Start by using the standard reference sources listed at right (white column).

If the information is not found in the standard reference sources, then proceed to other library and Internet resources listed by property to find other handbooks and internet resources (yellow column).

In addition to resources recommended under each property, when in the library, also look in the same call number area as the recommended resources for other handbooks that may have that data.

If the property data cannot be found in handbooks or internet resources, search in primary sources such as journal articles. Use the appropriate database/index for that subject to find journal articles. For a list of databases/indexes by subject see the Find Articles and More section (also called the "Indexes and Research Tools" section).

Know how to effectively search handbooks, databases, and indexes:

Search for every synonym of the property in each handbook/database, as a property may have several names. (more details)

Search the subject area as well as the property name; look in the index or table of contents for the subject area in which the property would be found. (more details)

Know what symbol and/or unit of measurement is used for the property as some tables may be labeled by the symbol rather than the property name. (more details)

Search molecular formulas in Hill order for organics, and strict alphabetical order for inorganics. (more details)

Convert structures into molecular formulas if a structure search is not available or if unfamiliar with structure searching. (more details)

Search for every synonym of the substance, as a substance may have several names. (more details)

Searching by registry number is easier than searching by name, molecular formula or structure. (more details)

For more details on property searching techniques see: 10 Easy Tips for Finding Property Data
