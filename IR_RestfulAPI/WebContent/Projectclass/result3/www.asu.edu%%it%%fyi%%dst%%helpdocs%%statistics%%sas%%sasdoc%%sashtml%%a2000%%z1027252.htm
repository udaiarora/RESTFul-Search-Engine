SAS/ACCESS Interface to SYSTEM 2000 Data Management Software: Reference

Introduction

This chapter provides reference information for the QUEST procedure. It includes the PROC QUEST syntax as well as details of some SYSTEM 2000 statements and parameters as they specifically pertain to using the QUEST procedure. For examples of how to use the statement options, refer to Browsing and Updating SYSTEM 2000 Data.

The QUEST procedure enables you to create, access, and manage SYSTEM 2000 databases.

Refer to SAS Language Reference: Dictionary and the SAS documentation for your host system if you need help with the terminology used in this procedure description. For information about SYSTEM 2000 statements, see the current version of SYSTEM 2000 Quick Reference Guide for SCF Commands. Help is also available from within the SAS System by choosing Help then SAS System Help from the menu bar, then Help on SAS System Products and SAS/ACCESS in the help system.
