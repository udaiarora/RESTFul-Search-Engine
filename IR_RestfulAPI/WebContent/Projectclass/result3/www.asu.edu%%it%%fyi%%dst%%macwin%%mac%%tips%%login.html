IT : Mac —   How to do a secure remote login with OpenSSH


The Mac OS X comes with OpenSSH which in turn includes ssh. The ssh is a secure client program for logging into and executing commands on a remote machine.

To execute this client, download ASUOpenSSH.sit available from:

     http://www.asu.edu/it/security/software/html/ssh/download.htm

The ASUOpenSSH.sit file will unstuff itself and create a folder called ASUOpenSSH. The folder consists of applications called ASUsftp and ASUssh. Down-click on ASUssh to execute it.

ASUssh is a basic interface to the OpenSSH ssh client that lets you select a server, asks you for a user id and then connects to that server from a Terminal window.

In the Terminal window you will be prompted for a password that correspondes to your user id. After successfully logging in, you may then proceed with your work.
