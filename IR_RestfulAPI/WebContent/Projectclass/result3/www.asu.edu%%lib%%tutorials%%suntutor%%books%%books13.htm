Truncation

Truncation is another technique used to broaden your search to increase the number of books retrieved.

Truncation allows you to search for words that begin with the same letters, but which have different endings, by using a punctuation mark called a truncation symbol. For example, violen*

media and violen*

The above search uses the asterisk (*) as the truncation symbol to search for the words “violence,” “violent,” “violently.”
