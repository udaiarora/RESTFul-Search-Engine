Class Class: _getInstanceList


Returns a list containing all direct instances of the class

Syntax
Details

Syntax

objectName._getInstanceList( instanceList );

Argument Type Use Description
instanceList Numeric Update specifies the identifier of a list that when returned is filled with all instances of the class

Details

Each class maintains an internal list of instances; _getInstanceList returns the identifier of that list.

  • You may fetch the object identifiers from instanceList by using the GETITEMN SCL function.

  • By comparison, the Class Class: _getInstances method can create a new list and copies the contents of the _getInstanceList list into it.
