External File List Model: _loadItems

External File List Model: _loadItems


Loads the list of items to display

Syntax

Syntax

objectName._loadItems( externalPath, file,directory,name );

Argument Type Use Description
externalPath Character Input specifies whether to build the list including the external path for each item
file Character Input specifies whether to build the list including the file name for each item
directory Character Input specifies whether to build the list including the directory for each item
name Character Input specifies whether to build the list including the external file name for each item
