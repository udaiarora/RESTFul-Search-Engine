Profile Macro

You can use various uni-XEDIT commands to modify or customize your editing sessions. These commands can be entered manually at the editor command prompt, or they can be executed from a disk file, i.e., a macro. (A macro is a disk file containing one or more uni-XEDIT commands or a uni-REXX program.)

If you name the macro ".profile.xedit" and place it in your home directory, it will be executed automatically whenever you begin an editing session (unless you indicate otherwise on the command line when starting the editor). If the macro has a different name or is not in the home directory, it will be executed at the start of an editing sessions only if you use the -PROF option on the command line.

Included in the profile macro should be the SET KEYBIND statements generated by the "key" utility which determines the keyboard mapping for your editing sessions. You can also include any other commands to be executed automatically when the editing session begins. Some sample commands you may want to include in the profile are shown below:

set case mixed ignore
set autosave 10
set number on

"set case mixed ignore" allows case-sensitive data to be entered into a file but ignores it for searches.

"set autosave 10" automatically saves to a ".AUTOSAVE" backup file after every 10 lines of data entry.

"set number on" displays the line numbers in the prefix area.
