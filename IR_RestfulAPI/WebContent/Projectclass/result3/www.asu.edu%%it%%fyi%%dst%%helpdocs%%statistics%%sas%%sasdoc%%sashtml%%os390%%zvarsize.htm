MVARSIZE=

MVARSIZE=


Specifies the maximum size for macro variables that are stored in memory

Default: 8,192 bytes
Valid in: configuration file, SAS invocation, OPTIONS statement, OPTIONS window
Category: Macro: MACRO
OS/390 specifics: default value, range of available values

Syntax
See Also

Syntax

MVARSIZE=n | nK | nM | nG | MIN | MAX | hexX

n | nK | nM | nG
specifies maximum macro variable size in bytes, kilobytes, megabytes, or gigabytes, respectively.

MIN
sets maximum macro variable size to 0 and requires SAS to use the default value.

MAX
sets maximum macro variable size to 2,147,483,647.

hexX
specifies maximum macro variable size as a hexadecimal number of bytes.
