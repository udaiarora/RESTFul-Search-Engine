E-Mail Retention Policy

 

For Exchange and IMAP users, only messages currently in the mailbox are available. From Exchange, if an item is "deleted", this item is recoverable by the user for a period of 7 days by using the recover Deleted Items tool in the Outlook client. See below for instructions.

For an IMAP user using EMMA, deleted items are recoverable until the Deleted Items have been "expunged". Once the "expunge" feature has been used the items are no longer available.

ASU IT does not perform recovery on individual mailboxes. Our backup tapes can only be used for "disaster recovery", which would restore ALL of the Exchange users mailboxes to the point of the backup tape.

Using 'Recover Deleted Items':

From the Outlook client, click on your Deleted Items folder to select it, then choose Tools> Recover Deleted Items

A dialog box will appear, with messages deleted within the past 7 days using the deleted Items folder, visible.

Select the message(s) you wish to recover and choose the Recover Selected Items icon. The item is then recovered and moved back to the Deleted Items folder. From here it can be moved back into your Inbox or a personally defined folder.
