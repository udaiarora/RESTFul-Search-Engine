PF-key Definitions

During an editing session, the PF keys can be defined to execute other commands, such that when you press a PF key (e.g., PF8) the corresponding command is executed (e.g., forward). (For information regarding keyboard mapping, refer to the Customized Keyboard Mapping topic or the Generic Keyboard Mapping topic.)

The predefined values (commands) assigned to the PF keys when an editing session is begun are shown below. Use the "SET PF" command to change the value associated with a PF key. Use the "QUERY PF" during an editing session to see the current definitions of the PF keys.

PF1 and PF13
BEFORE HELP - displays help information
PF2 and PF14
undefined
PF3 and PF15
BEFORE QUIT - quit the editing session for the current file
PF4 and PF16
BEFORE TABKEY - move cursor to the next tab position
PF5 and PF17
BEFORE SCHANGE 6 (and SCHANGE 18) - selective change
PF6 and PF18
ONLY ? - redisplays last command
PF7 and PF19
BEFORE BACKWARD - page backward
PF8 and PF20
BEFORE FORWARD - page forward
PF9 and PF21
ONLY = - re-execute last command
PF10 and PF22
BEFORE RGTLEFT - moves the screen left or right
PF11 and PF23
BEFORE SPLTJOIN - splits or joins lines
PF12 and PF24
BEFORE CURSOR HOME - relocates the cursor to the home position
