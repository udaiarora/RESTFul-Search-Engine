The ASSIGN Procedure

Output Data Set

The output data set contains the m cost variables in the input data set, any variables identified in the ID statement, and two new variables named _ASSIGN_ and _FCOST_. The variable named _ASSIGN_ is a character variable containing the names of the sink nodes (names of COST variables) assigned to the source nodes (values of the ID variable). The variable named _FCOST_ is a numeric variable containing the costs of assigning the sink nodes to the source nodes. Note that the values of the m cost variables in the output data set reflect any effects of scaling performed by PROC ASSIGN.
