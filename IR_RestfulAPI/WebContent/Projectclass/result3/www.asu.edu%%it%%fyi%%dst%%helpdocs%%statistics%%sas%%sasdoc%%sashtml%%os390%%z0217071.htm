SORTPARM=


Specifies a string of parameters to pass to your host sort utility

Default: none
Valid in: configuration file, SAS invocation, OPTIONS statement, OPTIONS window
Category: Sort: SORT
OS/390 specifics: all

Syntax
Details

Syntax

SORTPARM='string'

string
is a string of parameters. It can contain up to 255 characters. Single quotes are optional unless string contains blanks or special characters.

Details

The string of parameters that you specify is appended to the OPTIONS statement that is generated by the SAS host sort interface. This enables you to specify options that are unique to the particular sort utility you are using. The sort utility must accept a 31-bit parameter list and an OPTIONS statement; otherwise, this option is ignored.
