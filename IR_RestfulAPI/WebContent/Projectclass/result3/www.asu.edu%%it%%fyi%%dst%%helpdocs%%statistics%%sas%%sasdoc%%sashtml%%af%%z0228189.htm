Control: _setFill

Control: _setFill


Assigns a fill style to a control (empty or filled)

Syntax
Details

Syntax

CALL NOTIFY (control-name, '_setFill', type);

Argument Type Description
N
specifies the fill style to assign


1 filled


0 empty

Details

Fill style is ignored on non-graphics devices.
