Scatter Control: scale


the scale value for changing the size of the chart

Type: Numeric
Valid Values:

Details

Details

The scale property effectively performs a zoom on the graph. It is the same as actionMode=Zoom or the action performed when you select Zoom from the build-mode popup menu.

A value of 1 scales to default size, a value of 2 scales to twice the default size, etc. Successively smaller or larger values will ultimately result in an undesirable graphic, so you should experiment with input values. Values outside the ranges are ignored.

For viewMode=2D or 3D, valid values are from .001 to 30.0. For viewMode=3DPerspective, valid values are from .102 to 25.0.
