Image Icon: _setLabelPosition

Image Icon: _setLabelPosition


Assigns the position of the label for the image icon

Syntax

Syntax

CALL NOTIFY (icon-name, '_setLabelPosition', position-char);

Argument Type Description
position-char
 C the position in which to display the label for the image icon:


'R' to the right of the image


'U' underneath the image
