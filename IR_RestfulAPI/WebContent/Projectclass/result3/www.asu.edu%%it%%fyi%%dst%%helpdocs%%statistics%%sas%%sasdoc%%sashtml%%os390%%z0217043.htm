MSG=

MSG=


Specifies the file specification of the partitioned data set that contains SAS messages

Default: SASMSG
Alias: SASMSG=
Valid in: configuration file, SAS invocation
Category: Environment Control: ENVFILES
OS/390 specifics: all

Syntax
Details
See Also

Syntax

MSG=file-specification

file-specification
identifies an external file. Under OS/390, it can be a valid DDname or a physical file name. The DDname must have been previously associated with an external file using either a TSO ALLOCATE command or a JCL DD statement.

Details

Under OS/390, the MSG= system option specifies the file that contains error, warning, and informational messages that are issued during a SAS session.
