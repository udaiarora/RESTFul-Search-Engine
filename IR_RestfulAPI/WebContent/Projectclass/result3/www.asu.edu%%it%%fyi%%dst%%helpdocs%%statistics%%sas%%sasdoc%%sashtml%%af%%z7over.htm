Overview

The Radio Box Control creates a radio box, which includes a group of labeled buttons from which users can make a single selection. Users can choose radio buttons by selecting either the button or its text label.

Radio buttons are always mutually exclusive; only one button can be selected at any time. A previously selected radio button is automatically deselected when another radio button is selected.

Radio boxes (and buttons) are helpful when a user must choose between two or more items.
