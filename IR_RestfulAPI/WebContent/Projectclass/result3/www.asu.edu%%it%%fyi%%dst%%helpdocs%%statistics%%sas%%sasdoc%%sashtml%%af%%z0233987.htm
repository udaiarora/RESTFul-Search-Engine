SAS/AF Software: Class Dictionary

Overview

The Image Data Model class is a nonwidget class that defines an object-oriented interface to images. Using this class, together with the Image Color Map class, you can manipulate images in object-oriented applications.

Objects in the Image Data Model class can

  • read images from external files, SAS catalogs, or devices such as cameras and scanners

  • manipulate those images

  • write images to external files or SAS catalogs.

Parent:
sashelp.fsp.object.class

Class:
