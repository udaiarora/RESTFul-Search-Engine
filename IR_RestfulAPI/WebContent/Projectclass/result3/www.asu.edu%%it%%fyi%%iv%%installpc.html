Install from PC

The following instructions are for installing Windows-based software.

To use copyrighted software in a network setting, you must have appropriate licensing (one license or copy of a program usually gives the right to run one copy of the software at any one time). For example, if you have ten licenses for a program, only ten PCs can use that software at the same time. In some instances a vendor may grant permission to run multiple copies of their software from one licensed copy. Be sure to get this permission in writing. Information Technology has utilities that can meter (count/restrict) the number of copies of a windows-based program running at the same time. Send an e-mail note to DST if you want to meter a program.

Note: Installation of software to an IV from an office PC is not available because the install program needs to "see" the configuration of the PC on which the software will be running.

Do not test the running of programs on an Instructor Volume while logged on with your ASURITE UserID. Remember your UserID has write access to the Instructor Volume and student UserIDs do not. A student UserID is available for testing purposes. Send an e-mail note to DST to obtain the test UserID.

Installing Windows-Based Software

You have two options for installing software on an Instructor Volume:

  1. Place the installation files on the Instructor Volume for students to use.
  2. Send an e-mail to dst@asu.edu requesting assistance in setting up software on an Instructor Volume.
