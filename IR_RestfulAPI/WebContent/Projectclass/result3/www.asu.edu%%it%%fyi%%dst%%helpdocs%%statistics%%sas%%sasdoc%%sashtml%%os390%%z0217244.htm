Restrictions on the Use of These Engines

Because these are sequential engines, they cannot be used with the POINT= option of the SET statement, nor with the FSBROWSE, FSEDIT, or FSVIEW procedures in SAS/FSP software. However, you can use PROC COPY or a DATA step to copy a BMDP, SPSS, or OSIRIS file to a SAS data set, and then either use POINT= or use SAS/FSP to browse or edit the file.
