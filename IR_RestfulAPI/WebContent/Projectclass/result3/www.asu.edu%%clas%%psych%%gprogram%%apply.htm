Download the PDF version of the Application Packet:

The ASU Psychology Application Packet is provided as a single Adobe PDF file containing application materials: the letter of recommendation form, the statement of purpose form, the applicant response card, and the applicant response form.

To download the application packet, click the following link:

»» Download The Psychology Department Application Forms Packet

If you run into problems downloading the file by clicking on the provided link, then try the following steps:

Point at the link with your mouse and click the Right (other, non-left) mouse button. This pops up a menu similar to the one in this picture


Click on the 'Save Target As' option in this menu. You will be prompted to save the file to your system. Save the file to a directory you can find again. From there you should be able to successfully open the file in Adobe Reader by double clicking on the file name.

Or you may read it in your Browser

Or you can download the file in *.doc format.

By clicking this link you may download a copy of the packet in *.doc format that is readable in Word or WordPad.

Or you can download the file in *.rtf format.

Or you may request a copy of the PDF document to be sent to you in email:

To obtain a copy of Psychology Department application packet file by email, send e-mail to psygrad@asu.edu with your name, address, and area of interest (i.e., clinical, cognitive systems, behavioral neuroscience, developmental, quantitative, or social). We will email the PDF file to the email address you provide.

NOTE: The department strongly supports diversity among applicants to its six graduate programs, and encourages applications from members of underrepresented groups.
