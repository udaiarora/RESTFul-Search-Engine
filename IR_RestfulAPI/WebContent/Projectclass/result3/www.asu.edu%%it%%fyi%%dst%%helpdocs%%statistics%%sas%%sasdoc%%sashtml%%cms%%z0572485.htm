RTRACE=

RTRACE=


enables collection of data on SAS file I/O

Default: NONE
Valid in: configuration file, SAS invocation
CMS specifics: all

Syntax
See Also

Syntax

RTRACE=NONE | ALL

NONE
causes SAS software to log no file I/O activity.

ALL
causes SAS software to log all file I/O activity.
