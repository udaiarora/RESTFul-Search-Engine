SORTMSG=

SORTMSG=


Specifies the DDname to be dynamically allocated for the message print file of the host sort utility

Default: SYSOUT
Valid in: configuration file, SAS invocation, OPTIONS statement, OPTIONS window
Category: Sort: SORT
OS/390 specifics: all

Syntax
Details

Syntax

SORTMSG=DDname

DDname
can be any valid DDname or a null string. The DDname will be dynamically allocated to either a SYSOUT data set (with class *) under batch or a terminal under TSO, and the DDname passed to the host sort utility.

Details

The SORTMSG= option specifies a DDname that will be dynamically allocated to either a SYSOUT data set (with class *) or a terminal.
