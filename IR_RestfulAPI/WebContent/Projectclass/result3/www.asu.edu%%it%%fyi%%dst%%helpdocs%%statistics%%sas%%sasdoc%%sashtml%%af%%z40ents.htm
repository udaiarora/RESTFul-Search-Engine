Program Halt Component Events

Events specified for the Program Halt Component class are described here. See Object Class Events for inherited events.

attributeError
Occurs as a result of an attribute error

generic
Occurs as a result of a generic error

genericMath
Occurs as a result of a genericMath error

overFlow
Occurs as a result of an overflow error

underFlow
Occurs as a result of an underflow error

zeroDivide
Occurs as a result of a zeroDivide error
