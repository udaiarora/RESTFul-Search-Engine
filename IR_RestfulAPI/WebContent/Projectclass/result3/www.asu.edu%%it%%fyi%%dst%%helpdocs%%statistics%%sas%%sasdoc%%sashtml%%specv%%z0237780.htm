The G4GRID Procedure

Using the Graphical Interface

A FRAME entry application gives you a graphical interface for the G4GRID procedure so that you can process a G4GRID request without knowing the PROC G4GRID syntax. To use the G4GRID interface, issue the command: 

af c=sashelp.g4grid.g4grid.frame

When the Data Interpolation window displays, see its online help for how to use it.
