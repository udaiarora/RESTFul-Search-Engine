FILEDEST=


Specifies the default printer destination

Default: none
Valid in: configuration file, SAS invocation, OPTIONS statement, OPTIONS window
Category: File Control: EXTFILES
OS/390 specifics: all

Syntax
Details
See Also

Syntax

FILEDEST=printer-destination

Details

The FILEDEST= system option specifies the default destination to be used for printer data sets when the DEST= option is omitted. This can occur when the FILENAME statement or FILENAME function does not have a DEST= value or when the form being used does not have a DEST= value.
