Using IMSL Library in Programs

Q. How can I use the IMSL library in programs?

To use the IMSL library with your programs, simply include calls in your FORTRAN or C code to the IMSL functions you want to use and then compile with the -limsl option as follows:

For fortran:

f77 program.f -limsl

For C:

cc program.c -limslcmath

for the math library and

cc program.c -limslcstat

for the statistics library.

The machine where you compile and run your programs must have the IMSL library in order to use it. To find out more try the below Web link.

mathlib.htm
