SORTSUMF


Specifies whether the host sort utility supports the SUM FIELDS=NONE control card

Default: NOSORTSUMF
Valid in: configuration file, SAS invocation, OPTIONS statement, OPTIONS window
CMS specifics: all

Syntax

Syntax

SORTSUMF | NOSORTSUMF

SORTSUMF
specifies that the host sort utility supports the SUM FIELDS=NONE control card. If the SORTSUMF option is specified and the SORT procedure is invoked with the NODUPKEY parameter, the host sort utility can be used.

NOSORTSUMF
specifies that the host sort utility does not support the SUM FIELDS=NONE control card. If NOSORTSUMF is specified and the SORT procedure is invoked with the NODUPKEY parameter, the SAS sort utility will be used.
