The GOPTIONS Procedure

Example 2: Displaying Graphics Options without the Description

Procedure features:
PROC GOPTIONS statement:
SHORT
Sample library member: GR20N02

This example uses the SHORT option to display only the values of graphics options without the description of each graphics option. Using the SHORT Option (GR20N02) shows the listing that appears in the LOG window.
