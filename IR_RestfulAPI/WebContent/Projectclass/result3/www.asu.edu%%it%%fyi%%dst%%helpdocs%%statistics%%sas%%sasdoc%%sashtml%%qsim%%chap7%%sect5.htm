The Component Dictionary Data Set

The component dictionary data set is always in WORK.DICT. It has two variables, CMPONENT and ID. Figure 7.8 shows a printout of the data set for a simple M/M/1 model.


Figure 7.8: An M/M/1 Example Dictionary Data Set

The value of the CMPONENT variable will be whatever label is associated with the component. Note that only elementary components will appear in this data set.
