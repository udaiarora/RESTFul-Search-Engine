Print Manager class: _getBottommargin


Returns the bottom margin setting of the document (in inches)

Syntax
Details

Syntax

return = objectName._getBottommargin( );

Details

Any margin specified is in addition to margins that the target device may have.

This method will program halt if numberofunits is <= 0, or if units is not a valid unit of measure. See Using the Print Manager Class for information on valid units of measure.
