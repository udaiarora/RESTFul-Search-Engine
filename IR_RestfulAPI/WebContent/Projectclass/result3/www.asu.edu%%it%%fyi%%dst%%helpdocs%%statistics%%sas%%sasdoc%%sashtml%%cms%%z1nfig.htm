CONFIG=

CONFIG=


Specifies the name of the configuration file to be used when SAS software is invoked

Default: SASV8 CONFIG *
Valid in: SAS invocation
CMS specifics: all

Syntax
Details
See Also

Syntax

CONFIG='<filename filetype filemode | SFS-directory>' | NOCONFIG

'filename filetype filemode | SFS-directory'
specifies the complete CMS file id of an external file. The file id must be enclosed in quotes. If only the filename is specified, the quotes may be omitted. If filemode or SFS directory is omitted, * is assumed. If filetype is omitted, CONFIG is assumed.

NOCONFIG
tells SAS not to use any configuration file.

Details

If you do not want any configuration file used, including the default, specify NOCONFIG in the SAS command.
