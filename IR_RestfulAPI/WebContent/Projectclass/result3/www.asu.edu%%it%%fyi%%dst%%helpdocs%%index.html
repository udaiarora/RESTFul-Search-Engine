About the Academic UNIX Help Facility

The Help Facility provides information specific to the ASU UNIX servers and is developed for introductory to intermediate level UNIX users. Additionally, it provides information on related topics which users may also find useful. The help facility can be accessed from any web browser at the following URL:

http://www.asu.edu/it/fyi/dst/helpdocs/

You may use a web browser such as Netscape Navigator on an Microsoft Windows computer or on an Apple Macintosh to access the Help information.

The help facility can be accessed also from a telnet session to one of the following ASU UNIX servers:

  • general.asu.edu
  • research.asu.edu/stats.asu.edu

The command for accessing it is:

help

The command must be entered in all lowercase letters. Additional information for accessing the help system from a telnet session is available.

The UNIX man pages generally available on all UNIX systems, provide complete descriptions and command syntax for most UNIX commands. However, it may require some UNIX experience to become proficient at finding information on topics of interest. The help facility on the other hand, serves as direct source of information on some of the UNIX commands and on many of the applications currently available on the ASU UNIX servers.

The help facility among its many features has one which allows for the transferring of available writeups that are in the Rich Text Format to your local computer. The writeups can be displayed with help of a word processor (e.g. MS Word, Word Perfect) on an IBM compatible or Mac computer and if necessary sent to an attached printer for printing.

As mentioned above, the help facility besides providing useful information on the ASU UNIX systems, serves as a means also for providing access to information on several related topics and is intended for the benefit of the ASU community. Any feedback for improvement is welcome via an Email message to helpdesk@asu.edu.
