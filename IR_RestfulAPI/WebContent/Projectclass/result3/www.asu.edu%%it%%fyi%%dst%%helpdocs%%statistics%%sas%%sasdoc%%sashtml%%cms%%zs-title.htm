TITLE


Specifies title lines for SAS output

Valid: anywhere
CMS specifics: maximum length of title

Syntax
Details
See Also

Syntax

TITLE <n > <'text' |"text">;

Details

Under CMS, the maximum title length is determined by the value of the LINESIZE= system option. The maximum value of the LINESIZE= system option is 256 in the CMS environment.

If the specified title is greater than the value of the LINESIZE= option, the title is truncated to the specified line size.
