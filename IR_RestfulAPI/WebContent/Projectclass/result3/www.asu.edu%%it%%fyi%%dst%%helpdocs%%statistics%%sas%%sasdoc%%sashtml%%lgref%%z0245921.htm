NORMAL

NORMAL


Returns a random variate from a normal distribution

Category: Random Number
See: RANNOR

Syntax
Arguments

Syntax

NORMAL(seed)

Arguments

seed
is an integer.
Range: seed < 231 - 1
Note: If seed 0, the time of day is used to initialize the seed stream.
