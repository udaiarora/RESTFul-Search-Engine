Revenue Coding		  

Transfer-In

Applicability

Transfers may be made between agency/orgs with the exception of:

  1. state units, i.e., agency/orgs whose third digit of the agency code ends with a “1”

  2. agency units, i.e., agency/orgs whose agency code is YV7

    and

  3. Sponsored Property Fiscal Management (SPFM) units, i.e., agency/orgs whose agency code consists of three alpha characters

Policy

A transfer-in to an agency/org must be offset by a transfer-out from another agency/org for the same dollar amount. Transfer codes may only be used when processing a journal voucher (J1) or a transfer voucher (TV).

When using a transfer-in revenue code, a function code must be used. The function code consists of the agency code for the “transfer-out agency/org” plus the last digit of the org code of the “transfer-out agency/org.” For example, if funds are being transferred from agency/org WV5 1002, the function code used is WV52.
