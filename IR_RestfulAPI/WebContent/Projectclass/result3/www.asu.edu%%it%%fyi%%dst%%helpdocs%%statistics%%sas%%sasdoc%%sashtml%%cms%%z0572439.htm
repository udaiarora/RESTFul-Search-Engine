MSGCASE

MSGCASE


Specifies whether all messages in the message file are uppercase

Default: NOMSGCASE
Valid in: configuration file, SAS invocation
CMS specifics: all

Syntax
See Also

Syntax

MSGCASE | NOMSGCASE

MSGCASE
specifies that all messages in the message file are to be uppercase.

NOMSGCASE
specifies that messages in the message file are to be mixed case.
